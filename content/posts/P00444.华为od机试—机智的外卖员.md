## 二、 题目描述

外卖员每天在大厦中送外卖，大厦共有 $L$ 层（$0 < L \le 10^5$）。

当他处于第 $N$ 层楼时，可以通过以下三种方式移动：
1. 每分钟通过步行梯向上达到 $N+1$ 层。
2. 每分钟通过步行梯向下达到 $N-1$ 层。
3. 每分钟乘坐电梯达到 $2 \times N$ 层。

给定他当前所处位置 $N$，以及外卖配送的目的楼层 $M$，计算他送达的最短时间（分钟）。

## 三、 输入描述

输入两个整数 $N$ 和 $M$，分别表示当前所处楼层和目的楼层。
中间用空格隔开。

## 四、 输出描述

输出一个整数，表示送达的最短时间。

## 五、 用例

### 用例1
**输入**：
```text
5 17
```
**输出**：
```text
4
```
**说明**：
当前处于5层，目标楼层17层，最快的方式是：
- 步行下一层 -> 4层
- 电梯 -> 8层
- 电梯 -> 16层
- 步行上一层 -> 17层
共需要4分钟。

### 用例2
**输入**：
```text
10 10
```
**输出**：
```text
0
```

## 六、 题目解析

### 1. 题目分析
本题属于典型的**最短路径问题**。
在一个数轴（楼层）上，从起点 $N$ 出发，每次可以执行三种操作（$+1, -1, \times 2$），求到达 $M$ 的最少步骤。
由于每一步的时间成本都是 1 分钟（等权重），因此最适合使用 **广度优先搜索 (BFS)** 来求解。

### 2. 解题思路
- **特殊情况处理**：
    - 如果 $N \ge M$：因为乘梯只能使层数翻倍（变大），所以只能通过步行向下（$N-1$）。最短时间直接就是 $N - M$。
- **BFS 过程**：
    - 使用一个队列存放当前到达的楼层。
    - 使用一个 `visited` 数组记录已经访问过的楼层，避免重复搜索。
    - 楼层范围限制：题目给出 $L \le 10^5$。在搜索过程中，如果楼层远超 $M$（比如超过 $M+1$ 或 $10^5$），继续向上或翻倍通常不是最优解，因此可以将边界限制在 $[0, 100000]$ 范围内。
- **状态转移**：
    1. $curr + 1$ (且 $\le 100000$)
    2. $curr - 1$ (且 $\ge 0$)
    3. $curr \times 2$ (且 $\le 100000$)

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) return;
        int n = sc.nextInt();
        int m = sc.nextInt();

        // 如果起始位置在目标位置之上，只能往下走
        if (n >= m) {
            System.out.println(n - m);
            return;
        }

        // BFS 寻找最短路径
        System.out.println(bfs(n, m));
    }

    private static int bfs(int start, int target) {
        // 题目给定范围 10^5
        int limit = 100001;
        boolean[] visited = new boolean[limit + 1];
        Queue<int[]> queue = new LinkedList<>();

        // 队列存储 [当前楼层, 消耗时间]
        queue.offer(new int[]{start, 0});
        visited[start] = true;

        while (!queue.isEmpty()) {
            int[] current = queue.poll();
            int pos = current[0];
            int step = current[1];

            // 达到目标楼层
            if (pos == target) {
                return step;
            }

            // 尝试三种移动方式
            int[] nextPositions = {pos + 1, pos - 1, pos * 2};
            for (int next : nextPositions) {
                // 边界检查及是否访问检查
                if (next >= 0 && next < limit && !visited[next]) {
                    visited[next] = true;
                    queue.offer(new int[]{next, step + 1});
                }
            }
        }
        return -1;
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(L)$。在最坏情况下，我们需要遍历大厦的所有楼层（$L=10^5$），每个节点只会被放入队列一次。
- **空间复杂度**：$O(L)$。需要一个 `visited` 数组和队列来存储访问过的节点。
