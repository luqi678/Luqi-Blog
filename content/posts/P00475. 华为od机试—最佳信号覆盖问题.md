## 一、 题目描述

模拟 AP（无线访问接入点）安装，将 AP 的位置投影到二维坐标系中。给出每个 AP 的 WIFI 信号强度，信号强度会随着距离的增加而减弱。

**计算规则与条件：**
1.  **输入**：
    * 第一行包含两个整数 $N, D$ ($N \le 100, D \le 100$)。
    * $N$ 表示 AP 的数量。
    * $D$ 表示 AP 信号能够覆盖的最大距离。
    * 接下来 $N$ 行，每行 3 个整数 $x, y, s$，表示 AP 的位置 $(x,y)$ 和初始信号强度 $s$。
    * 题目限定 $x, y > 0$。
2.  **距离计算**：
    * 使用 **切比雪夫距离 (Chebyshev Distance)**。
    * 对于两点 $(x_1, y_1)$ 和 $(x_2, y_2)$，距离 $d = \max(|x_1 - x_2|, |y_1 - y_2|)$。
3.  **信号衰减**：
    * 如果第 $i$ 个 AP 能到达坐标 $(x, y)$（即距离 $d \le D$），则该 AP 在此处的信号强度为 $\lfloor s / (1 + d) \rfloor$（向下取整）。
    * 如果 $d > D$，则该 AP 在此处的信号强度为 0。
4.  **总信号强度**：
    * 一个坐标的信号强度是所有能到达该坐标的 AP 的信号强度之和。

**目标**：
找到 WIFI 信号最好的坐标。

## 二、输入描述

1.  第一行：2 个整数 $N, D$。
    * $N \le 100$
    * $D \le 100$
2.  接下来 $N$ 行：每行 3 个整数 $x, y, s$。
    * 表示 AP 的坐标 $(x, y)$ 和强度 $s$。

## 三、输出描述

输出 WIFI 信号最好的坐标 $(x, y)$。

**排序/冲突规则**：
如果存在多个信号强度一致的坐标，输出 **字典序最小** 的非负坐标。
坐标 $(x_1, y_1)$ 字典序比 $(x_2, y_2)$ 小，需满足以下条件之一：
* $x_1 < x_2$
* $x_1 == x_2$ 且 $y_1 < y_2$

## 四、用例

**示例 1**

```text
输入：
3 2
1 2 3
2 1 3
3 1 3

输出：
1 2

说明：
3个AP分别在(1,2), (2,1), (3,1)，初始强度均为3，覆盖范围D=2。
对于坐标(1,2)：
- AP1(1,2): 距离=0, 信号=floor(3/(1+0)) = 3
- AP2(2,1): 距离=max(|1-2|, |2-1|) = 1, 信号=floor(3/(1+1)) = 1
- AP3(3,1): 距离=max(|1-3|, |2-1|) = 2, 信号=floor(3/(1+2)) = 1
总信号 = 3+1+1 = 5。
经遍历计算，该点信号最优。
```

## 五、备注

1.  坐标点是在 X-Y 坐标系内的整数坐标。
2.  输入 AP 坐标 $x,y > 0$，但输出要求是“非负坐标”，通常意味着从 $(0,0)$ 开始的有效区域。
3.  切比雪夫距离的定义重点在于横纵坐标差值的最大值。

## 六、题目解析

### 1. 核心思路
这是一道典型的**暴力枚举（Simulation/Brute Force）**题目。

由于题目给出的数据范围非常小：
* $N \le 100$
* $D \le 100$
* AP 坐标虽然没有明确上限，但通常在合理的测试范围内。

我们可以确定一个**搜索范围**，遍历该范围内的每一个整数坐标点，计算该点接收到的总信号强度，并维护一个最大值。

### 2. 确定搜索范围
我们需要遍历哪些点？
* 理论上，最佳位置一定在某个 AP 的覆盖范围内。
* 我们可以找出所有 AP 坐标的边界：$min\_x, max\_x, min\_y, max\_y$。
* 搜索范围的 $x$ 轴应为 $[min\_x - D, max\_x + D]$，且 $x \ge 0$。
* 搜索范围的 $y$ 轴应为 $[min\_y - D, max\_y + D]$，且 $y \ge 0$。
* 由于 $N$ 和 $D$ 很小，这个矩形区域内的点数完全在计算机 1秒 ($10^8$次运算) 的处理能力范围内（假设坐标最大值不超过 200，面积约为 $400 \times 400 = 160,000$ 个点，每个点计算 100 个 AP，总运算量约 $1.6 \times 10^7$，非常安全）。

### 3. 算法步骤
1.  读取输入，存储所有 AP 的信息。同时记录 AP 坐标的最大值和最小值，确定遍历边界。
2.  初始化 `maxSignal = -1`，`bestX = 0`，`bestY = 0`。
3.  使用双重循环遍历 $x$ 和 $y$（按照字典序：先 $x$ 从小到大，再 $y$ 从小到大）。
4.  对于每个点 $(i, j)$：
    * 遍历所有 AP。
    * 计算该 AP 到 $(i, j)$ 的切比雪夫距离 $d$。
    * 如果 $d \le D$，则累加信号 $s / (1 + d)$。
5.  如果当前点的 `totalSignal > maxSignal`，更新 `maxSignal` 和最佳坐标。注意：由于我们是按字典序遍历的，如果 `totalSignal == maxSignal`，不需要更新，因为当前的坐标一定比记录的坐标字典序大（或者相等，不影响）。
6.  输出结果。

### 4. Java 代码实现

```java
import java.util.Scanner;
import java.util.ArrayList;
import java.util.List;

public class BestSignalCoverage {

    static class AP {
        int x, y, s;
        public AP(int x, int y, int s) {
            this.x = x;
            this.y = y;
            this.s = s;
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        if (!scanner.hasNext()) return;

        int N = scanner.nextInt();
        int D = scanner.nextInt();

        List<AP> aps = new ArrayList<>();
        int minX = Integer.MAX_VALUE, maxX = Integer.MIN_VALUE;
        int minY = Integer.MAX_VALUE, maxY = Integer.MIN_VALUE;

        for (int i = 0; i < N; i++) {
            int x = scanner.nextInt();
            int y = scanner.nextInt();
            int s = scanner.nextInt();
            aps.add(new AP(x, y, s));
            
            // 更新边界
            minX = Math.min(minX, x);
            maxX = Math.max(maxX, x);
            minY = Math.min(minY, y);
            maxY = Math.max(maxY, y);
        }

        // 确定搜索范围
        // 最佳点一定在 [minX-D, maxX+D] 和 [minY-D, maxY+D] 之间
        // 且题目要求非负，所以下限至少为 0
        int searchMinX = Math.max(0, minX - D);
        int searchMaxX = maxX + D;
        int searchMinY = Math.max(0, minY - D);
        int searchMaxY = maxY + D;

        int maxSignal = -1;
        int bestX = 0;
        int bestY = 0;

        // 按字典序遍历：先 x 从小到大，再 y 从小到大
        for (int x = searchMinX; x <= searchMaxX; x++) {
            for (int y = searchMinY; y <= searchMaxY; y++) {
                
                int currentTotalSignal = 0;
                
                // 计算该点 (x, y) 处所有 AP 的信号总和
                for (AP ap : aps) {
                    // 切比雪夫距离
                    int dist = Math.max(Math.abs(ap.x - x), Math.abs(ap.y - y));
                    
                    if (dist <= D) {
                        currentTotalSignal += Math.floor(ap.s * 1.0 / (1 + dist));
                    }
                }

                // 更新最大值
                // 因为是按字典序从小到大遍历的，所以当信号严格大于 maxSignal 时才更新
                // 这样能自然保证相同信号强度下，保留的是字典序最小的（最早遇到的）
                if (currentTotalSignal > maxSignal) {
                    maxSignal = currentTotalSignal;
                    bestX = x;
                    bestY = y;
                }
            }
        }

        System.out.println(bestX + " " + bestY);
    }
}
```
