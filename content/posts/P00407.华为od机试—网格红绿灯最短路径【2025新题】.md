## 二、 题目描述

给定一个二维的 $m \times n$ 网格地图（`grids` 二维数组），每个单元格的取值如下：
- `0`：表示空地，可以通行。
- `1`：表示障碍物，不可通行。
- `2`：表示红绿灯。

每一步可以向上下左右四个方向移动一个单元格，耗时 1 秒。
如果移动到的单元格是红绿灯（值为 2），则在该单元格需要额外等待指定的时间才能通过。
红绿灯的具体信息由 `lights` 二维数组给出，例如 `(2, 2, 3)` 表示坐标 `(2, 2)` 处的红绿灯等待时间为 3 秒。

问：从左上角 `(0, 0)` 出发，到达右下角 `(m-1, n-1)` 所需的最短时间。

## 三、 输入描述

1. 第一行输入 `grids` 二维数组，表示地图。内部数据只有 0, 1, 2。 $1 < m, n \le 100$。
2. 第二行输入 `lights` 二维数组，表示红绿灯的位置及等待时间。 $1 < x \le m \times n$。

注意：输入格式为类似 `[[0,1,0],[0,2,1],[0,0,0]]` 的字符串。

## 四、 输出描述

从坐标 `(0, 0)` 到 `(m-1, n-1)` 所需的最短时间。如果没有路径，则返回 -1。

## 五、 用例

### 用例1

**输入**
```text
[[0,1,0],[0,2,1],[0,0,0]]
[[1,1,3]]
```

**输出**
```text
4
```

**说明**
地图如下：
(0,0)=0, (0,1)=1, (0,2)=0
(1,0)=0, (1,1)=2, (1,2)=1
(2,0)=0, (2,1)=0, (2,2)=0

红绿灯在 (1,1)，等待 3 秒。
路径1：(0,0) -> (1,0) -> (2,0) -> (2,1) -> (2,2)。总耗时：1+1+1+1 = 4 秒。
路径2：经过 (1,1)，耗时会更长。
最短时间为 4。

### 用例2

**输入**
```text
[[0,2,0],[0,1,0],[0,0,0]]
[[0,1,5]]
```

**输出**
```text
4
```

## 六、 备注

1. 起点 `(0,0)` 保证为 0。
2. 终点 `(m-1, n-1)` 如果是红绿灯，也需要计算等待时间。
3. 坐标系以 `(行, 列)` 表示。

## 七、 题目解析

### 1. 题目分析
本题属于典型的**带权路径最短路问题**。
- 网格中的移动代价通常为 1（从一个格子移动到相邻格子）。
- 如果目标格子是红绿灯，代价则变为 `1 + waitTime`。
- 因为权重（时间）是非负的且不完全相等，最适合的算法是 **Dijkstra 算法**。

### 2. 解题思路
1.  **数据解析**：输入是嵌套数组格式的字符串，需要解析成二维整数数组 `grid` 和一个方便查询的红绿灯等待时间表（可以用二维数组 `lightMap` 存储）。
2.  **Dijkstra 核心**：
    -   使用优先队列 `PriorityQueue`，存储 `[row, col, currentTime]`，并按 `currentTime` 从小到大排序。
    -   使用 `dist[m][n]` 数组记录到达每个格子的最短时间，初始化为无穷大。
    -   从 `(0, 0)` 开始，初始时间为 0。
    -   每次从队列取出当前时间最小的点，遍历其上下左右邻居。
    -   计算到达邻居的时间：`newTime = currentTime + 1 + (如果是红绿灯 ? waitTime : 0)`。
    -   如果 `newTime` 小于 `dist[neighbor]`，更新并入队。
3.  **边界处理**：
    -   遇到 `1`（障碍物）直接跳过。
    -   超出网格边界跳过。
    -   如果无法到达终点，最终返回 -1。

### 3. 代码实现

#### Java 实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String gridStr = sc.nextLine();
        String lightStr = sc.nextLine();

        int[][] grid = parseMatrix(gridStr);
        int[][] lightData = parseMatrix(lightStr);

        int m = grid.length;
        int n = grid[0].length;

        // 记录每个位置红绿灯的等待时间
        int[][] waitTimes = new int[m][n];
        for (int[] light : lightData) {
            if (light.length >= 3) {
                waitTimes[light[0]][light[1]] = light[2];
            }
        }

        int result = dijkstra(grid, waitTimes, m, n);
        System.out.println(result);
    }

    private static int dijkstra(int[][] grid, int[][] waitTimes, int m, int n) {
        int[][] dist = new int[m][n];
        for (int[] row : dist) Arrays.fill(row, Integer.MAX_VALUE);
        
        // PriorityQueue 存储 {row, col, time}
        PriorityQueue<int[]> pq = new PriorityQueue<>(Comparator.comparingInt(a -> a[2]));
        
        dist[0][0] = 0;
        pq.offer(new int[]{0, 0, 0});

        int[][] dirs = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};

        while (!pq.isEmpty()) {
            int[] curr = pq.poll();
            int r = curr[0];
            int c = curr[1];
            int t = curr[2];

            if (t > dist[r][c]) continue;
            if (r == m - 1 && c == n - 1) return t;

            for (int[] d : dirs) {
                int nr = r + d[0];
                int nc = c + d[1];

                if (nr >= 0 && nr < m && nc >= 0 && nc < n && grid[nr][nc] != 1) {
                    int weight = 1;
                    if (grid[nr][nc] == 2) {
                        weight += waitTimes[nr][nc];
                    }
                    
                    if (dist[r][c] + weight < dist[nr][nc]) {
                        dist[nr][nc] = dist[r][c] + weight;
                        pq.offer(new int[]{nr, nc, dist[nr][nc]});
                    }
                }
            }
        }

        return dist[m - 1][n - 1] == Integer.MAX_VALUE ? -1 : dist[m - 1][n - 1];
    }

    // 简单的解析方法，处理类似 [[0,1],[0,2]] 格式
    private static int[][] parseMatrix(String str) {
        str = str.substring(1, str.length() - 1); // 去掉最外层 []
        List<int[]> list = new ArrayList<>();
        int i = 0;
        while (i < str.length()) {
            if (str.charAt(i) == '[') {
                int j = i;
                while (str.charAt(j) != ']') j++;
                String rowStr = str.substring(i + 1, j);
                String[] parts = rowStr.split(",");
                int[] row = new int[parts.length];
                for (int k = 0; k < parts.length; k++) {
                    row[k] = Integer.parseInt(parts[k].trim());
                }
                list.add(row);
                i = j + 1;
            } else {
                i++;
            }
        }
        return list.toArray(new int[0][]);
    }
}
```

### 4. 复杂度分析
-   **时间复杂度**：$O(E \log V)$，其中 $V$ 是格点总数 ($m \times n$)，$E$ 是边数（每个点最多 4 条边）。在本题中约为 $O(mn \log(mn))$。
-   **空间复杂度**：$O(mn)$，用于存储 `dist` 数组、`waitTimes` 数组以及优先队列。
