## 一、 题目描述

小杨申请了一个保密柜，但是他忘记了密码。只记得密码都是数字，而且所有数字都是不重复的。

请你根据他记住的数字范围和密码的最小数字数量，帮他算下有哪些可能的组合，规则如下：
1. 输出的组合都是从可选的数字范围中选取的，且不能重复；
2. 输出的密码数字要按照从小到大的顺序排列，密码组合需要按照字母顺序，从小到大的顺序排序。
3. 输出的每一个组合的数字的数量要大于等于密码最小数字数量；
4. 如果可能的组合为空，则返回“None”

## 二、输入描述

1. 输入的第一行是可能的密码数字列表，数字间以半角逗号分隔。
2. 输入的第二行是密码最小数字数量。

## 三、输出描述

可能的密码组合，每种组合显示成一行，每个组合内部的数字以半角逗号分隔，从小到大的顺序排列。
输出的组合间需要按照字典序排序。
比如：
`2,3,4` 放到 `2,4` 的前面。

## 四、用例

### 用例1

**输入**
```
2,3,4
2
```

**输出**
```
2,3
2,3,4
2,4
3,4
```

**说明**
最小密码数量是两个，可能有三种组合：2,3；2,4；3,4。
三个密码有一种：2,3,4。
按照字典序排列后输出如上。

### 用例2

**输入**
```
1,2
3
```

**输出**
```
None
```

## 五、备注
数字范围和最小数量均为正整数。

## 六、题目解析

### 1. 题目分析
本题要求从给定的一组不重复数字中，选出长度大于等于 $k$ 的所有组合。
关键约束：
- 组合内的数字必须升序排列。
- 所有组合必须按字典序排列输出。
- 若无满足条件的组合，输出 `None`。

### 2. 解题思路
1. **排序输入**：由于要求组合内部升序且组合间字典序，我们首先对输入的数字列表进行从小到大的排序。
2. **回溯生成子集**：使用 DFS（回溯法）遍历所有可能的子集。
   - 排序后的数组进行 DFS，能够天然地保证生成的组合在字典序上是有序的。
   - 例如：对于 `[2, 3, 4]`，DFS 路径为：
     - `2` -> `2,3` -> `2,3,4`
     - `2` -> `2,4`
     - `3` -> `3,4`
     - `4`
3. **过滤长度**：在搜索过程中，如果当前记录的路径（组合）长度 $\ge$ 最小长度 $k$，则将其记录到结果集中。
4. **输出格式**：将结果按行输出，注意 `None` 的处理。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;
import java.util.stream.Collectors;

public class Main {
    static List<String> results = new ArrayList<>();
    static int minLength;
    static int[] nums;

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextLine()) return;

        // 1. 输入解析
        String line1 = sc.nextLine();
        if (line1.isEmpty()) {
            System.out.println("None");
            return;
        }
        nums = Arrays.stream(line1.split(","))
                     .mapToInt(Integer::parseInt)
                     .toArray();
        
        if (!sc.hasNextInt()) return;
        minLength = sc.nextInt();

        // 2. 排序，为回溯和字典序做准备
        Arrays.sort(nums);

        // 3. 回溯搜索
        backtrack(0, new ArrayList<>());

        // 4. 输出结果
        if (results.isEmpty()) {
            System.out.println("None");
        } else {
            for (String res : results) {
                System.out.println(res);
            }
        }
    }

    /**
     * @param index 当前遍历到的数组下标
     * @param path  当前组合已选取的数字
     */
    private static void backtrack(int index, List<Integer> path) {
        // 如果当前路径长度符合要求，加入结果集
        if (path.size() >= minLength) {
            results.add(path.stream()
                    .map(String::valueOf)
                    .collect(Collectors.joining(",")));
        }

        // 递归终止条件
        if (index == nums.length) {
            return;
        }

        // 标准子集生成回溯
        for (int i = index; i < nums.length; i++) {
            path.add(nums[i]);
            backtrack(i + 1, path);
            path.remove(path.size() - 1); // 回溯
        }
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(2^n)$，其中 $n$ 是输入数字的数量。在最坏情况下，我们需要遍历所有可能的子集。
- **空间复杂度**：$O(2^n \cdot n)$，用于存储结果集，以及递归调用的深度 $O(n)$。
