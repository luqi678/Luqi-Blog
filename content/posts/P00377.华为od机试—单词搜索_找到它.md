## 二、 题目描述

找到它是一个小游戏，你需要在一个矩阵中找到给定的单词。

假设给定单词 **HELLOWORLD**，在矩阵中只要能找到 H->E->L->L->O->W->O->R->L->D 连成的单词，就算通过。

**注意：**
1. 区分英文字母大小写。
2. 您只能上下左右行走。
3. **不能走回头路**（即在同一条单词路径中，同一个坐标的字母不能重复使用）。
4. 题目保证在整个矩阵中给定单词 W 只会出现一次。

## 二、 输入描述

1. 输入第 1 行包含两个整数 $n, m$ ($0 < n, m < 21$) 分别表示 $n$ 行 $m$ 列的矩阵。
2. 第 2 行是长度不超过 100 的单词 $W$。
3. 从第 3 行到第 $n+2$ 行是包含大小写英文字母的长度为 $m$ 的字符串矩阵。

## 三、 输出描述

* 如果能在矩阵中连成给定的单词，则输出给定单词首字母在矩阵中的位置（行 跨格 列，均为从 1 开始计数）。
* 否则输出 “NO”。

## 四、 用例

### 用例1

**输入**
```text
5 5
HELLOWORLD
CPUCY
EKLQH
CHELL
LROWO
DGRBC
```

**输出**
```text
3 2
```

**说明**
矩阵中从第3行第2列的 'H' 开始，可以找到 HELLOWORLD。

### 用例2

**输入**
```text
2 2
HELLO
HE
LL
```

**输出**
```text
NO
```

## 五、 备注

* 矩阵规模较小（最大 $20 \times 20$），适合使用搜索算法。
* 单词长度最大为 100。

## 六、 题目解析

### 1. 题目分析
本题是一个经典的**矩阵搜索问题**。我们需要在 $n \times m$ 的字符矩阵中寻找一个特定的字符串 $W$。搜索过程中只能向上下左右四个方向移动，且不能重复访问已经包含在当前路径中的单元格。

### 2. 解题思路
1.  **遍历起点**：遍历矩阵的每一个位置 $(i, j)$。如果 `matrix[i][j]` 等于单词 $W$ 的第一个字符，则从该位置开始进行深度优先搜索 (DFS)。
2.  **DFS 过程**：
    *   传入当前坐标 $(r, c)$ 和当前匹配到单词的索引 `k`。
    *   **终止条件**：如果 `k` 等于单词长度减 1，说明已经找齐了所有字符，返回 `true`。
    *   **标记访问**：为了防止“走回头路”，需要使用一个 `visited` 二维布尔数组记录当前路径已访问的坐标。
    *   **方向探索**：尝试向上、下、左、右四个方向移动。如果新坐标 $(nr, nc)$ 在矩阵范围内，且未被访问，且 `matrix[nr][nc]` 等于 `W[k+1]`，则递归搜索。
    *   **回溯**：如果当前路径无法匹配成功，需要将 `visited[r][c]` 重置为 `false`，以便其他路径搜索。
3.  **输出结果**：一旦 DFS 返回 `true`，立即输出当前起点坐标 $(i+1, j+1)$ 并结束程序。如果遍历完所有起点仍未找到，输出 "NO"。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;

public class Main {
    static int n, m;
    static String target;
    static char[][] matrix;
    static boolean[][] visited;
    static int[][] dirs = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) return;
        n = sc.nextInt();
        m = sc.nextInt();
        target = sc.next();
        matrix = new char[n][m];
        for (int i = 0; i < n; i++) {
            matrix[i] = sc.next().toCharArray();
        }

        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                // 如果首字母匹配，开始DFS
                if (matrix[i][j] == target.charAt(0)) {
                    visited = new boolean[n][m];
                    if (dfs(i, j, 0)) {
                        System.out.println((i + 1) + " " + (j + 1));
                        return;
                    }
                }
            }
        }

        System.out.println("NO");
    }

    /**
     * @param r 当前行
     * @param c 当前列
     * @param index 当前匹配到单词的第几个字符(0-based)
     * @return 是否找到完整单词
     */
    private static boolean dfs(int r, int c, int index) {
        // 基础判断：如果当前索引已经是单词最后一位，说明找到了
        if (index == target.length() - 1) {
            return true;
        }

        visited[r][c] = true;

        // 向四个方向递归
        for (int[] dir : dirs) {
            int nr = r + dir[0];
            int nc = c + dir[1];

            if (nr >= 0 && nr < n && nc >= 0 && nc < m 
                && !visited[nr][nc] 
                && matrix[nr][nc] == target.charAt(index + 1)) {
                
                if (dfs(nr, nc, index + 1)) {
                    return true;
                }
            }
        }

        // 回溯：撤销访问标记
        visited[r][c] = false;
        return false;
    }
}
```

### 4. 复杂度分析

*   **时间复杂度**：$O(N \times M \times 3^L)$，其中 $N, M$ 是矩阵行列数，$L$ 是单词长度。每个起点出发最多有四个方向，除去回头的方向，分支因子约为 3。鉴于 $N, M \le 20$ 且题目保证单词只出现一次，实际搜索空间远小于理论上限。
*   **空间复杂度**：$O(N \times M)$，主要用于存储矩阵、访问状态数组以及递归调用的栈深度（最大为 $L$）。
