## 一、 题目描述

中秋节，公司分月饼，`m` 个员工，买了 `n` 个月饼，`m <= n`，每个员工至少分 1 个月饼，但可以分多个。

单人分到最多月饼的个数为 `Max1`，单人分到第二多月饼的个数是 `Max2`，满足 `Max1 - Max2 <= 3`。
同理，单人分到第 `n-1` 多月饼的个数是 `Max(n-1)`，单人分到第 `n` 多月饼的个数是 `Max(n)`，满足 `Max(n-1) - Max(n) <= 3`。

请问有多少种分月饼的方法？

## 二、输入描述

第一行输入 `m` `n`，表示 `m` 个员工，`n` 个月饼，`m <= n`。

## 三、输出描述

输出有多少种月饼分法。

## 四、备注

- 题目隐含了对分配方案排序后的约束。
- 所谓的分法不考虑员工的具体身份（即 `1, 3` 和 `3, 1` 被视为同一种分法），本质是整数拆分问题。
- 约束条件总结为：将 `n` 个月饼分给 `m` 个人，分得的数量序列排序后（设为 $a_1 \ge a_2 \ge ... \ge a_m$），需满足相邻元素的差值不超过 3，且每个元素至少为 1。

## 五、用例

| 输入     | 输出  | 说明                                                                                                                                                                                                                                 |
| :----- | :-- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `2 4`  | `2` | **方案1**：`4 = 1 + 3` (差值为2，满足<=3)<br>**方案2**：`4 = 2 + 2` (差值为0，满足<=3)<br>共 2 种方案。                                                                                                                                                   |
| `3 12` | `6` | 满足条件的分配方案有：<br>1. `4, 4, 4`<br>2. `5, 4, 3`<br>3. `5, 5, 2` (5-5=0, 5-2=3, ok)<br>4. `6, 3, 3` (6-3=3, 3-3=0, ok)<br>5. `6, 4, 2` (6-4=2, 4-2=2, ok)<br>6. `6, 5, 1` (6-5=1, 5-1=4 > 3, **不满足**，排除)<br>*注：这里仅为举例分析逻辑，具体计算需通过算法求解。* |

## 六、题目解析

### 核心思路

这是一个典型的**整数划分**问题，并带有特定的限制条件（相邻差值 $\le 3$）。由于 $n$ 和 $m$ 的范围通常较小（在机考题目中，此类递归题通常 $m, n \le 30$ 或稍大），我们可以使用**递归（DFS）**来解决。

**逻辑步骤：**
1.  由于题目说明 `1+3` 和 `3+1` 算同一种，我们默认按照**降序**（或升序）来分配，以避免重复统计。假设分配序列为 $a_1, a_2, ..., a_m$，且 $a_1 \ge a_2 \ge ... \ge a_m$。
2.  约束条件转化为：
    * $a_i \ge 1$ (每个人至少一个)
    * $a_i \ge a_{i+1}$ (保持降序)
    * $a_i - a_{i+1} \le 3$ (相邻差值限制)
3.  **剪枝策略**（非常重要，否则会超时）：
    * **上界剪枝**：剩余的月饼如果每个人都分当前最大的允许值，总和还不够 $n$，则当前路径无效。
    * **下界剪枝**：剩余的月饼如果每个人都分最小的允许值（例如 1 或 当前值-3），总和超过了 $n$，则当前路径无效。

### 算法实现 (Java)

```java
import java.util.Scanner;

public class Main {
static int m, n;  
static int count = 0;  
  
public static void main(String[] args) {  
    Scanner sc = new Scanner(System.in);  
    while (sc.hasNext()) {  
        m = sc.nextInt(); // 员工数  
        n = sc.nextInt(); // 月饼数  
  
        if (m > n) {  
            System.out.println(0);  
            return;  
        }  
  
        count = 0;  
        // 递归入口：  
        // 第1个人开始分，剩余n个，preVal传n（对于第1个人来说preVal不生效，但作为初始上界参考）  
        distribute(1, n, n);  
        System.out.println(count);  
    }  
}  
  
/**  
 * 递归分配  
 * @param personIdx 当前正在给第几个人分 (1 ~ m)  
 * @param remain    剩余月饼数量  
 * @param preVal    上一个人分到的月饼数量  
 */  
public static void distribute(int personIdx, int remain, int preVal) {  
    // Base Case: 已经分到了最后一个人  
    if (personIdx == m) {  
        // 最后一个人直接拿走剩下的所有月饼  
        // 必须满足：  
        // 1. 剩下的月饼数 <= 上一个人 (保持降序)  
        // 2. 上一个人 - 剩下的月饼数 <= 3 (满足差值约束)  
        // 3. 剩下的月饼数 >= 1 (题目要求至少分1个)  
        // 注意：如果 m=1，personIdx也是1，此时没有上一个人，前两个条件自动满足或忽略  
        if (remain >= 1) {  
            if (personIdx == 1) {  
                count++;  
            } else if (remain <= preVal && (preVal - remain) <= 3) {  
                count++;  
            }  
        }  
        return;  
    }  
  
    // --- 确定当前人能拿的范围 ---  
    // 上界 (Max)：  
    // 1. 不能超过剩余总数减去后面人最少拿的量（后面至少每人1个）：remain - (m - personIdx)  
    // 2. 如果不是第1个人，不能超过上一个人 preVal（保持降序）  
    int max = remain - (m - personIdx);  
    if (personIdx > 1) {  
        max = Math.min(preVal, max);  
    }  
  
    // 下界 (Min)：  
    // 1. 至少为 1    // 2. 如果不是第1个人，受差值限制：current >= preVal - 3  
    int min = 1;  
    if (personIdx > 1) {  
        min = Math.max(1, preVal - 3);  
    }  
  
    // --- 剪枝与递归 ---    for (int i = max; i >= min; i--) {  
        // 剪枝判断：  
        // 如果当前选 i，剩下的人数 (m - personIdx) 即使每个人都拿最大的 i，  
        // 总和 (m - personIdx) * i 依然小于剩余月饼 (remain - i)，  
        // 说明当前选的 i 太小了，导致剩下的月饼分不完（不满足降序条件），  
        // 既然 i 都不行，比 i 更小的数更不行，直接 break。  
        if ((remain - i) > (m - personIdx) * i) {  
            break;  
        }  
  
        distribute(personIdx + 1, remain - i, i);  
    }  
}
}
```
