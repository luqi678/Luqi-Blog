## 一、 题目描述

学校组织活动，将学生排成一个矩形方阵。
请在矩形方阵中找到最大的位置相连的男生数量。
这个相连位置在一个直线上，方向可以是水平的（Horizontal），垂直的（Vertical），对角线的（Diagonal，从左上到右下）或者呈反对角线的（Anti-diagonal，从右上到左下）。

注：学生个数不会超过 10000。男生用字符 'M' 表示，女生用字符 'F' 表示。

## 二、输入描述

1. 输入的第一行为矩阵的行数 $n$ 和列数 $m$，以逗号分隔。
2. 接下来的 $n$ 行为矩阵元素，每行包含 $m$ 个元素，元素间用“,”分隔。

## 三、输出描述

输出一个整数，表示矩阵中最长的位置相连的男生个数。

## 四、用例

### 用例1
**输入**
```text
3,4
F,M,M,F
F,M,M,F
F,F,F,M
```
**输出**
```text
3
```
**说明**：在最后一行最后一列的 'M' 与倒数第二行第三列、倒数第三行第二列构成反对角线上的 3 个相连男生。

### 用例2
**输入**
```text
1,2
M,M
```
**输出**
```text
2
```

### 用例3
**输入**
```text
2,1
M
F
```
**输出**
```text
1
```

## 五、备注
*   学生总个数 $\le 10000$。
*   需要考虑四种方向：水平、垂直、正对角线（\）、反对角线（/）。

## 六、题目解析

### 1. 题目分析
本题是一道典型的矩阵遍历模拟题。我们需要在一个二维网格中寻找最长的连续 'M' 序列。题目明确指出“相连位置在一个直线上”，并给出了四个搜索方向。

### 2. 解题思路
1.  **数据读取**：首先解析行数和列数，然后将矩阵存入二维字符数组中。
2.  **方向定义**：定义四个方向的偏移量：
    *   水平：`(0, 1)`
    *   垂直：`(1, 0)`
    *   正对角线（左上到右下）：`(1, 1)`
    *   反对角线（右上到左下）：`(1, -1)`
3.  **遍历与计数**：
    *   遍历矩阵中的每一个点 `(i, j)`。
    *   如果该点是 'M'，则分别沿四个方向进行探测。
    *   **优化**：为了避免重复计算，可以进行判断：如果当前点 `(i, j)` 在某个方向的前一个点已经是 'M'，说明当前点已经作为序列的一部分被计算过了，可以跳过（例如，在检查水平方向时，如果 `matrix[i][j-1]` 也是 'M'，则不需要从 `matrix[i][j]` 重新开始计数）。
4.  **维护最大值**：在所有搜索过程中，记录出现的最大连续长度。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextLine()) return;

        // 1. 读取行数和列数
        String[] dims = sc.nextLine().split(",");
        int rows = Integer.parseInt(dims[0]);
        int cols = Integer.parseInt(dims[1]);

        String[][] matrix = new String[rows][cols];
        for (int i = 0; i < rows; i++) {
            if (sc.hasNextLine()) {
                matrix[i] = sc.nextLine().split(",");
            }
        }

        int maxBoys = 0;

        // 四个搜索方向：水平、垂直、正对角线(\)、反对角线(/)
        int[][] directions = {
            {0, 1},  // 水平
            {1, 0},  // 垂直
            {1, 1},  // 正对角线
            {1, -1}  // 反对角线
        };

        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                // 只从男生 'M' 开始探测
                if (matrix[i][j].equals("M")) {
                    for (int[] dir : directions) {
                        int dr = dir[0];
                        int dc = dir[1];

                        // 优化：检查当前点的前一个点是否也是 'M'
                        // 如果是，说明这个序列已经从前一个点探测过了
                        int prevR = i - dr;
                        int prevC = j - dc;
                        if (prevR >= 0 && prevR < rows && prevC >= 0 && prevC < cols) {
                            if (matrix[prevR][prevC].equals("M")) {
                                continue;
                            }
                        }

                        // 开始计算当前方向的连续长度
                        int count = 0;
                        int currR = i;
                        int currC = j;
                        while (currR >= 0 && currR < rows && currC >= 0 && currC < cols 
                               && matrix[currR][currC].equals("M")) {
                            count++;
                            currR += dr;
                            currC += dc;
                        }
                        maxBoys = Math.max(maxBoys, count);
                    }
                }
            }
        }

        System.out.println(maxBoys);
    }
}
```

### 4. 复杂度分析
*   **时间复杂度**：$O(R \times C)$。虽然有四重循环（两层遍历矩阵，一层遍历方向，一层 `while` 循环），但由于优化逻辑（跳过中间节点），每个节点的每个方向最多只会被访问两次。
*   **空间复杂度**：$O(R \times C)$，用于存储矩阵内容。
