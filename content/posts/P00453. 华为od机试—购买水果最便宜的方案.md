## 一、 题目描述

有 $m$ 个水果超市在 $1 \sim n$ 个小时的不同时间段提供不同价格的打折水果，如果某餐厅每个小时都要新采购一种水果给餐厅使用的话， 请选出 $n$ 个小时内，采购水果的最便宜的花费总和。 （假设 $m$ 个超市打折时间段可以覆盖 $n$ 小时）

## 二、输入描述

1.  第一行：整数 $N$ — 总小时数。
2.  第二行：整数 $m$ — 水果超市个数。
3.  接下来 $m$ 行：每行包含 3 个整数 $X[0], X[1], X[2]$。
    * $X[0]$：开始时间（包含）。
    * $X[1]$：结束时间（包含）。
    * $X[2]$：价格。

**范围提示:** $1 \le n < 2^{10}$ （1024）。

## 三、输出描述

采购水果的最便宜的花费总和。

## 四、用例

### 示例 1
**输入：**
```text
4 
6 
2 3 10 
2 4 20 
1 3 15 
1 4 25 
3 4 8 
1 4 16 
```
**输出：**
```text
41
```
**解释：**
* **第 1 小时**：
    * 覆盖该小时的超市有：[1,3]价15，[1,4]价25，[1,4]价16。
    * 最低价为 **15**。
* **第 2 小时**：
    * 覆盖该小时的超市有：[2,3]价10，[2,4]价20，[1,3]价15，[1,4]价25，[1,4]价16。
    * 最低价为 **10**。
* **第 3 小时**：
    * 覆盖该小时的超市有：[2,3]价10，[2,4]价20，[1,3]价15，[1,4]价25，[3,4]价8，[1,4]价16。
    * 最低价为 **8**。
* **第 4 小时**：
    * 覆盖该小时的超市有：[2,4]价20，[1,4]价25，[3,4]价8，[1,4]价16。
    * 最低价为 **8**。
* **总和**：15 + 10 + 8 + 8 = **41**。

## 五、备注

* 题目保证所有 $m$ 个超市的时间段并集一定能覆盖 $1$ 到 $n$ 的所有小时，不需要处理无法购买的情况。
* 时间复杂度要求不高，因为 $N$ 较小 ($<1024$)。

## 六、题目解析

### 1. 题目分析
这是一个典型的**贪心**或**模拟**问题。
我们需要在 $1$ 到 $n$ 的每一个小时都买水果，为了总花费最便宜，我们只需要保证**每一个小时买到的水果都是当前小时所有可选方案中最便宜的**即可。即局部最优导致全局最优。

### 2. 解题思路
1.  **初始化容器**：
    创建一个长度为 $N+1$ 的数组 `min_costs`，用于存储第 $1$ 小时到第 $N$ 小时每一小时的最低价格。
    将数组所有元素初始化为一个极大值（如 `Integer.MAX_VALUE`），表示尚未找到报价。

2.  **处理输入**：
    遍历每一个超市的报价信息 `start, end, price`。
    对于该报价覆盖的每一个小时 $t$（从 `start` 到 `end`），尝试更新 `min_costs[t]`：
    $$min\_costs[t] = \min(min\_costs[t], price)$$

3.  **计算结果**：
    遍历 `min_costs` 数组（从下标 1 到 $N$），将所有小时的最低价格累加，即为最终结果。

### 3. 复杂度分析
* **时间复杂度**：$O(m \times n)$。最坏情况下，每个超市都覆盖全时段。由于 $n < 1024$，计算量非常小，完全满足要求。
* **空间复杂度**：$O(n)$。用于存储每个小时的最低价。

### 4. 代码实现

#### Java 实现

```java
import java.util.Arrays;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // 读取总小时数 N
        if (!sc.hasNext()) return;
        int n = sc.nextInt();
        
        // 读取超市个数 M
        int m = sc.nextInt();
        
        // 定义数组存储每个小时的最小花费，下标 1~n 对应第 1~n 小时
        int[] minCosts = new int[n + 1];
        // 初始化为最大值，方便后续取较小值
        Arrays.fill(minCosts, Integer.MAX_VALUE);
        
        // 处理每一个超市的报价
        for (int i = 0; i < m; i++) {
            int start = sc.nextInt();
            int end = sc.nextInt();
            int price = sc.nextInt();
            
            // 确保时间段在有效范围内 (1 ~ n)
            // 题目虽暗示覆盖，但做边界保护是好习惯
            int validStart = Math.max(1, start);
            int validEnd = Math.min(n, end);
            
            // 更新该时间段内每个小时的最低价格
            for (int t = validStart; t <= validEnd; t++) {
                if (price < minCosts[t]) {
                    minCosts[t] = price;
                }
            }
        }
        
        // 计算总和
        int totalCost = 0;
        for (int t = 1; t <= n; t++) {
            // 如果某小时没有被覆盖（理论上题目说会覆盖），这里会是MAX_VALUE，需注意溢出风险
            // 但题目保证覆盖，此处直接累加即可
            if (minCosts[t] != Integer.MAX_VALUE) {
                totalCost += minCosts[t];
            }
        }
        
        System.out.println(totalCost);
    }
}
```

#### Python 实现

```python
import sys

def solve():
    try:
        input_data = sys.stdin.read().split()
        if not input_data:
            return
        
        iterator = iter(input_data)
        
        n = int(next(iterator))
        m = int(next(iterator))
        
        # 初始化花费数组，索引 0 不使用，1~n 对应各小时
        # 使用一个足够大的数初始化
        min_costs = [float('inf')] * (n + 1)
        
        for _ in range(m):
            start = int(next(iterator))
            end = int(next(iterator))
            price = int(next(iterator))
            
            # 边界修正
            start_idx = max(1, start)
            end_idx = min(n, end)
            
            for t in range(start_idx, end_idx + 1):
                if price < min_costs[t]:
                    min_costs[t] = price
                    
        total_cost = 0
        for t in range(1, n + 1):
            total_cost += min_costs[t]
            
        print(total_cost)
        
    except StopIteration:
        pass

if __name__ == "__main__":
    solve()
```

#### C++ 实现

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <climits>

using namespace std;

int main() {
    int n, m;
    if (cin >> n >> m) {
        // 使用 vector 存储每小时的最小花费，初始化为最大整数
        vector<int> min_costs(n + 1, INT_MAX);
        
        for (int i = 0; i < m; ++i) {
            int start, end, price;
            cin >> start >> end >> price;
            
            // 边界处理
            int valid_start = max(1, start);
            int valid_end = min(n, end);
            
            for (int t = valid_start; t <= valid_end; ++t) {
                if (price < min_costs[t]) {
                    min_costs[t] = price;
                }
            }
        }
        
        long long total_cost = 0;
        for (int t = 1; t <= n; ++t) {
            total_cost += min_costs[t];
        }
        
        cout << total_cost << endl;
    }
    return 0;
}
```
