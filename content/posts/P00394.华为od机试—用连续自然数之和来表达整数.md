## 一、 题目描述

一个整数可以由连续的自然数之和来表示。
给定一个整数，计算该整数有几种连续自然数之和的表达式，且打印出每种表达式。

## 二、输入描述

一个目标整数 $T$ ($1 \le T \le 1000$)

## 三、输出描述

该整数的所有表达式和表达式的个数。如果有多种表达式，输出要求为：
1. 自然数个数最少的表达式优先输出。
2. 每个表达式中按自然数递增的顺序输出，具体的格式参见样例。
3. 在每个测试数据结束时，输出一行 ”Result:X”，其中 X 是最终的表达式个数。

## 四、用例
### 用例1
输入：
```
9
```
输出：
```
9=9
9=4+5
9=2+3+4
Result:3
```

### 用例2
输入：
```
10
```
输出：
```
10=10
10=1+2+3+4
Result:2
```

## 五、备注
1. 自然数是指：$1, 2, 3, 4, ...$ （本题语境下通常指正整数）。
2. $T$ 的取值范围较小（最大1000），可以考虑暴力枚举或数学推导。

## 六、题目解析
### 1. 题目分析
本题要求将一个整数 $T$ 分解为若干个连续自然数的和。
关键点：
- 连续自然数：意味着如果起始数字是 $a$，项数是 $n$，则序列为 $a, a+1, \dots, a+n-1$。
- 输出顺序：按项数 $n$ 从小到大排序。
- 格式要求：`T=a+...+(a+n-1)`。

### 2. 解题思路
设起始项为 $a$（$a \ge 1$），项数为 $n$（$n \ge 1$）。
根据等差数列求和公式，总和 $S$ 为：
$$S = \frac{n(2a + n - 1)}{2}$$
题目给定 $S = T$，则有：
$$2T = n(2a + n - 1)$$

我们需要找到所有满足条件的 $(a, n)$ 组合：
1. $n$ 是 $2T$ 的因数。
2. $2a = \frac{2T}{n} - n + 1$。
3. $a$ 必须是正整数，即 $\frac{2T}{n} - n + 1$ 必须是正偶数。

因为要求按项数 $n$ 从小到大输出，我们可以直接从 $n=1$ 开始枚举 $n$。
$n$ 的上限：当 $a=1$ 时，$n$ 最大。此时 $T = \frac{n(n+1)}{2}$，大约 $n \approx \sqrt{2T}$。对于 $T=1000$，$n$ 最大约为 44。

### 3. 代码实现
#### Java8实现

```java
import java.util.Scanner;
import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) return;
        int T = sc.nextInt();
        
        int count = 0;
        // n 代表连续自然数的个数，要求个数少的先输出，所以从1开始增加
        // 根据公式 2T = n(2a + n - 1)，n 不会超过 sqrt(2T) 太多
        for (int n = 1; n * (n + 1) / 2 <= T + n; n++) {
            // 2a = (2T / n) - n + 1
            // 必须满足 2T 能被 n 整除
            if ((2 * T) % n == 0) {
                int val = (2 * T) / n - n + 1;
                // val 必须是偶数且大于0 (因为 a = val / 2 >= 1)
                if (val > 0 && val % 2 == 0) {
                    int a = val / 2;
                    printExpression(T, a, n);
                    count++;
                }
            }
        }
        System.out.println("Result:" + count);
    }

    private static void printExpression(int T, int a, int n) {
        StringBuilder sb = new StringBuilder();
        sb.append(T).append("=");
        for (int i = 0; i < n; i++) {
            sb.append(a + i);
            if (i < n - 1) {
                sb.append("+");
            }
        }
        System.out.println(sb.toString());
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(\sqrt{T})$。我们只需要枚举项数 $n$，而 $n$ 的最大值约为 $\sqrt{2T}$。对于 $T=1000$，循环次数极少。
- **空间复杂度**：$O(1)$。除了存储输入和少量中间变量外，不需要额外的空间。
