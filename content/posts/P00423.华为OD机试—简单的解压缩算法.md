## 一、 题目描述

现需要实现一种算法，能将一组压缩字符串还原成原始字符串，还原规则如下：

1. **字符后面加数字N**，表示重复字符N次。例如：压缩内容为 `A3`，表示原始字符串为 `AAA`。
2. **花括号中的字符串加数字N**，表示花括号中的字符重复N次。例如压缩内容为 `{AB}3`，表示原始字符串为 `ABABAB`。
3. **字符加N和花括号后面加N，支持任意的嵌套**，包括互相嵌套，例如：压缩内容可以为 `{A3B1{C}3}3`。

## 二、输入描述

输入一行压缩后的字符串，字符串仅由大写字母、数字和花括号 `{}` 组成。

## 三、输出描述

输出解压缩后的原始字符串。

## 四、用例

### 用例1

**输入**：
`{A3B1{C}3}3`

**输出**：
`AAABCCCAAABCCCAAABCCC`

**说明**：
`{A3B1{C}3}3` 代表 `A` 字符重复 3 次，`B` 字符重复 1 次，花括号中的 `C` 字符重复 3 次（即 `CCC`），拼接后得到 `AAABCCC`。最后最外层花括号表示将 `AAABCCC` 重复 3 次。

### 用例2

**输入**：
`A11B2{C3}2`

**输出**：
`AAAAAAAAAAABBC CCCCC`

## 五、备注

1. 数字 $N$ 的范围通常为正整数（可能包含多位数字，如 11）。
2. 如果字符或括号后没有数字，通常默认为重复 1 次（根据用例分析，一般题目会给出明确数字，但逻辑上需兼容）。
3. 嵌套深度没有明确限制。

## 六、题目解析

### 1. 题目分析

本题属于经典的**字符串解析（Parsing）**问题。由于存在花括号 `{}` 的嵌套结构，处理这类问题有两种常用方法：
1. **栈（Stack）**：模拟递归过程，遇到 `{` 进栈，遇到 `}` 出栈。
2. **递归（Recursion）**：定义一个递归函数来解析当前层级的字符串，遇到 `{` 时递归调用。

### 2. 解题思路

采用**递归**或**全局索引扫描**的方法最为直观：

- **定义一个全局索引 `i`**，用于遍历字符串。
- **处理字母**：遇到大写字母，先记录该字母，然后向后查找紧跟的数字 $N$。将该字母重复 $N$ 次加入结果。
- **处理左花括号 `{`**：
  1. 跳过 `{`。
  2. 递归调用解析函数，获取花括号内部被还原后的子串。
  3. 此时索引 `i` 应该停在对应的 `}` 处，跳过 `}`。
  4. 向后查找紧跟的数字 $N$。将子串重复 $N$ 次加入结果。
- **处理右花括号 `}`**：直接返回当前层级构造的字符串。
- **处理数字**：编写一个辅助函数，从当前位置读取完整的数字（可能有多位）。

### 3. 代码实现

#### Java 实现

```java
import java.util.Scanner;

public class Main {
    private static String src;
    private static int ptr;

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (sc.hasNextLine()) {
            src = sc.nextLine();
            ptr = 0;
            System.out.println(decompress());
        }
    }

    private static String decompress() {
        StringBuilder sb = new StringBuilder();

        while (ptr < src.length()) {
            char c = src.charAt(ptr);

            if (c == '{') {
                ptr++; // 跳过 '{'
                String sub = decompress(); // 递归处理括号内部内容
                // 此时 ptr 应该在 '}' 之后，或者当前 c 为 '}'
                if (ptr < src.length() && src.charAt(ptr) == '}') {
                    ptr++; // 跳过 '}'
                }
                int repeat = getNumber();
                for (int i = 0; i < repeat; i++) {
                    sb.append(sub);
                }
            } else if (c == '}') {
                // 遇到右括号，说明当前层级处理结束，由外层逻辑跳过此 '}' 并获取重复次数
                return sb.toString();
            } else if (Character.isLetter(c)) {
                ptr++;
                int repeat = getNumber();
                for (int i = 0; i < repeat; i++) {
                    sb.append(c);
                }
            } else {
                ptr++;
            }
        }
        return sb.toString();
    }

    // 获取紧跟在后面的数字，如果没有数字则默认为 1
    private static int getNumber() {
        if (ptr >= src.length() || !Character.isDigit(src.charAt(ptr))) {
            return 1;
        }
        int num = 0;
        while (ptr < src.length() && Character.isDigit(src.charAt(ptr))) {
            num = num * 10 + (src.charAt(ptr) - '0');
            ptr++;
        }
        return num;
    }
}
```

### 4. 复杂度分析

- **时间复杂度**：$O(L_{out})$，其中 $L_{out}$ 是解压后原始字符串的长度。因为我们需要构造并输出整个字符串，每个字符都被访问并添加。
- **空间复杂度**：$O(L_{out} + D)$，其中 $D$ 是花括号嵌套的深度（递归调用栈的深度），$L_{out}$ 是存储最终结果所需的空间。
