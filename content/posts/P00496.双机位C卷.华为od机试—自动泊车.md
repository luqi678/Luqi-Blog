## 二、 题目描述

在某商场的地下停车场，部署了一套智能导航系统。停车场可以看作是一个 $r \times c$ 的网格矩阵，其中：

- `0` 表示该位置是空的行车道，车辆可以通行。
- `1` 表示该位置存有障碍物、立柱或其他已停放的车辆，车辆无法通行。

停车场的入口统一设在坐标 `[0, 0]` 处。现在有一辆车进入停车场，需要前往指定的目标车位 `[m, n]`。

车辆在停车场内只能沿着上、下、左、右四个方向移动，每移动一个格子计为步数 1。请你帮车主规划一条从入口到目标车位的最短路径。

## 二、 输入描述

1. 第一行输入两个整数 `m` 和 `n`，表示目标车位的行下标和列下标。
2. 第二行输入两个整数 `row` 和 `col`，表示停车场的总行数和总列数。
3. 接下来的 `row` 行，每行包含 `col` 个以空格分隔的整数（`0` 或 `1`），表示停车场的状态信息。

**约束条件：**
- $1 < row, col < 200$
- $0 \le m < row, 0 \le n < col$
- 入口 `[0, 0]` 保证为空位。

## 三、 输出描述

输出一个整数，表示从入口 `[0, 0]` 到目标车位 `[m, n]` 的最短路径步数。
如果由于障碍物阻挡无法到达目标位置，则输出 `-1`。

## 四、 用例

### 用例 1
**输入：**
```text
1 1
3 3
0 0 0
0 0 0
0 0 0
```
**输出：**
```text
2
```
**说明：**
停车场大小为 3x3，目标位置在 [1, 1]。最短路径序列可为：[0, 0] -> [0, 1] -> [1, 1]，共移动 2 步。或者：[0, 0] -> [1, 0] -> [1, 1]，共移动 2 步。

### 用例 2
**输入：**
```text
2 2
3 3
0 1 0
1 1 0
0 0 0
```
**输出：**
```text
-1
```
**说明：**
由于入口 [0, 0] 的右边 [0, 1] 和下边 [1, 0] 均为障碍物（1），车辆被困在入口处，无法到达目标位置 [2, 2]，因此输出 -1。

### 用例 3
**输入：**
```text
0 2
3 3
0 0 0
1 1 0
0 0 0
```
**输出：**
```text
2
```
**说明：**
目标位置在 [0, 2]。虽然中间有障碍物，但可以沿着第一行直接向右行驶：[0, 0] -> [0, 1] -> [0, 2]，步数为 2。

## 五、 备注
本题属于典型的网格搜索最短路径问题，适合使用广度优先搜索（BFS）求解。

## 六、 题目解析

### 1. 题目分析
题目要求在一个带有障碍物的网格中，寻找从起点 `(0, 0)` 到终点 `(m, n)` 的**最短路径**。由于每个格子的移动代价都是 1，这属于典型的非权值图最短路径问题。

### 2. 解题思路
1. **算法选择**：
   - **广度优先搜索 (BFS)** 是求解无权图最短路径的最佳方案。BFS 能够保证在第一次到达目标节点时，所经过的路径一定是最短的。
   - 深度优先搜索 (DFS) 虽然可以找到路径，但不能直接保证找到的是最短路径（需要遍历所有路径），效率较低。

2. **状态记录**：
   - 使用一个队列来存储当前待访问的坐标。
   - 使用一个布尔矩阵 `visited` 或者直接修改原矩阵来记录该位置是否已被访问，防止陷入死循环。
   - 记录步数：可以将步数与坐标一起存入队列，或者按层遍历。

3. **具体步骤**：
   - 初始化队列，将 `(0, 0, 0)`（行、列、步数）入队。
   - 标记 `(0, 0)` 为已访问。
   - 当队列不为空时：
     - 弹出队首元素。
     - 判断当前位置是否为 `(m, n)`，若是，返回当前步数。
     - 遍历四个方向（上下左右）。
     - 对于新坐标，检查是否在边界内、是否为可行走路径（0）、且未被访问。
     - 若满足条件，将其标记为已访问并入队，步数 + 1。
   - 若队列为空仍未到达终点，返回 -1。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 输入目标坐标 m, n
        if (!sc.hasNextInt()) return;
        int targetM = sc.nextInt();
        int targetN = sc.nextInt();

        // 输入网格大小 row, col
        int rows = sc.nextInt();
        int cols = sc.nextInt();

        // 输入网格数据
        int[][] grid = new int[rows][cols];
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                grid[i][j] = sc.nextInt();
            }
        }

        // BFS 求解最短路径
        int result = bfs(grid, rows, cols, targetM, targetN);
        System.out.println(result);
    }

    private static int bfs(int[][] grid, int rows, int cols, int targetM, int targetN) {
        // 如果起点或终点是障碍物，直接返回 -1
        if (grid[0][0] == 1 || grid[targetM][targetN] == 1) {
            return -1;
        }

        // 如果起点就是终点
        if (targetM == 0 && targetN == 0) {
            return 0;
        }

        // 定义方向向量：上下左右
        int[][] directions = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};

        // 队列存储 [row, col, steps]
        Queue<int[]> queue = new LinkedList<>();
        queue.offer(new int[]{0, 0, 0});

        // 访问标记数组
        boolean[][] visited = new boolean[rows][cols];
        visited[0][0] = true;

        while (!queue.isEmpty()) {
            int[] current = queue.poll();
            int r = current[0];
            int c = current[1];
            int steps = current[2];

            // 尝试四个方向
            for (int[] dir : directions) {
                int nr = r + dir[0];
                int nc = c + dir[1];

                // 检查边界、障碍物、是否访问过
                if (nr >= 0 && nr < rows && nc >= 0 && nc < cols 
                        && grid[nr][nc] == 0 && !visited[nr][nc]) {
                    
                    // 到达目标点
                    if (nr == targetM && nc == targetN) {
                        return steps + 1;
                    }

                    // 标记访问并入队
                    visited[nr][nc] = true;
                    queue.offer(new int[]{nr, nc, steps + 1});
                }
            }
        }

        return -1; // 无法到达
    }
}
```

### 4. 复杂度分析

- **时间复杂度**：$O(R \times C)$。在最坏情况下，我们需要访问网格中的每一个格子。其中 $R$ 是行数，$C$ 是列数。
- **空间复杂度**：$O(R \times C)$。主要消耗在 `visited` 数组和 BFS 使用的队列上。
