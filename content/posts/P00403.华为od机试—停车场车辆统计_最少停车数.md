## 二、题目描述

特定大小的停车场，数组 `cars[]` 表示，其中 `1` 表示有车，`0` 表示没车。

车辆大小不一：
- 小车占一个车位（长度 1）
- 货车占两个车位（长度 2）
- 卡车占三个车位（长度 3）

统计停车场最少可以停多少辆车，返回具体的数目。

## 三、输入描述

整型字符串数组 `cars[]`，其中 `1` 表示有车，`0` 表示没车，数组长度小于 1000。
输入格式通常为逗号隔开的字符串，例如 `1,0,1`。

## 四、输出描述

整型数字字符串，表示最少停车数目。

## 五、用例

### 用例1
**输入**
```
1,0,1
```
**输出**
```
2
```
**说明**
1个小车占第1个车位，第二个车位空，1个小车占第3个车位，最少有两辆车。

### 用例2
**输入**
```
1,1,0,0,1,1,1,0,1
```
**输出**
```
3
```
**说明**
1个货车占第1、2个车位；第3、4个车位空；1个卡车占第5、6、7个车位；第8个车位空；1个小车占第9个车位。最少3辆车。

## 六、题目解析

### 1. 题目分析
本题要求计算停放的最少车辆数。题目给出了三种车型及其长度（1, 2, 3）。为了使车辆总数最少，我们应该**尽可能多地使用长度长的车**来覆盖连续的 `1`（即有车位的部分）。

### 2. 解题思路
1.  **预处理**：首先将输入的逗号隔开的字符串转换为一个纯净的 `01` 序列（可以通过 `replace(",", "")` 处理）。
2.  **分割**：利用 `0`（空位）作为分隔符，将连续的 `1` 序列（有车位的部分）提取出来。例如，`1,1,0,0,1,1,1,0,1` 会被分割为 `11`, `111`, `1`。
3.  **贪心策略计算**：
    *   对于每一个连续的 `1` 序列，假设其长度为 `L`。
    *   由于我们要最少化车辆数，应优先安排卡车（长3），剩下的空间安排货车（长2），最后是小车（长1）。
    *   实际上，这等价于计算 $\lceil L / 3 \rceil$（即 $L$ 除以 3 向上取整）。
    *   计算公式为：`carsCount += (L + 2) / 3` （整数除法）。
4.  **求和**：将所有连续序列计算出的结果相加即为答案。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextLine()) return;
        
        String input = sc.nextLine();
        // 1. 去掉逗号，统一处理
        String carsStr = input.replace(",", "");
        
        // 2. 以 "0" 分割，得到连续的 "1" 序列
        // 注意：split("0+") 可以处理多个连续的0
        String[] segments = carsStr.split("0+");
        
        int totalCars = 0;
        for (String seg : segments) {
            if (seg.length() > 0) {
                int L = seg.length();
                // 3. 贪心策略：每3个车位尽量用一辆车覆盖
                // 向上取整公式：(L + 3 - 1) / 3
                totalCars += (L + 2) / 3;
            }
        }
        
        // 4. 输出结果
        System.out.println(totalCars);
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N)$，其中 $N$ 是输入字符串的长度。我们需要遍历一次字符串进行替换、分割以及统计。
- **空间复杂度**：$O(N)$，用于存储处理后的字符串和分割后的字符串数组。
