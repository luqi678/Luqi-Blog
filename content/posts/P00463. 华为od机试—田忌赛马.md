## 一、 题目描述

给定两个只包含数字的数组 $a, b$，调整数组 $a$ 里面数字的顺序，使得尽可能多的 $a[i] > b[i]$。

数组 $a$ 和 $b$ 中的数字各不相同。

输出所有可以达到**最优结果**的 $a$ 数组的数量。

## 二、输入描述

1.  输入的第一行是数组 $a$ 中的数字，其中只包含数字，每两个数字之间相隔一个空格，**$a$ 数组大小不超过 10**。
2.  输入的第二行是数组 $b$ 中的数字，其中只包含数字，每两个数字之间相隔一个空格，**$b$ 数组大小不超过 10**。

## 三、输出描述

输出所有可以达到最优结果的 $a$ 数组数量。

## 四、用例

**示例 1：**

```text
输入：
11 8 20
10 13 7

输出：
1

说明：
最优结果只有一个，a = [11, 20, 8]，对应 b = [10, 13, 7]。
比较结果：
11 > 10 (胜)
20 > 13 (胜)
8 > 7 (胜)
共3胜。故输出 1。
```

**示例 2：**

```text
输入：
11 12 20
10 13 7

输出：
2

说明：
有两个 a 数组的排列可以达到最优结果：
1. [12, 20, 11] 对比 [10, 13, 7] -> 胜, 胜, 胜 (3胜)
2. [11, 20, 12] 对比 [10, 13, 7] -> 胜, 胜, 胜 (3胜)
故输出 2。
```

## 五、备注

* 数组大小 $N \le 10$ 是本题的关键提示。
* 题目明确指出数字各不相同，意味着不需要考虑重复元素的全排列去重问题。

## 六、题目解析

### 1. 核心思路

这道题虽然叫“田忌赛马”，但与传统的贪心算法版田忌赛马有所不同。传统的题目通常只要求输出“最大胜场数”，而本题要求输出**能达到最大胜场数的排列方案数量**。

由于题目中明确给出的数据规模非常小（$N \le 10$），我们可以利用**全排列（Full Permutation）**或者**深度优先搜索（DFS）**来暴力枚举数组 $a$ 的所有排列情况。

### 2. 复杂度分析

* **数据规模**：$N \le 10$。
* **全排列数量**：$10! = 3,628,800$。
* **单次比较耗时**：$O(N)$。
* **总复杂度**：$O(N \cdot N!)$。
* 在现代计算机上，3000万次左右的基本运算可以在 1 秒内完成，因此暴力枚举是可行的。

### 3. 算法步骤

1.  **读取输入**：将两行输入解析为两个整数列表 $a$ 和 $b$。
2.  **生成排列**：使用 DFS 递归生成 $a$ 的所有排列方式。
3.  **计算胜场**：对于 $a$ 的每一种排列，计算它与 $b$ 对应位置比较的胜场数 `current_score`。
4.  **更新结果**：
    * 维护两个全局变量：`max_score`（当前找到的最大胜场数）和 `count`（方案数）。
    * 如果 `current_score > max_score`：更新 `max_score = current_score`，并将 `count` 重置为 1。
    * 如果 `current_score == max_score`：将 `count` 加 1。
5.  **输出**：最终输出 `count`。

### 4. 代码实现 (Java)

```java
import java.util.*;

public class Main {
    // 全局变量用于记录结果
    static int maxScore = -1;
    static int count = 0;
    static int[] b; // 将b设为全局以便访问
    static int n;

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // 读取数据
        if (!scanner.hasNextLine()) return;
        String[] line1 = scanner.nextLine().trim().split("\\s+");
        String[] line2 = scanner.nextLine().trim().split("\\s+");
        
        n = line1.length;
        int[] a = Arrays.stream(line1).mapToInt(Integer::parseInt).toArray();
        b = Arrays.stream(line2).mapToInt(Integer::parseInt).toArray();
        
        // 标记数组用于全排列
        boolean[] visited = new boolean[n];
        // 临时排列数组
        int[] currentPermutation = new int[n];
        
        // 开始DFS全排列
        dfs(a, visited, currentPermutation, 0);
        
        System.out.println(count);
    }

    /**
     * DFS生成全排列
     * @param a 原数组
     * @param visited 访问标记
     * @param currentPerm 也就是当前生成的a的某种排列
     * @param index 当前填坑的索引
     */
    public static void dfs(int[] a, boolean[] visited, int[] currentPerm, int index) {
        // Base Case: 排列生成完毕
        if (index == n) {
            calculateScore(currentPerm);
            return;
        }

        // 尝试每一个未被使用的数字
        for (int i = 0; i < n; i++) {
            if (!visited[i]) {
                visited[i] = true;
                currentPerm[index] = a[i];
                dfs(a, visited, currentPerm, index + 1);
                visited[i] = false; // 回溯
            }
        }
    }

    /**
     * 计算当前排列的得分并更新全局状态
     */
    private static void calculateScore(int[] currentA) {
        int currentScore = 0;
        for (int i = 0; i < n; i++) {
            if (currentA[i] > b[i]) {
                currentScore++;
            }
        }

        if (currentScore > maxScore) {
            maxScore = currentScore;
            count = 1; // 发现更大的胜场，重置计数
        } else if (currentScore == maxScore) {
            count++;   // 胜场相同，累加方案数
        }
    }
}
```

### 5. 代码实现 (Python)

```python
import sys
from itertools import permutations

def solve():
    # 读取输入
    try:
        line1 = sys.stdin.readline().strip()
        line2 = sys.stdin.readline().strip()
        if not line1 or not line2:
            return
        
        a = list(map(int, line1.split()))
        b = list(map(int, line2.split()))
    except ValueError:
        return

    n = len(a)
    max_score = -1
    count = 0

    # 获取a的所有全排列
    # itertools.permutations 返回的是元组
    all_perms = permutations(a)

    for perm_a in all_perms:
        current_score = 0
        # 计算当前排列的胜场
        for i in range(n):
            if perm_a[i] > b[i]:
                current_score += 1
        
        # 更新逻辑
        if current_score > max_score:
            max_score = current_score
            count = 1
        elif current_score == max_score:
            count += 1

    print(count)

if __name__ == "__main__":
    solve()
```
