## 一、 题目描述

唐僧师徒四人去西天取经，一路翻山越岭。一日，师徒四人途径一个 $m \times n$ 长方形区域，已知：
1.  将取经队伍作为一个整体，4人行走相同路线。
2.  取经队伍的起点为该长方形区域的左上角 $(0,0)$，目的地为该长方形区域的右下角 $(m-1, n-1)$。
3.  行走路线可以向前、后、左、右四个方向前进 (不允许超出该长方形区域)。
4.  输入包含该区域的长 $m$ 和宽 $n$、前后移动允许的高度差 $t$，以及该长方形区域内各点的高度 $h$。
5.  要求该区域内相邻两次移动的高度差在高度 $t$ 范围以内（即 $|h_{next} - h_{cur}| \le t$）。
6.  取经队伍最多有 **3次爆发机会**，每使用一次爆发机会，可以让取经队伍一次移动突破高度差限制。

请问取经队伍通过该区域最小的移动次数是多少？返回 -1 表示师徒四人无法直接通过该区域。

## 二、输入描述

输入第一行为三个整数，分别对应为长方形场地的两条边长 $m, n$，和前后移动允许的高度差 $t$。三个整数之间以空格分割。
后面是 $m$ 行，每行 $n$ 列个数据，表示长方形场地各点的高度。数据之间以空格分割。

**约束：**
* $0 < m, n \le 200$
* $0 < t \le 20$
* 每个点的高度 $h[i,j]$ 满足 $0 \le h[i,j] \le 4000$

## 三、输出描述

一个整数表示通过该区域最小的移动次数。如果无法到达，输出 -1。

## 四、用例

### 示例 1
**输入：**
```text
4 4 10
10  20  30  40
100 120 140 160
200 230 260 290
300 400 500 600
```
**输出：**
```text
6
```
**解释：**
路径如下：
(0,0)->(0,1)->(0,2)->(0,3) [高度 10->20->30->40，差值都在10以内，步数3]
(0,3)->(1,3) [高度 40->160，差值120>10，使用第1次爆发，步数4]
(1,3)->(2,3) [高度 160->290，差值130>10，使用第2次爆发，步数5]
(2,3)->(3,3) [高度 290->600，差值310>10，使用第3次爆发，步数6]
到达终点，共6步。

### 示例 2
**输入：**
```text
1 10 1
11 12 200 14 15 16 317 18 19 20
```
**输出：**
```text
-1
```
**解释：**
从12到200需要1次爆发。
从200到14需要1次爆发。
从16到317需要1次爆发。
从317到18需要1次爆发。
共需4次爆发，但只有3次机会，故无法到达。

## 五、备注

这是一道经典的网格图最短路径变种问题。如果不考虑“爆发”机会，这就是最简单的BFS。加入了“爆发”次数限制后，状态不仅仅是坐标 $(x, y)$，还需要加上剩余爆发次数 $k$。

## 六、题目解析

### 1. 题目分析
* **目标**：寻找从起点到终点的**最短路径**（最小步数）。
* **移动规则**：上下左右移动，一步算一次移动。
* **限制条件**：
    1.  坐标不越界。
    2.  高度差 $\le t$，否则必须消耗一次“爆发”机会。
    3.  “爆发”机会初始为 3 次，耗尽后不能通过高度差 $> t$ 的路径。

### 2. 解题思路
由于每一步的代价都是 1（即求最短路），最适合的算法是 **广度优先搜索 (BFS)**。

**状态定义**：
普通 BFS 的状态通常是 `(x, y)`。但在本题中，到达同一个坐标点，如果剩余的爆发次数不同，其后续的潜力是不同的（剩余次数越多越好）。
因此，我们需要定义状态为三元组：
$$State = (x, y, rest)$$
其中：
* $x, y$：当前坐标。
* $rest$：剩余爆发次数（0 到 3）。

**Visited 数组**：
为了防止重复处理和死循环，我们需要记录访问过的状态。
`visited[x][y][rest]`：表示是否已经以剩余 `rest` 次爆发机会到达过点 `(x, y)`。

**BFS 流程**：
1.  **初始化**：
    * 构建队列 `queue`，将起点压入：`{x:0, y:0, rest:3, step:0}`。
    * 标记 `visited[0][0][3] = true`。
2.  **循环处理**：
    * 取出队首元素 `curr`。
    * 如果 `curr.x == m-1` 且 `curr.y == n-1`，直接返回 `curr.step`（因为 BFS 保证第一次到达终点时步数最小）。
    * 遍历 4 个方向（上、下、左、右）得到新坐标 `(nx, ny)`。
    * **判断合法性**：
        * 越界检查：`nx, ny` 必须在网格内。
        * **计算高度差**：`diff = abs(map[nx][ny] - map[curr.x][curr.y])`。
        * **情况 A（普通移动）**：如果 `diff <= t`，则不需要消耗爆发次数。
            * 新状态：`(nx, ny, curr.rest)`。
            * 如果该状态未被访问，加入队列，标记访问，步数 +1。
        * **情况 B（爆发移动）**：如果 `diff > t`，必须消耗爆发次数。
            * 检查 `curr.rest > 0`。
            * 新状态：`(nx, ny, curr.rest - 1)`。
            * 如果该状态未被访问，加入队列，标记访问，步数 +1。
3.  **结束**：如果队列为空仍未到达终点，返回 -1。

### 3. 代码实现

#### Java 实现

```java
import java.util.LinkedList;
import java.util.Queue;
import java.util.Scanner;

public class Main {
    // 定义节点类保存状态
    static class Node {
        int x, y;
        int rest; // 剩余爆发次数
        int step; // 当前步数

        public Node(int x, int y, int rest, int step) {
            this.x = x;
            this.y = y;
            this.rest = rest;
            this.step = step;
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (sc.hasNextInt()) {
            int m = sc.nextInt();
            int n = sc.nextInt();
            int t = sc.nextInt();

            int[][] map = new int[m][n];
            for (int i = 0; i < m; i++) {
                for (int j = 0; j < n; j++) {
                    map[i][j] = sc.nextInt();
                }
            }

            System.out.println(bfs(m, n, t, map));
        }
    }

    private static int bfs(int m, int n, int t, int[][] map) {
        // 方向数组：上下左右
        int[][] directions = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};
        
        // 队列
        Queue<Node> queue = new LinkedList<>();
        // 初始状态：(0,0), 3次机会, 0步
        queue.offer(new Node(0, 0, 3, 0));

        // 访问标记数组 visited[row][col][rest]
        boolean[][][] visited = new boolean[m][n][4];
        visited[0][0][3] = true;

        while (!queue.isEmpty()) {
            Node curr = queue.poll();

            // 到达终点
            if (curr.x == m - 1 && curr.y == n - 1) {
                return curr.step;
            }

            for (int[] dir : directions) {
                int nx = curr.x + dir[0];
                int ny = curr.y + dir[1];

                // 越界检查
                if (nx < 0 || nx >= m || ny < 0 || ny >= n) {
                    continue;
                }

                int diff = Math.abs(map[nx][ny] - map[curr.x][curr.y]);

                // 情况1：高度差在t以内，不需要爆发
                if (diff <= t) {
                    if (!visited[nx][ny][curr.rest]) {
                        visited[nx][ny][curr.rest] = true;
                        queue.offer(new Node(nx, ny, curr.rest, curr.step + 1));
                    }
                } 
                // 情况2：高度差超过t，需要爆发
                else {
                    if (curr.rest > 0 && !visited[nx][ny][curr.rest - 1]) {
                        visited[nx][ny][curr.rest - 1] = true;
                        queue.offer(new Node(nx, ny, curr.rest - 1, curr.step + 1));
                    }
                }
            }
        }

        return -1;
    }
}
```

#### Python 实现

```python
import sys
from collections import deque

def solve():
    # 读取第一行
    input_line1 = sys.stdin.readline().strip()
    if not input_line1:
        return
    m, n, t = map(int, input_line1.split())

    # 读取地图
    grid = []
    for _ in range(m):
        grid.append(list(map(int, sys.stdin.readline().split())))

    # BFS 队列: (x, y, remaining_bursts, steps)
    queue = deque([(0, 0, 3, 0)])
    
    # Visited 集合: (x, y, remaining_bursts)
    visited = set([(0, 0, 3)])

    directions = [(-1, 0), (1, 0), (0, -1), (0, 1)]

    while queue:
        x, y, rest, step = queue.popleft()

        # 到达终点
        if x == m - 1 and y == n - 1:
            print(step)
            return

        for dx, dy in directions:
            nx, ny = x + dx, y + dy

            # 越界检查
            if 0 <= nx < m and 0 <= ny < n:
                diff = abs(grid[nx][ny] - grid[x][y])
                
                # 情况1：不需要爆发
                if diff <= t:
                    if (nx, ny, rest) not in visited:
                        visited.add((nx, ny, rest))
                        queue.append((nx, ny, rest, step + 1))
                
                # 情况2：需要爆发
                elif rest > 0:
                    if (nx, ny, rest - 1) not in visited:
                        visited.add((nx, ny, rest - 1))
                        queue.append((nx, ny, rest - 1, step + 1))

    print("-1")

if __name__ == "__main__":
    solve()
```

#### C++ 实现

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <cmath>

using namespace std;

struct Node {
    int x, y, rest, step;
};

int main() {
    int m, n, t;
    if (!(cin >> m >> n >> t)) return 0;

    vector<vector<int>> grid(m, vector<int>(n));
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            cin >> grid[i][j];
        }
    }

    // 队列
    queue<Node> q;
    q.push({0, 0, 3, 0});

    // Visited: m x n x 4 (rest 0-3)
    vector<vector<vector<bool>>> visited(m, vector<vector<bool>>(n, vector<bool>(4, false)));
    visited[0][0][3] = true;

    int dx[] = {-1, 1, 0, 0};
    int dy[] = {0, 0, -1, 1};

    while (!q.empty()) {
        Node curr = q.front();
        q.pop();

        if (curr.x == m - 1 && curr.y == n - 1) {
            cout << curr.step << endl;
            return 0;
        }

        for (int i = 0; i < 4; i++) {
            int nx = curr.x + dx[i];
            int ny = curr.y + dy[i];

            if (nx >= 0 && nx < m && ny >= 0 && ny < n) {
                int diff = abs(grid[nx][ny] - grid[curr.x][curr.y]);
                
                // 情况1：不需要爆发
                if (diff <= t) {
                    if (!visited[nx][ny][curr.rest]) {
                        visited[nx][ny][curr.rest] = true;
                        q.push({nx, ny, curr.rest, curr.step + 1});
                    }
                }
                // 情况2：需要爆发
                else if (curr.rest > 0) {
                    if (!visited[nx][ny][curr.rest - 1]) {
                        visited[nx][ny][curr.rest - 1] = true;
                        q.push({nx, ny, curr.rest - 1, curr.step + 1});
                    }
                }
            }
        }
    }

    cout << -1 << endl;
    return 0;
}
```
