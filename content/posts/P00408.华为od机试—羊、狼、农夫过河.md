## 一、 题目描述

羊、狼、农夫都在岸边，当羊的数量小于狼的数量时，狼会攻击羊，农夫则会损失羊。农夫有一艘容量固定的船，能够承载固定数量的动物。

要求求出在**不损失羊**的情况下，将全部羊和狼运到对岸需要的**最小次数**。
**注意：**
1. 只计算农夫去对岸的次数，回程时农夫不会运送羊和狼（即回程不计入次数，且回程船上无动物）。
2. 农夫在场或农夫离开后，只要羊的数量大于等于狼的数量，或者羊的数量为 0，狼就不会攻击羊。
3. 农夫自身不占用船的容量。

## 二、输入描述

第一行输入为 $M$，$N$，$X$。
- $M$：羊的数量
- $N$：狼的数量
- $X$：小船的容量

## 三、输出描述

输出不损失羊情况下将全部羊和狼运到对岸需要的最小次数。
若无法满足条件则输出 `0`。

## 四、用例

### 用例1
**输入：**
`5 3 3`

**输出：**
`3`

**说明：** 
- 第一次运 2 只狼（左边剩 5 羊 1 狼，右边 0 羊 2 狼，均安全）。
- 第二次运 3 只羊（左边剩 2 羊 1 狼，右边 3 羊 2 狼，均安全）。
- 第三次运 2 只羊和 1 只狼（全部运完）。

### 用例2
**输入：**
`5 4 1`

**输出：**
`9`

**说明：** 
船容量太小，无论如何运送，都会导致某一边羊的数量小于狼的数量（除非羊为0），在这种限制下无法完成任务。

## 五、备注
- 农夫不在场时，只要该岸边羊的数量 $S > 0$ 且 $S < W$（狼的数量），羊就会被吃。
- 农夫在场时，羊也是安全的？根据题意“农夫在或农夫离开后羊的数量大于狼的数量时狼不会攻击羊”，其实意味着在任何时刻（运送前、运送后），两岸的状态都必须满足 $S=0$ 或 $S \ge W$。

## 六、题目解析

### 1. 题目分析
这是一个经典的**状态空间搜索问题**。我们需要求“最小次数”，这暗示了最适合使用 **广度优先搜索 (BFS)**。

- **核心状态：** `(当前左岸羊的数量, 当前左岸狼的数量)`。
- **目标状态：** `(0, 0)`。
- **约束条件：** 
    - 每一侧（左岸和右岸）必须满足：`羊的数量 == 0` 或 `羊的数量 >= 狼的数量`。
    - 船的最大载客量为 $X$。
    - 每次运送至少要带走 1 只动物（否则会陷入死循环）。

### 2. 解题思路
1. **初始化：** 使用一个队列存储状态 `(s_left, w_left, step)`，并使用一个 `visited` 集合（或二维布尔数组）记录已经处理过的状态，防止重复搜索。
2. **搜索过程：**
    - 从队列中取出一个状态。
    - 尝试所有可能的运送组合 $(i, j)$，其中 $i$ 是运送羊的数量，$j$ 是运送狼的数量。
    - **组合限制：** 
        - $0 \le i \le s_{left}$
        - $0 \le j \le w_{left}$
        - $1 \le i + j \le X$ （至少运一个，不超过船容量）
    - **合法性检查：**
        - **左岸剩余：** $s_{rem} = s_{left} - i$, $w_{rem} = w_{left} - j$。需满足 `s_rem == 0 || s_rem >= w_rem`。
        - **右岸新增：** $s_{new} = s_{right} + i$, $w_{new} = w_{right} + j$。需满足 `s_new == 0 || s_new >= w_new`。
        - 其中 $s_{right} = M - s_{left}$，$w_{right} = N - w_{left}$。
    - 如果新状态合法且未访问过，加入队列。
3. **终点：** 一旦发现 $s_{rem} == 0$ 且 $w_{rem} == 0$，返回当前的步数。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    static class State {
        int sLeft, wLeft, steps;

        State(int sLeft, int wLeft, int steps) {
            this.sLeft = sLeft;
            this.wLeft = wLeft;
            this.steps = steps;
        }

        @Override
        public boolean equals(Object o) {
            if (this == o) return true;
            if (!(o instanceof State)) return false;
            State state = (State) o;
            return sLeft == state.sLeft && wLeft == state.wLeft;
        }

        @Override
        public int hashCode() {
            return Objects.hash(sLeft, wLeft);
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) return;
        int M = sc.nextInt(); // 羊
        int N = sc.nextInt(); // 狼
        int X = sc.nextInt(); // 容量

        System.out.println(solve(M, N, X));
    }

    private static int solve(int M, int N, int X) {
        Queue<State> queue = new LinkedList<>();
        Set<String> visited = new HashSet<>();

        queue.add(new State(M, N, 0));
        visited.add(M + "," + N);

        while (!queue.isEmpty()) {
            State curr = queue.poll();

            // 尝试运送 i 只羊和 j 只狼
            for (int i = 0; i <= curr.sLeft; i++) {
                for (int j = 0; j <= curr.wLeft; j++) {
                    // 1. 船不能载 0 个，也不能超过 X 个
                    if (i + j == 0 || i + j > X) continue;

                    int nextSLeft = curr.sLeft - i;
                    int nextWLeft = curr.wLeft - j;
                    int nextSRight = M - nextSLeft;
                    int nextWRight = N - nextWLeft;

                    // 2. 检查左岸安全 (如果还有羊，羊必须 >= 狼)
                    if (nextSLeft > 0 && nextSLeft < nextWLeft) continue;

                    // 3. 检查右岸安全 (如果还有羊，羊必须 >= 狼)
                    if (nextSRight > 0 && nextSRight < nextWRight) continue;

                    // 4. 到达目标
                    if (nextSLeft == 0 && nextWLeft == 0) {
                        return curr.steps + 1;
                    }

                    // 5. 加入新状态
                    String stateKey = nextSLeft + "," + nextWLeft;
                    if (!visited.contains(stateKey)) {
                        visited.add(stateKey);
                        queue.add(new State(nextSLeft, nextWLeft, curr.steps + 1));
                    }
                }
            }
        }

        return 0;
    }
}
```

### 4. 复杂度分析
- **时间复杂度：** $O(M \cdot N \cdot X^2)$。状态空间的大小是 $M \times N$，对于每个状态，我们枚举运送羊和狼的组合，组合数约为 $X^2$。在机考规模下（通常 $M, N \le 100$），这个复杂度完全可以接受。
- **空间复杂度：** $O(M \cdot N)$，用于存储 `visited` 集合和 BFS 队列。
