## 一、 题目描述

某个充电站，可提供 $n$ 个充电设备，每个充电设备均有对应的输出功率。任意个充电设备组合的输出功率总和，均构成功率集合 $P$ 的 1 个元素。功率集合 $P$ 的最优元素，表示最接近充电站最大输出功率 $P_{max}$ 的元素。

请根据给定的设备功率列表和最大功率限制，找出功率集合 $P$ 中的最优元素。

## 二、输入描述

输入为 3 行：
1. 第 1 行为充电设备个数 $n$。
2. 第 2 行为每个充电设备的输出功率 $P_i$（以空格分隔）。
3. 第 3 行为充电站最大输出功率 $P_{max}$。

## 三、输出描述

输出功率集合 $P$ 的最优元素（即小于或等于 $P_{max}$ 的最大功率总和）。

## 四、用例

### 用例1
**输入**
```text
4
50 20 20 60
90
```
**输出**
```text
90
```
**说明**
当充电设备输出功率 50、20、20 组合时，其输出功率总和为 90，最接近充电站最大充电输出功率 90，因此最优元素为 90。

### 用例2
**输入**
```text
2
50 40
30
```
**输出**
```text
0
```
**说明**
所有充电设备的输出功率组合均大于充电站最大充电输出功率 30，此时最优元素值为 0。

## 五、备注

*   充电设备个数 $n > 0$。
*   最优元素必须小于或等于充电站最大输出功率 $P_{max}$。

## 六、题目解析

### 1. 题目分析
本题要求从给定的 $n$ 个充电设备中选择若干个，使得它们的功率总和在不超过最大限制 $P_{max}$ 的前提下尽可能大。

这本质上是一个**经典的 0/1 背包问题**：
*   **物品**：充电设备。
*   **重量（Weight）**：每个设备的输出功率 $P_i$。
*   **价值（Value）**：每个设备的输出功率 $P_i$（因为我们要最大化总功率）。
*   **背包容量（Capacity）**：最大输出功率 $P_{max}$。

### 2. 解题思路
1.  **定义状态**：设 `dp[j]` 表示在容量为 `j` 的情况下，能获得的最大功率总和。
2.  **状态转移方程**：
    对于每一个设备功率 $p$：
    $$dp[j] = \max(dp[j], dp[j - p] + p)$$
    其中 $j$ 从 $P_{max}$ 递减到 $p$（为了保证每个设备只被使用一次，即 0/1 背包的特性）。
3.  **初始化**：`dp` 数组长度为 $P_{max} + 1$，全部初始化为 0。
4.  **遍历顺序**：外层遍历每一个设备，内层倒序遍历功率值。
5.  **结果**：最后输出 `dp[P_max]` 即可。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 1. 读取设备个数
        if (!sc.hasNextInt()) return;
        int n = sc.nextInt();

        // 2. 读取每个设备的功率
        int[] powers = new int[n];
        for (int i = 0; i < n; i++) {
            powers[i] = sc.nextInt();
        }

        // 3. 读取最大输出功率限制
        if (!sc.hasNextInt()) return;
        int pMax = sc.nextInt();

        // 4. 0/1 背包 动态规划
        // dp[j] 表示在最大限制为 j 时，能达到的最大总功率
        int[] dp = new int[pMax + 1];

        for (int i = 0; i < n; i++) {
            int currentPower = powers[i];
            // 只有当前设备功率小于等于限制时才有必要更新
            for (int j = pMax; j >= currentPower; j--) {
                dp[j] = Math.max(dp[j], dp[j - currentPower] + currentPower);
            }
        }

        // 5. 输出结果
        System.out.println(dp[pMax]);
    }
}
```

### 4. 复杂度分析
*   **时间复杂度**：$O(n \times P_{max})$。其中 $n$ 为设备数量，$P_{max}$ 为最大限制功率。我们需要两层循环。
*   **空间复杂度**：$O(P_{max})$。使用了一维 DP 数组来存储中间状态。
