## 一、 题目描述

给定一个正整数 `n`，如果能够分解为 `m` (`m > 1`) 个连续正整数之和，请输出所有分解中，`m` **最小**的分解。

如果给定整数无法分解为连续正整数，则输出字符串 `"N"`。

## 二、输入描述

输入数据为一整数，范围为 $(1, 2^{30}]$。

## 三、输出描述

输出分解后的字符串形式。

**示例：**
输入：`21`
输出：`21=10+11`

## 四、备注

- 题目要求 `m` 最小，即分解出的连续序列**长度最短**。
- `m > 1`，意味着不能是数字本身。
- 数学性质：连续正整数之和本质上是一个等差数列求和。

## 五、用例

| 输入 | 输出 | 说明 |
| :--- | :--- | :--- |
| `21` | `21=10+11` | 21有多种分解：<br>1. `1+2+3+4+5+6` (m=6)<br>2. `6+7+8` (m=3)<br>3. `10+11` (m=2)<br>题目要求m最小，所以输出m=2的情况。 |
| `16` | `N` | 16 是 $2^4$，所有的 $2^k$ 都无法分解为连续自然数之和（除了它自己，但题目要求m>1）。 |
| `15` | `15=7+8` | `1+2+3+4+5` (m=5), `4+5+6` (m=3), `7+8` (m=2)。最小m为2。 |

## 六、题目解析

### 核心思路

这是一个数学问题。设这 $m$ 个连续正整数的起始值为 $x$，则这串序列为 $x, x+1, ..., x+m-1$。
根据等差数列求和公式，和为：
$$n = \frac{m \times (x + (x + m - 1))}{2}$$
变换公式得到：
$$2n = m \times (2x + m - 1)$$

我们需要找到 **最小的 m** ($m > 1$) 和对应的正整数 $x$ ($x \ge 1$)。

### 求解步骤

1.  **枚举 m**：
    由于题目要求 `m` 最小，我们可以从 $m=2$ 开始从小到大枚举。
    
2.  **判断 m 是否合法**：
    由公式 $2x = \frac{2n}{m} - m + 1$ 可知，为了让 $x$ 成为正整数，必须满足：
    * **条件1**：$2n$ 能被 $m$ 整除 (即 `(2*n) % m == 0`)。
    * **条件2**：计算出的 $\frac{2n}{m} - m + 1$ 必须是偶数 (这样除以2才能得到整数 $x$)。
    * **条件3**：计算出的 $x$ 必须 $\ge 1$ (题目要求正整数)。

3.  **枚举范围**：
    因为 $x \ge 1$，代入公式 $2n = m(2x + m - 1) \ge m(2 + m - 1) = m(m+1)$。
    所以 $m(m+1) \le 2n$，即 $m \approx \sqrt{2n}$。
    对于 $n = 2^{30}$，$\sqrt{2n} \approx 46340$。这个循环次数非常少，直接从小到大枚举即可，一旦找到立刻停止，就是最小的 `m`。

4.  **特殊情况**：
    如果循环结束都没有找到，说明该数字无法分解（通常是 2 的幂次方），输出 "N"。

### 算法实现 (Java)

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        // 输入范围 (1, 2^30]，使用 long 防止计算溢出
        if (sc.hasNextLong()) {
            long n = sc.nextLong();
            solve(n);
        }
    }

    public static void solve(long n) {
        // 从 m=2 开始尝试，寻找最短序列
        // 根据公式 2n = m * (2x + m - 1)
        // 推导 x = [ (2n/m) - m + 1 ] / 2
        // 约束：m * (m + 1) <= 2n (保证 x >= 1)
        
        for (long m = 2; m * (m + 1) <= 2 * n; m++) {
            // 1. 2n 必须能被 m 整除
            if ((2 * n) % m == 0) {
                long temp = (2 * n) / m - m + 1;
                // 2. 分子必须是偶数，才能除以2得到整数x
                if (temp % 2 == 0) {
                    long x = temp / 2;
                    // x 必须是正整数 (其实由循环条件 m*(m+1)<=2n 已隐式保证 x>=1)
                    if (x >= 1) {
                        // 找到解，因为是从小到大枚举m，第一个找到的一定是m最小的
                        StringBuilder sb = new StringBuilder();
                        sb.append(n).append("=");
                        for (long i = 0; i < m; i++) {
                            sb.append(x + i);
                            if (i != m - 1) {
                                sb.append("+");
                            }
                        }
                        System.out.println(sb.toString());
                        return;
                    }
                }
            }
        }
        
        // 如果循环结束未找到
        System.out.println("N");
    }
}
```
