## 一、 题目描述

有一个二维的天线矩阵，每根天线可以向其他天线发射信号也能接收其他天线的信号，为了简化起见，我们约定每根天线只能向**东**和向**南**发射信号，换言之，每根天线只能接收**西向**或**北向**发送的信号。

每根天线有自己的高度 $anth$，各根天线的高度存储在一个二维数组中。各个天线的位置用 $[r, c]$ 表示，$r$ 代表天线的行位置(从 0 开始编号)，$c$ 代表天线的列位置(从 0 开始编号)。

在某一方向(东向或南向)，某根天线可以收到多根其他天线的信号(也可能收不到任何其他天线的信号)，对任一天线 X 和天线 Y，天线 X 能接收到天线 Y 的信号的条件是:

1. 天线 X 在天线 Y 的东边（同一行，右侧）或南边（同一列，下方）；
2. 天线 X 和天线 Y 之间的其他天线的高度都**严格低于**天线 X 和天线 Y，或天线 X 和天线 Y 之间无其他天线，即无遮挡。

**示例说明：**
在天线矩阵的第 0 行上：
- 天线 [0,0] 接收不到任何信号。
- 天线 [0,1] 可以接收到 [0,0] 的信号。
- 天线 [0,2] 可以接收到 [0,1] 的信号。
- 天线 [0,3] 可以接收到 [0,1] 和 [0,2] 的信号（因为 [0,2] 的高度如果低于 [0,1] 和 [0,3]，则 [0,1] 的信号可以传到 [0,3]）。

给一个 $m$ 行 $n$ 列的矩阵，矩阵存储各根天线的高度。求出每根天线可以收到多少根其他天线的信号，结果输出到 $m \times n$ 的矩阵中。

## 二、输入描述

1. 第 1 行为输入矩阵的行数 $m$ 和列数 $n$。
2. 第 2 行为输入矩阵的元素值，按行优先顺序输入。

- $1 \le m \le 500$
- $1 \le n \le 500$
- $0 < anth[r][c] < 10^5$

## 三、输出描述

1. 第 1 行为输出矩阵的行数 $m$ 和列数 $n$。
2. 第 2 行为输出矩阵的元素值，按行优先顺序输出。

## 四、用例

### 用例1
**输入：**
```
1 6
2 4 1 5 3 3
```
**输出：**
```
1 6
0 1 1 2 1 1
```

### 用例2
**输入：**
```
2 6
2 5 4 3 2 8 9 7 5 10 10 3
```
**输出：**
```
2 6
0 1 1 1 1 4 1 2 2 4 2 2
```

## 五、备注
- 天线只向东和向南发射。
- 意味着：对于位置 $(r, c)$，它只能接收来自 $(r, 0 \dots c-1)$ 同行左侧和 $(0 \dots r-1, c)$ 同列上方的信号。

## 六、题目解析

### 1. 题目分析
本题要求计算每个点能接收到的信号数量。信号传输有两个方向：水平（从左向右）和垂直（从上向下）。
对于接收点 $X$，若要接收点 $Y$ 的信号：
- $Y$ 必须在 $X$ 的左侧（同行）或上方（同列）。
- $X$ 与 $Y$ 之间的所有天线高度必须小于 $\min(height[X], height[Y])$。

这实际上是在寻找每个元素左侧/上方满足特定遮挡条件的元素个数。

### 2. 解题思路
这是一个经典的**单调栈**应用场景。我们可以分别计算“水平方向接收到的信号”和“垂直方向接收到的信号”，最后求和。

以水平方向为例（处理某一行）：
我们维护一个**单调递减栈**。遍历当前行中的每一个高度 $h$：
1. 只要栈不为空，栈顶元素 $top$ 就可以向 $h$ 发射信号（因为它是离 $h$ 最近的或者中间没有更高的遮挡）。
2. 如果栈顶 $top < h$：说明 $top$ 这一根天线会被当前高度 $h$ 彻底挡住，以后右侧的天线不可能接收到 $top$ 的信号了。因此，计数加 1，并将 $top$ **弹出**。
3. 如果栈顶 $top == h$：说明 $top$ 可以发信号给 $h$，但 $top$ 也会被 $h$ 挡住（因为中间不能有大于等于的高度）。计数加 1，并将 $top$ **弹出**（用当前的 $h$ 替代它，效果一样）。
4. 如果栈顶 $top > h$：说明 $top$ 可以发信号给 $h$，且 $top$ 暂时不会被挡住。计数加 1，**停止**继续向左看。
5. 将当前高度 $h$ 压入栈。

同理，对每一列进行垂直方向的操作。

### 3. 代码实现

#### Java 实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) return;
        int m = sc.nextInt();
        int n = sc.nextInt();
        
        int[][] anth = new int[m][n];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                anth[i][j] = sc.nextInt();
            }
        }

        int[][] res = new int[m][n];

        // 1. 处理每一行 (水平方向：从左向右发射信号)
        for (int i = 0; i < m; i++) {
            Deque<Integer> stack = new ArrayDeque<>();
            for (int j = 0; j < n; j++) {
                int curHeight = anth[i][j];
                while (!stack.isEmpty()) {
                    res[i][j]++; // 只要栈不空，栈顶就能发信号给当前
                    if (stack.peek() < curHeight) {
                        stack.pop(); // 栈顶矮，被当前遮挡，弹出
                    } else if (stack.peek() == curHeight) {
                        stack.pop(); // 栈顶相等，挡住后面，弹出（当前入栈替代）
                        break; 
                    } else {
                        // 栈顶高，能发信号但不会被遮挡，停止
                        break;
                    }
                }
                stack.push(curHeight);
            }
        }

        // 2. 处理每一列 (垂直方向：从上向下发射信号)
        for (int j = 0; j < n; j++) {
            Deque<Integer> stack = new ArrayDeque<>();
            for (int i = 0; i < m; i++) {
                int curHeight = anth[i][j];
                while (!stack.isEmpty()) {
                    res[i][j]++;
                    if (stack.peek() < curHeight) {
                        stack.pop();
                    } else if (stack.peek() == curHeight) {
                        stack.pop();
                        break;
                    } else {
                        break;
                    }
                }
                stack.push(curHeight);
            }
        }

        // 输出结果
        System.out.println(m + " " + n);
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                sb.append(res[i][j]).append(" ");
            }
        }
        System.out.println(sb.toString().trim());
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(M \times N)$。矩阵中每个元素在水平方向和垂直方向的单调栈操作中各入栈和出栈一次。
- **空间复杂度**：$O(M \times N)$。用于存储高度矩阵和结果矩阵，以及单调栈所需的额外空间（最大为 $\max(M, N)$）。
