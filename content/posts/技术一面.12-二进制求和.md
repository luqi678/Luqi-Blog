## 一、 题目描述
给你两个二进制字符串 `a` 和 `b` ，以二进制字符串的形式返回它们的和。

**要求：**
1. 输入字符串只包含字符 `'0'` 和 `'1'`。
2. **禁止使用** 编程语言内置的任意进制转换函数（如 Java 的 `Integer.parseInt(s, 2)`）。
3. 需要考虑各种边界情况，例如字符串长度不一致、最高位进位等。

## 二、输入描述
* 输入为两个字符串 `a` 和 `b`。
* 字符串长度范围：`1 <= a.length, b.length <= 10^4`。
* 字符串仅由 `'0'` 或 `'1'` 组成，且除 `'0'` 本身外，不会以 `0` 开头。

## 三、输出描述
* 输出为一个二进制字符串，表示两数之和。

## 四、用例
### 用例1
输入：`a = "11"`, `b = "1"`  
输出：`"100"`

### 用例2
输入：`a = "1010"`, `b = "1011"`  
输出：`"10101"`

## 五、备注
* 本题考察对基础位运算逻辑的理解。
* 在处理大数时，由于字符串可能非常长，直接转换为整数类型（int/long）会导致溢出，因此必须使用字符串模拟。

## 六、题目解析

### 1. 题目分析
二进制加法的规则与十进制类似：
- `0 + 0 = 0`
- `0 + 1 = 1`
- `1 + 1 = 0`（产生进位 1）
- `1 + 1 + 1` (含进位) = `1`（产生进位 1）

### 2. 解题思路
我们采取**从后往前（从低位到高位）**遍历的策略：
1. 使用两个指针 `i` 和 `j` 分别指向 `a` 和 `b` 的末尾。
2. 定义一个变量 `carry` 用于存储进位，初始值为 0。
3. 只要 `i >= 0` 或 `j >= 0` 或 `carry > 0`，就继续循环：
   - 获取 `a` 在 `i` 位置的数字（若越界则补 0）。
   - 获取 `b` 在 `j` 位置的数字（若越界则补 0）。
   - 计算当前位的总和：`sum = valA + valB + carry`。
   - 当前位的结果为 `sum % 2`，更新进位 `carry = sum / 2`。
4. 将计算结果按顺序存入 `StringBuilder`。
5. 循环结束后，由于是从低位开始添加的，最后需要将结果字符串反转（`reverse`）。

### 3. 代码实现

#### Java8实现
```java
public class Solution {
    public String addBinary(String a, String b) {
        StringBuilder res = new StringBuilder();
        int i = a.length() - 1;
        int j = b.length() - 1;
        int carry = 0; // 进位

        // 只要还有位没算完，或者还有进位，就继续
        while (i >= 0 || j >= 0 || carry != 0) {
            int digitA = i >= 0 ? a.charAt(i) - '0' : 0;
            int digitB = j >= 0 ? b.charAt(j) - '0' : 0;

            int sum = digitA + digitB + carry;
            
            // 当前位的结果
            res.append(sum % 2);
            // 更新进位
            carry = sum / 2;

            i--;
            j--;
        }

        // 因为是从低位开始append的，最后需要反转
        return res.reverse().toString();
    }

    public static void main(String[] args) {
        Solution sol = new Solution();
        System.out.println(sol.addBinary("11", "1"));      // 输出 "100"
        System.out.println(sol.addBinary("1010", "1011")); // 输出 "10101"
    }
}
```

### 4. 复杂度分析
*   **时间复杂度**：O(max(N, M))，其中 N 和 M 分别是字符串 `a` 和 `b` 的长度。我们需要遍历较长字符串的所有位数。
*   **空间复杂度**：O(max(N, M))，用于存储结果的 `StringBuilder` 长度最多为 `max(N, M) + 1`。
