## 一、 题目描述

请实现一个计票统计系统。你会收到很多投票，其中有合法的也有不合法的，请统计每个候选人得票的数量以及不合法的票数。

（注：不合法的投票指的是投票的名字不存在 n 个候选人的名字中！！）

数据范围：每组输入中候选人数量满足 $1 \le n \le 100$，总票数量满足 $1 \le m \le 100$。

## 二、输入描述

1. 第一行输入候选人的人数 $n$。
2. 第二行输入 $n$ 个候选人的名字（均为大写字母的字符串）。
3. 第三行输入投票人的人数 $m$。
4. 第四行输入具体的投票。

## 三、输出描述

1. 按照输入的顺序，每行输出候选人的名字和得票数量（以 " : " 隔开，注：英文冒号左右两边都有一个空格！）。
2. 最后一行输出不合法的票数，格式为 "Invalid : " + 不合法的票数。

## 四、用例

### 用例1

**输入：**
```text
4
A B C D
8
A D E CF A GG A B
```

**输出：**
```text
A : 3
B : 1
C : 0
D : 1
Invalid : 3
```

**说明：**
E、CF、GG 三张票是无效的，因为它们不在候选人名单（A, B, C, D）中，所以 Invalid 的数量是 3。

## 五、备注
1. 候选人名字区分大小写（题目给定均为大写）。
2. 输出顺序必须与输入的候选人名字顺序一致。

## 六、题目解析

### 1. 题目分析
本题属于基础的字符串处理和计数类题目。核心要求有两个：
- 统计每个已知候选人的得票数。
- 统计所有不在名单内的非法票数。
- **输出顺序**必须与输入的候选人顺序一致。

### 2. 解题思路
- **存储结构**：由于需要按照输入顺序输出候选人及其票数，我们可以使用 `LinkedHashMap<String, Integer>`。它既能通过 Key 快速查找和更新票数，又能维护插入顺序。
- **流程步骤**：
  1. 读取候选人数量 $n$。
  2. 读取 $n$ 个候选人名字，将其存入 `LinkedHashMap` 中，初始票数设为 0。
  3. 读取投票人数 $m$。
  4. 遍历 $m$ 个投票名字：
     - 如果名字存在于 `Map` 中，则对应的 Value 加 1。
     - 如果名字不存在于 `Map` 中，则专门设置一个 `invalidCount` 变量进行累加。
  5. 按照 `Map` 遍历输出，最后输出 `invalidCount`。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;
import java.util.LinkedHashMap;
import java.util.Map;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 处理可能存在的多组输入
        while (sc.hasNext()) {
            // 1. 读取候选人人数
            int n = sc.nextInt();
            // 使用LinkedHashMap保证输出顺序与输入顺序一致
            Map<String, Integer> candidates = new LinkedHashMap<>();
            for (int i = 0; i < n; i++) {
                candidates.put(sc.next(), 0);
            }

            // 2. 读取投票人数
            int m = sc.nextInt();
            int invalidCount = 0;

            // 3. 处理每一张选票
            for (int i = 0; i < m; i++) {
                String vote = sc.next();
                if (candidates.containsKey(vote)) {
                    candidates.put(vote, candidates.get(vote) + 1);
                } else {
                    invalidCount++;
                }
            }

            // 4. 按照顺序输出候选人得票情况
            for (Map.Entry<String, Integer> entry : candidates.entrySet()) {
                System.out.println(entry.getKey() + " : " + entry.getValue());
            }

            // 5. 输出非法票数
            System.out.println("Invalid : " + invalidCount);
        }
        sc.close();
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N + M)$，其中 $N$ 是候选人数，$M$ 是投票总数。每个元素只需要遍历一次。
- **空间复杂度**：$O(N)$，需要一个哈希表来存储候选人的信息。
