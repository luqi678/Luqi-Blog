## 二、 题目描述

给定一个字符串，只包含字母和数字，按要求找出字符串中的最长（连续）子串的长度，字符串本身是其最长的子串，子串要求：

1. 只包含 1 个字母 (a~z, A~Z)，其余必须是数字；
2. 字母可以在子串中的任意位置；

如果找不到满足要求的子串，如全是字母或全是数字，则返回 -1。

## 二、输入描述

输入为一个字符串（只包含字母和数字）。

## 三、输出描述

输出为满足条件的子串的最大长度。

## 四、用例

### 用例1

**输入**
```text
abC124ACb
```

**输出**
```text
4
```

**说明**
满足条件的最长子串是 `C124` 或者 `124A`，长度都是 4。

### 用例2

**输入**
```text
a5
```

**输出**
```text
2
```

**说明**
字符串自身就是满足条件的子串，长度为 2。

### 用例3

**输入**
```text
aBB9
```

**输出**
```text
2
```

**说明**
满足条件的子串为 `B9`，长度为 2。

### 用例4

**输入**
```text
abcdef
```

**输出**
```text
-1
```

**说明**
没有满足要求的子串（必须包含至少一个数字），返回 -1。

## 五、备注

- 字符串长度 $1 \le L \le 10^5$。
- 子串必须包含且仅包含一个字母。
- 子串中除了该字母外，必须全是数字（意味着如果只有字母没有数字，也不符合“其余必须是数字”的隐含逻辑，参考用例4）。

## 六、题目解析

### 1. 题目分析
题目要求找一个最长的连续子串，满足：
- 含有且仅有一个字母。
- 剩下的字符必须全是数字。
- 如果没有满足条件的子串，返回 -1。

关键点在于：一个符合条件的子串，其左右边界是由该字母前后的“其他字母”决定的。为了使包含某个字母的子串最长，我们应该尽可能地向左和向右延伸，直到遇到另一个字母为止。

### 2. 解题思路
1. **记录位置**：首先遍历一遍字符串，将所有字母出现的索引位置记录在一个列表 `letterIdxs` 中。
2. **处理特殊情况**：
   - 如果 `letterIdxs` 为空（全是数字），或者 `letterIdxs` 的长度等于原字符串长度（全是字母），根据题意返回 -1。
3. **寻找最长长度**：
   - 遍历 `letterIdxs` 列表。
   - 对于第 `i` 个字母，其索引为 `curr`。
   - 它能向左延伸到的极限是上一个字母的索引 `prev`（如果没有上一个字母，则延伸到字符串开头，即 `-1`）。
   - 它能向右延伸到的极限是下一个字母的索引 `next`（如果没有下一个字母，则延伸到字符串末尾，即 `n`）。
   - 以这个字母为唯一字母的最长子串范围是 `(prev + 1)` 到 `(next - 1)`。
   - 该子串的长度为 `(next - 1) - (prev + 1) + 1 = next - prev - 1`。
4. **验证条件**：
   - 题目隐含要求除了字母外，其余必须是数字。如果 `len` 等于 1，说明该子串只包含一个字母，没有数字，不符合“其余是数字”的要求（参考示例4）。因此只有当 `len > 1` 时，我们才更新最大长度。
5. **返回结果**：如果 `maxLen` 没被更新过，返回 -1，否则返回 `maxLen`。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNext()) return;
        String s = sc.next();
        int n = s.length();

        // 记录所有字母的下标
        List<Integer> letterIdxs = new ArrayList<>();
        for (int i = 0; i < n; i++) {
            char c = s.charAt(i);
            if (Character.isLetter(c)) {
                letterIdxs.add(i);
            }
        }

        // 如果没有字母，不符合“包含1个字母”的条件
        if (letterIdxs.isEmpty()) {
            System.out.println("-1");
            return;
        }

        int maxLen = -1;
        int m = letterIdxs.size();

        for (int i = 0; i < m; i++) {
            // 当前字母的位置
            int curr = letterIdxs.get(i);
            
            // 左边界（上一个字母的位置，没有则为-1）
            int prev = (i == 0) ? -1 : letterIdxs.get(i - 1);
            
            // 右边界（下一个字母的位置，没有则为字符串长度n）
            int next = (i == m - 1) ? n : letterIdxs.get(i + 1);

            // 包含当前字母，且不包含其他字母的最大区间长度
            int len = next - prev - 1;

            // 题目要求其余必须是数字，如果len=1，说明只有字母没有数字
            if (len > 1) {
                maxLen = Math.max(maxLen, len);
            }
        }

        System.out.println(maxLen);
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N)$，其中 $N$ 是字符串的长度。我们需要遍历一次字符串记录字母位置，再遍历一次字母列表（长度小于等于 $N$）。
- **空间复杂度**：$O(N)$，用于存储所有字母的索引位置。
