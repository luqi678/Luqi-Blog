## 一、 题目描述

输入一个单词前缀和一个字典，输出包含该前缀的单词。

## 二、输入描述

1.  输入格式通常为一行或多行，包含：单词前缀 + 字典长度 + 字典内容。
2.  字典是一个**有序**单词数组。
3.  输入和输出中的字符均为**小写**。
4.  各元素之间通常用空格分隔。

例如：`b 3 a b c` 表示前缀为 `b`，字典长度为 `3`，字典内容为 `["a", "b", "c"]`。

## 三、输出描述

1.  输出所有包含该前缀的单词。
2.  如果有多个单词，需要**换行输出**。
3.  若没有找到任何匹配的单词，则返回 `-1`。

## 四、用例

**示例 1**

```text
输入：
b 3 a b c

输出：
b
```

**示例 2**

```text
输入：
abc 4 a ab abc abcd

输出：
abc
abcd
```

**示例 3**

```text
输入：
a 3 b c d

输出：
-1
```

## 五、备注

* 输入数据保证字典是有序的（虽然对于简单的 `startsWith` 判断，是否有序不影响核心逻辑，但有助于理解题目背景）。
* 判断前缀时严格区分长度，只有当字典中的单词以给定的 `prefix` 开头时才算匹配。

## 六、题目解析

### 1. 解题思路

这是一道基础的字符串处理题目。核心逻辑非常简单：
1.  **解析输入**：首先读取前缀字符串 `prefix`，然后读取整数 `N` 表示字典大小，最后读取 `N` 个字符串作为字典单词。
2.  **遍历匹配**：遍历这 `N` 个单词，检查每一个单词是否以 `prefix` 开头。在 Java 中可以使用 `String.startsWith()` 方法，Python 中可以使用 `str.startswith()` 方法。
3.  **结果存储与输出**：
    * 将匹配的单词存入一个结果列表。
    * 遍历结束后，检查列表是否为空。
    * 如果为空，输出 `-1`。
    * 如果不为空，依次输出列表中的单词（每个单词占一行）。

### 2. 参考代码

#### Java 实现

```java
import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // 1. 读取前缀
        if (!sc.hasNext()) return;
        String prefix = sc.next();
        
        // 2. 读取字典长度
        if (!sc.hasNextInt()) return;
        int n = sc.nextInt();
        
        List<String> result = new ArrayList<>();
        
        // 3. 读取字典单词并进行匹配
        for (int i = 0; i < n; i++) {
            String word = sc.next();
            // 判断是否以前缀开头
            if (word.startsWith(prefix)) {
                result.add(word);
            }
        }
        
        // 4. 输出结果
        if (result.isEmpty()) {
            System.out.println("-1");
        } else {
            for (String str : result) {
                System.out.println(str);
            }
        }
    }
}
```

#### Python 实现

```python
import sys

def solve():
    input_data = sys.stdin.read().split()
    
    if not input_data:
        return

    # 1. 解析输入
    prefix = input_data[0]
    n = int(input_data[1])
    
    # 剩下的部分是字典单词
    # 注意：有时候题目输入的 n 可能和实际单词数不一致（防御性编程），
    # 但通常机考题是对应的。这里取 input_data[2:] 即可。
    words = input_data[2:]
    
    results = []
    
    # 2. 遍历匹配
    # 只需要遍历前 n 个单词（防止输入有多余数据）
    for i in range(min(n, len(words))):
        word = words[i]
        if word.startswith(prefix):
            results.append(word)
            
    # 3. 输出结果
    if not results:
        print("-1")
    else:
        for res in results:
            print(res)

if __name__ == "__main__":
    solve()
```
