## 二、题目描述

游戏规则：输入一个只包含英文字母的字符串，字符串中的两个字母如果相邻且相同，就可以消除。
在字符串上反复执行消除的动作，直到无法继续消除为止，此时游戏结束。
输出最终得到的字符串长度。

## 二、输入描述

输入原始字符串 `str`，只能包含大小写英文字母，字母的大小写敏感，`str` 长度不超过100。

## 三、输出描述

输出游戏结束后，最终得到的字符串长度。

## 四、用例

### 用例1
**输入**
```
gg
```
**输出**
```
0
```
**说明**
`gg` 可以直接消除，得到空串，长度为0。

### 用例2
**输入**
```
mMbccbc
```
**输出**
```
3
```
**说明**
在 `mMbccbc` 中，可以先消除 `cc`；此时字符串变成 `mMbbc`，可以再消除 `bb`；此时字符串变成 `mMc`，此时没有相邻且相同的字符，无法继续消除。最终得到的字符串为 `mMc`，长度为3。

## 五、备注

输入中包含非大小写英文字母时，均为异常输入，直接输出 `0`。

## 六、题目解析

### 1. 题目分析
本题属于经典的**相邻元素消除问题**。规则是：只要有两个相邻且相同的字符，它们就可以被抵消。消除后，原本不相邻的字符可能会变得相邻，从而可能引发连锁消除。

### 2. 解题思路
处理这类“后进先出”或“相邻抵消”逻辑的最优数据结构是**栈（Stack）**：
1.  **合法性校验**：首先遍历字符串，检查是否包含非英文字母（`a-z` 或 `A-Z`）。如果包含，直接返回 0。
2.  **模拟消除过程**：
    *   遍历字符串中的每一个字符。
    *   如果栈为空，将当前字符入栈。
    *   如果栈不为空，查看栈顶元素：
        *   如果栈顶元素与当前字符**相同**，说明触发了消除规则，将栈顶元素弹出（Pop），且不将当前字符入栈。
        *   如果栈顶元素与当前字符**不同**，将当前字符入栈（Push）。
3.  **结果输出**：遍历结束后，栈中剩余的元素即为最终无法消除的字符，返回栈的大小即可。

### 3. 代码实现

#### Java实现

```java
import java.util.Scanner;
import java.util.Stack;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextLine()) {
            return;
        }
        String input = sc.nextLine();

        // 1. 异常输入校验：如果包含非字母字符，直接输出0
        if (!input.matches("[a-zA-Z]+")) {
            System.out.println(0);
            return;
        }

        // 2. 使用栈来模拟消除过程
        Stack<Character> stack = new Stack<>();

        for (int i = 0; i < input.length(); i++) {
            char c = input.charAt(i);
            
            if (!stack.isEmpty() && stack.peek() == c) {
                // 如果当前字符与栈顶字符相同，消除栈顶
                stack.pop();
            } else {
                // 否则入栈
                stack.push(c);
            }
        }

        // 3. 输出最终栈的大小
        System.out.println(stack.size());
    }
}
```

#### Python实现
```python
import re

def solve():
    try:
        s = input().strip()
    except EOFError:
        return

    # 异常输入校验
    if not re.match("^[a-zA-Z]+$", s):
        print(0)
        return

    stack = []
    for char in s:
        if stack and stack[-1] == char:
            stack.pop()
        else:
            stack.append(char)
            
    print(len(stack))

if __name__ == "__main__":
    solve()
```

### 4. 复杂度分析
*   **时间复杂度**：$O(n)$，其中 $n$ 是字符串的长度。我们只需要遍历一遍字符串，每个字符最多入栈一次、出栈一次。
*   **空间复杂度**：$O(n)$，在最坏情况下（没有字符被消除），栈需要存储所有的字符。
