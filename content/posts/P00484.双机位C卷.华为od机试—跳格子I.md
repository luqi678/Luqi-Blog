## 二、 题目描述

小明和朋友玩跳格子游戏，有 $n$ 个连续格子，每个格子有不同的分数，小朋友可以选择以任意格子起跳，但是不能跳连续的格子，也不能回头跳； 

给定一个代表每个格子得分的非负整数数组，计算能够得到的最高分数。

**限制条件：**
- $1 \le nums.length \le 100$
- $0 \le nums[i] \le 1000$

## 二、输入描述

给定一个数列，例如：`1 2 3 1`

## 三、输出描述

输出能够得到的最高分，例如：`4`

## 四、用例

### 用例1
输入：
```
1 2 3 1
```
输出：
```
4
```
说明：跳格子 1 (分数 = 1) ，然后跳格子 3 (分数 = 3)。总分数 = 1 + 3 = 4。

### 用例2
输入：
```
2 7 9 3 1
```
输出：
```
12
```
说明：跳格子 1 (分数 = 2)，跳格子 3 (分数 = 9)，跳格子 5 (分数 = 1)。总分数 = 2 + 9 + 1 = 12。

## 五、备注
该题目为经典的动态规划问题，类似于力扣（LeetCode）上的“打家劫舍”（House Robber）问题。

## 六、题目解析

### 1. 题目分析
本题要求在不能选取相邻格子的情况下，从一排格子中选出若干个，使得选出的分数总和最大。
- 关键限制：**不能跳连续的格子**。
- 目标：**最大化得分总和**。

### 2. 解题思路
我们可以使用**动态规划（Dynamic Programming）**来解决：

1.  **状态定义**：
    设 $dp[i]$ 为前 $i$ 个格子能获得的最高分数。

2.  **状态转移方程**：
    对于第 $i$ 个格子，我们有两种选择：
    - **不跳第 $i$ 个格子**：那么最大分数等于前 $i-1$ 个格子的最大分数，即 $dp[i] = dp[i-1]$。
    - **跳第 $i$ 个格子**：那么我们不能跳第 $i-1$ 个格子，最大分数等于前 $i-2$ 个格子的最大分数加上当前格子的分数，即 $dp[i] = dp[i-2] + nums[i]$。
    - 综上所述：$dp[i] = \max(dp[i-1], dp[i-2] + nums[i])$。

3.  **边界条件**：
    - 当只有 1 个格子时，$dp[0] = nums[0]$。
    - 当有 2 个格子时，$dp[1] = \max(nums[0], nums[1])$。

4.  **空间优化**：
    由于 $dp[i]$ 只依赖于 $dp[i-1]$ 和 $dp[i-2]$，我们可以使用两个变量来维护之前的状态，从而将空间复杂度从 $O(n)$ 降低到 $O(1)$。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;
import java.util.Arrays;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // 读取输入并转换为整数数组
        if (!sc.hasNextLine()) return;
        String line = sc.nextLine();
        String[] parts = line.trim().split("\\s+");
        int n = parts.length;
        int[] nums = new int[n];
        for (int i = 0; i < n; i++) {
            nums[i] = Integer.parseInt(parts[i]);
        }

        // 处理特殊情况
        if (n == 0) {
            System.out.println(0);
            return;
        }
        if (n == 1) {
            System.out.println(nums[0]);
            return;
        }

        // 动态规划
        int[] dp = new int[n];
        dp[0] = nums[0];
        dp[1] = Math.max(nums[0], nums[1]);

        for (int i = 2; i < n; i++) {
            // 状态转移方程：要么不选当前格子，要么选当前格子+前前一个的最大值
            dp[i] = Math.max(dp[i - 1], dp[i - 2] + nums[i]);
        }

        // 输出结果
        System.out.println(dp[n - 1]);
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(n)$，其中 $n$ 是格子的数量。我们需要遍历一遍数组。
- **空间复杂度**：$O(n)$，使用了一个长度为 $n$ 的 $dp$ 数组。如果使用空间优化技巧，可以将空间复杂度降低到 $O(1)$。
