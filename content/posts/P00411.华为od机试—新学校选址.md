## 二、题目描述
为了解新学期学生暴涨的问题,小乐村要建立所新学校。
考虑到学生上学安全问题,需要所有学生家到学校的距离最短。
假设学校和所有学生家都走在一条直线之上,请问学校建立在什么位置,能使得到学校到各个学生家的距离和最短。

## 三、输入描述
第一行: 整数 $n$ 取值范围 $[1, 1000]$, 表示有 $n$ 户家庭。
第二行: 一组整数 $m$ 取值范围 $[0, 10000]$，表示每户家庭的位置，所有家庭的位置都不相同。

## 四、输出描述
一个整数, 确定的学校的位置。如果有多个位置满足距离和最短, 则输出其中最小的一个。

## 五、用例
### 用例1
输入：
```
5
0 20 40 10 30
```
输出：
```
20
```

### 用例2
输入：
```
4
0 10 20 30
```
输出：
```
10
```

## 六、备注
对于 $n$ 个点 $x_1, x_2, \dots, x_n$，求一个点 $x$ 使得 $\sum |x_i - x|$ 最小，这在数学上是一个经典的**中位数**问题。

## 七、题目解析
### 1. 题目分析
本题要求在一维数轴上找一个点，使得该点到所有给定坐标点的绝对距离之和最小。这是一个典型的**几何中值问题**。
- 当家庭数量 $n$ 为奇数时，距离之和最小的点唯一，即为所有坐标排序后的**中位数**。
- 当家庭数量 $n$ 为偶数时，距离之和最小的点可以位于排序后中间两个点之间的**任意位置（包括端点）**。题目要求如果有多个位置，则输出最小的，因此我们选择中间两个点中**靠左**的那一个。

### 2. 解题思路
1. 读取输入的家庭数量 $n$。
2. 读取 $n$ 个家庭的坐标，存入数组或集合中。
3. 对坐标进行升序排序。
4. 根据中位数的性质，选取索引为 `(n - 1) / 2` 的位置（在 0 索引下，这能同时处理奇数和偶数情况：奇数取正中间，偶数取中间偏左）。
5. 输出该位置的坐标。

### 3. 代码实现
#### Java8实现
```java
import java.util.Arrays;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // 1. 读取家庭数量
        if (!sc.hasNextInt()) return;
        int n = sc.nextInt();
        
        // 2. 读取坐标
        int[] positions = new int[n];
        for (int i = 0; i < n; i++) {
            positions[i] = sc.nextInt();
        }
        
        // 3. 排序
        Arrays.sort(positions);
        
        // 4. 计算中位数位置
        // 如果n是奇数，(n-1)/2 正好是中间位置
        // 如果n是偶数，(n-1)/2 是中间两个数中较小的一个
        // 例如 n=4, 索引为 (4-1)/2 = 1 (即第2个数)
        int schoolLocation = positions[(n - 1) / 2];
        
        // 5. 输出结果
        System.out.println(schoolLocation);
        
        sc.close();
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(n \log n)$。主要耗时在对 $n$ 个家庭坐标进行排序。
- **空间复杂度**：$O(n)$。需要存储 $n$ 个坐标点的数组。
