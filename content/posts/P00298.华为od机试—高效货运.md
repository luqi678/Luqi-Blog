## 一、 题目描述

老李是货运公司承运人，老李的货车额定载货重量为 `wt`。现有两种货物：
- 货物 A 单件重量为 `wa`，单件运费利润为 `pa`。
- 货物 B 单件重量为 `wb`，单件运费利润为 `pb`。

老李每次发车时载货总重量刚好为货车额定的载货重量 `wt`，车上必须同时有货物 A 和货物 B，货物 A、B 不可切割。老李单车次满载运输可获得的最高利润是多少?

## 二、输入描述

第一列输入为货物 A 的单件重量 `wa`，0 < wa < 10000
第二列输入为货物 B 的单件重量 `wb`，0 < wb < 10000
第三列输入为货车的额定载重 `wt`，0 < wt < 100000
第四列输入为货物 A 的单件运费利润 `pa`，0 < pa < 1000
第五列输入为货物 B 的单件运费利润 `pb`，0 < pb < 1000

## 三、输出描述

单次满载运输的最高利润。如果无法恰好满载且同时包含 A 和 B，则输出 0。

## 四、用例

### 用例1
**输入**
```text
10 8 36 15 7
```
**输出**
```text
44
```

### 用例2
**输入**
```text
1 1 2 1 1
```
**输出**
```text
2
```

## 五、备注
1. 必须同时拥有货物 A 和货物 B，即 A 的数量 $x \ge 1$，B 的数量 $y \ge 1$。
2. 总重量必须刚好等于 `wt`，即 $wa \times x + wb \times y = wt$。

## 六、题目解析

### 1. 题目分析
本题属于典型的**不定方程求解并求最值**的问题。
我们需要找到整数 $x, y$ 使得：
- $wa \cdot x + wb \cdot y = wt$
- $x > 0$
- $y > 0$
并在满足条件的解中，找到 $Profit = x \cdot pa + y \cdot pb$ 的最大值。

### 2. 解题思路
由于题目给出的数据范围中，载重 `wt` 最大为 100,000，而货物单件重量 `wa` 最小虽然为 1，但我们可以通过枚举其中一种货物的数量来解决：

1. 遍历货物 A 的可能数量 $x$。$x$ 的取值范围是从 1 到 $(wt - wb) / wa$（因为必须留出至少一个 $wb$ 的重量给货物 B）。
2. 对于每一个固定的 $x$，计算剩余重量 $rest = wt - (x \cdot wa)$。
3. 检查 $rest$ 是否能被 $wb$ 整除：
   - 如果 $rest \% wb == 0$，说明存在合法的 $y = rest / wb$。
   - 计算此时的利润 $current\_profit = x \cdot pa + y \cdot pb$。
4. 更新最大利润。
5. 最后输出结果。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // 读取输入
        if (!sc.hasNextInt()) return;
        int wa = sc.nextInt();
        int wb = sc.nextInt();
        int wt = sc.nextInt();
        int pa = sc.nextInt();
        int pb = sc.nextInt();
        
        long maxProfit = 0;
        
        // 枚举货物 A 的数量 x
        // x 必须大于等于 1
        // 同时要保证剩余的重量至少能放下一个货物 B (wt - x*wa >= wb)
        for (int x = 1; x * wa < wt; x++) {
            int remainingWeight = wt - (x * wa);
            
            // 检查剩余重量是否能被 wb 整除，且 y 必须 >= 1
            if (remainingWeight > 0 && remainingWeight % wb == 0) {
                int y = remainingWeight / wb;
                long currentProfit = (long) x * pa + (long) y * pb;
                
                if (currentProfit > maxProfit) {
                    maxProfit = currentProfit;
                }
            }
        }
        
        System.out.println(maxProfit);
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(wt / wa)$。在最坏情况下（$wa=1, wt=100,000$），循环执行 100,000 次，对于机考的时间限制（通常 1s）来说完全可以接受。
- **空间复杂度**：$O(1)$。仅使用了常数级别的额外变量。
