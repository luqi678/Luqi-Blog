## 一、 题目描述

幻方 (Magic Square) 是一个由 $1 \sim N^2$，共 $N^2$ 个整数构成的 $N \times N$ 矩阵，满足每行、列和两条对角线上的数字和相等。

上回你已经帮助小明将写错一个数字的幻方进行了修复，小明在感谢之余也想进一步试试你的水平，于是他准备了**有两个数字发生了位置交换**的幻方。

你可以把这两个交换的数字找出来并且改正吗？

## 二、 输入描述

第一行输入一个整数 $N$，代表待校验幻方的阶数 ($3 \le N < 50$)。
接下来的 $N$ 行，每行 $N$ 个整数，空格隔开 ($1 \le \text{每个整数} \le N^2$)。

## 三、 输出描述

输出两行，代表两条纠正信息。
注意：
1. 先输出行号小的，若行号相同则先输出列号小的。
2. 每行输出空格隔开的三个整数，分别是：**出错行号、出错列号、应填入的数字**（行号列号从1开始，末尾无空格）。

## 四、 用例

### 用例1

**输入**
```text
3
8 1 9
3 5 7
4 6 2
```

**输出**
```text
1 3 6
3 2 9
```

**说明**
原本的幻方可能是：
8 1 6
3 5 7
4 9 2
此幻方每行、每列、对角线和均为15。输入的矩阵中，第一行第三列的9和第三行第二列的6发生了交换。

## 五、 备注
$3 \le N < 50$，暴力枚举交换对并检查是可行的。

## 六、 题目解析

### 1. 题目分析
幻方的特性是：
- 所有数字在 $1$ 到 $N^2$ 之间且不重复。
- 每行之和 = 每列之和 = 两条主对角线之和 = 幻方常数 $S$。
- $S = \frac{N(N^2+1)}{2}$。

题目指出有两个数字发生了位置交换，我们的目标是找到这两个位置 $(r1, c1)$ 和 $(r2, c2)$，并确定它们原本应该是什么数字。其实原本的数字就是对方现在的数字。

### 2. 解题思路
1. **计算幻方常数 $S$**：$S = N \times (N^2 + 1) / 2$。
2. **记录当前的行列和对角线和**：
   - 计算每一行的和 `rowSum[i]`。
   - 计算每一列的和 `colSum[j]`。
   - 计算两条对角线的和 `diag1` (左上到右下) 和 `diag2` (右上到左下)。
3. **暴力枚举**：
   - 由于 $N < 50$，总共 $N^2 = 2500$ 个位置。
   - 交换两个数的位置，共有 $\binom{N^2}{2}$ 种可能，大约是 $3 \times 10^6$ 种组合。
   - 对于每一对位置 $(r1, c1)$ 和 $(r2, c2)$，尝试将它们的值交换回来。
   - **优化检查**：不需要重新计算整个矩阵的和。只需要检查受影响的行、列和对角线。
     - 如果 $r1 \neq r2$，检查 `rowSum[r1]` 和 `rowSum[r2]` 交换后是否等于 $S$。
     - 如果 $c1 \neq c2$，检查 `colSum[c1]` 和 `colSum[c2]` 交换后是否等于 $S$。
     - 如果位置在对角线上，检查对角线和是否变为 $S$。
     - 同时，那些原本就不等于 $S$ 的行、列、对角线，在交换后必须全部变成 $S$；而原本就等于 $S$ 的，在交换后也必须依然是 $S$。
4. **输出结果**：一旦找到满足所有条件（所有行、列、对角线和均为 $S$）的交换对，按照题目要求的顺序输出。

### 3. 代码实现

#### Java 实现

```java
import java.util.*;

public class Main {
    static int N;
    static int[][] matrix;
    static long targetSum;
    static long[] rowSum, colSum;
    static long d1, d2;

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) return;
        N = sc.nextInt();
        matrix = new int[N][N];
        targetSum = (long) N * (N * N + 1) / 2;

        rowSum = new long[N];
        colSum = new long[N];
        d1 = 0; d2 = 0;

        for (int i = 0; i < N; i++) {
            for (int j = 0; j < N; j++) {
                matrix[i][j] = sc.nextInt();
                rowSum[i] += matrix[i][j];
                colSum[j] += matrix[i][j];
                if (i == j) d1 += matrix[i][j];
                if (i + j == N - 1) d2 += matrix[i][j];
            }
        }

        // 寻找异常的行列和对角线
        List<Integer> badRows = new ArrayList<>();
        List<Integer> badCols = new ArrayList<>();
        for (int i = 0; i < N; i++) {
            if (rowSum[i] != targetSum) badRows.add(i);
            if (colSum[i] != targetSum) badCols.add(i);
        }

        // 遍历所有可能的交换对 (r1, c1) 和 (r2, c2)
        for (int i = 0; i < N * N; i++) {
            for (int j = i + 1; j < N * N; j++) {
                int r1 = i / N, c1 = i % N;
                int r2 = j / N, c2 = j % N;

                if (check(r1, c1, r2, c2)) {
                    // 输出，注意题目要求行号小优先，列号小优先
                    // 由于 i < j，已经保证了顺序
                    System.out.println((r1 + 1) + " " + (c1 + 1) + " " + matrix[r2][c2]);
                    System.out.println((r2 + 1) + " " + (c2 + 1) + " " + matrix[r1][c1]);
                    return;
                }
            }
        }
    }

    private static boolean check(int r1, int c1, int r2, int c2) {
        int v1 = matrix[r1][c1];
        int v2 = matrix[r2][c2];
        int diff = v2 - v1;

        // 检查行
        if (r1 == r2) {
            if (rowSum[r1] != targetSum) return false;
        } else {
            if (rowSum[r1] + diff != targetSum) return false;
            if (rowSum[r2] - diff != targetSum) return false;
        }

        // 检查列
        if (c1 == c2) {
            if (colSum[c1] != targetSum) return false;
        } else {
            if (colSum[c1] + diff != targetSum) return false;
            if (colSum[c2] - diff != targetSum) return false;
        }

        // 检查对角线 1
        long newD1 = d1;
        if (r1 == c1) newD1 += diff;
        if (r2 == c2) newD1 -= diff;
        if (newD1 != targetSum) return false;

        // 检查对角线 2
        long newD2 = d2;
        if (r1 + c1 == N - 1) newD2 += diff;
        if (r2 + c2 == N - 1) newD2 -= diff;
        if (newD2 != targetSum) return false;

        // 必须确保除了这几个被修改的，其他的行和列本身就是正确的
        // 在外层已经检查过 badRows 和 badCols 的数量吗？
        // 其实只要原本不等的行在修正后变相等，原本就等的行由于没被换数字，依然会相等
        // 唯一需要担心的是：如果有其他行本来就不等于 targetSum 且 r1, r2 没覆盖到它们
        for (int i = 0; i < N; i++) {
            if (i != r1 && i != r2 && rowSum[i] != targetSum) return false;
            if (i != c1 && i != c2 && colSum[i] != targetSum) return false;
        }

        return true;
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N^4)$。我们遍历了 $N^2 \times N^2$ 个点对，每个点对的 `check` 函数中包含一个 $O(N)$ 的循环来确认其他行列是否正确。$50^4 = 6.25 \times 10^6$，在 Java 的 1s 限制内完全可以运行。
- **空间复杂度**：$O(N^2)$，用于存储矩阵。
