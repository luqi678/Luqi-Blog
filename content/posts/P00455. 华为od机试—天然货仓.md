## 一、 题目描述

有一个天然形成的大坑，为台阶状结构，每个台阶的长度都为1,每个都的值为整数(正整数表示高于地平面，零表示与地平面平齐，负整数表示低于地平面)。

有一批同等规格的货品(长度为N,高度为1)，货品只能平放，且货物的上表面不能超过地平面(高度为零) ,或者说，高于地平面的地中也不可存放货物。

计算一个给定的大坑中最多可以放多少个货品?

## 二、输入描述

1.  第一行：整数，表示物品的宽度 N。
2.  第二行：整数，表示坑的宽度 M。
3.  第三行：数组，表示坑的深度（用逗号分隔的整数）。

## 三、输出描述

整数，表示给定的大坑中最多可以放的货品数。

## 四、用例

### 示例 1
**输入：**
```text
2 
4 
0,-1,-2,0
```
**输出：**
```text
1
```
**解释：**
物品宽度为2。
* 在下标 `[0,1]` 放置：高度为 `0, -1`，最高点为 0。货品必须低于地平线，故无法放置（因为放置后顶部高度为1，超过0）。
* 在下标 `[1,2]` 放置：高度为 `-1, -2`，最高点为 -1。放置在深度 -1 处，顶部刚好为 0。容量 = 1。
* 在下标 `[2,3]` 放置：高度为 `-2, 0`，最高点为 0。无法放置。
* 最优解为 1。

### 示例 2
**输入：**
```text
3
8
0,-1,-2,2,1,1,1,2
```
**输出：**
```text
0
```
**解释：**
任意连续3个位置的最高点都 >= 0，无法放入任何货物。

## 五、备注

* 货物不能悬空，必须有支撑，但题目隐含的是填满空间的概念。根据物理规律，宽度为N的货物在区间 `[i, i+N-1]` 放置时，其实际可用的最大高度由该区间内**最高**的地形决定（即短板效应，最浅的地方决定了能放多深）。
* 货物不能重叠放置在同一个空间坐标，但可以在垂直方向堆叠（题目问的是最多放多少个货品，且货品高度为1，暗示可以堆叠）。
* 本质是在水平方向上选择不重叠的区间，或者通过DP解决最优填充问题。

## 六、题目解析

### 1. 题目分析
本题属于**动态规划（DP）**或**加权区间调度**类问题。
核心约束条件如下：
1.  **尺寸限制**：货物宽度为 $N$，高度为 1。
2.  **高度限制**：货物顶部不能超过 0。
3.  **放置逻辑**：
    * 在任意水平区间 `[i, i+N-1]` 放置货物时，该区间的有效深度由该区间内地形的**最大值**（最浅处）决定。
    * 假设区间内地形高度最大值为 $h_{max}$。
    * 如果 $h_{max} \ge 0$，则该位置无法放置货物（空间被占满或溢出）。
    * 如果 $h_{max} < 0$，则该位置可以堆叠的货物数量为 $|h_{max}|$（即 $-h_{max}$）。
4.  **目标**：最大化总货物数量。由于在位置 $i$ 放置货物会占用 $i$ 到 $i+N-1$ 的空间，这会影响后续位置的放置，因此需要决策在哪些位置放置能收益最大。

### 2. 解题思路

这是一个经典的线性动态规划问题。

**定义状态：**
设 $DP[i]$ 表示在地块前 $i$ 个单位长度（下标 $0$ 到 $i-1$）内，最多能放置的货物数量。

**状态转移方程：**
对于当前位置 $i$（假设我们正在考虑以索引 $i-1$ 结尾的区间）：

1.  **不在此处放置货物**：
    此时最大数量等于前一个位置的最大数量。
    $$DP[i] = DP[i-1]$$

2.  **尝试在此处放置一摞货物**：
    如果我们在以 $i-1$ 结尾的地方放置一个宽度为 $N$ 的货物（占据下标 $i-N$ 到 $i-1$）：
    * 前提：$i \ge N$。
    * 计算该区间 $H[i-N \dots i-1]$ 的最大高度 $maxH$。
    * 如果 $maxH < 0$，则该处能容纳的货物数为 $-maxH$。
    * 此时的总数量 = (放置该摞货物的收益) + (去除这段宽度后，前 $i-N$ 个位置的最大收益)。
    * $$DP[i] = \max(DP[i], \quad DP[i-N] + \text{capacity})$$

**最终结果：**
$DP[M]$ 即为在宽度为 $M$ 的坑中能放置的最大货物数。

### 3. 代码实现

#### Java 实现

```java
import java.util.Scanner;
import java.util.Arrays;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        if (!sc.hasNext()) return;
        int N = Integer.parseInt(sc.nextLine().trim()); // 物品宽度
        int M = Integer.parseInt(sc.nextLine().trim()); // 坑宽度
        
        String[] parts = sc.nextLine().trim().split(",");
        int[] heights = new int[M];
        for (int i = 0; i < M; i++) {
            heights[i] = Integer.parseInt(parts[i]);
        }
        
        // dp[i] 表示前 i 个单位长度（下标 0 到 i-1）能放的最大商品数
        int[] dp = new int[M + 1];
        
        // 初始化 dp[0] = 0
        
        for (int i = 1; i <= M; i++) {
            // 选项1：当前位置不放任何以 i-1 结尾的货物
            dp[i] = dp[i-1];
            
            // 选项2：尝试在 i-N 到 i-1 这个区间放货物
            if (i >= N) {
                int maxH = -999999; // 寻找区间内的最大高度（最浅深度）
                for (int k = i - N; k < i; k++) {
                    if (heights[k] > maxH) {
                        maxH = heights[k];
                    }
                }
                
                // 只有当区间内最高点都在地平面以下(maxH < 0)时，才能放货
                if (maxH < 0) {
                    int count = -maxH; // 可以堆叠的数量
                    dp[i] = Math.max(dp[i], dp[i-N] + count);
                }
            }
        }
        
        System.out.println(dp[M]);
    }
}
```

#### Python 实现

```python
import sys

def solve():
    try:
        lines = sys.stdin.read().splitlines()
        if len(lines) < 3:
            return
            
        N = int(lines[0].strip())
        M = int(lines[1].strip())
        heights = list(map(int, lines[2].strip().split(',')))
        
        if N > M:
            print(0)
            return

        # dp[i] 表示长度为 i 的坑能放的最大数量
        dp = [0] * (M + 1)
        
        for i in range(1, M + 1):
            # 策略1：不放以 i-1 结尾的货
            dp[i] = dp[i-1]
            
            # 策略2：放以 i-1 结尾的货，需占用 [i-N, i-1]
            if i >= N:
                # 截取对应区间的地形
                segment = heights[i-N : i]
                max_h = max(segment)
                
                if max_h < 0:
                    capacity = -max_h
                    dp[i] = max(dp[i], dp[i-N] + capacity)
                    
        print(dp[M])
        
    except Exception as e:
        print(0)

if __name__ == "__main__":
    solve()
```

int main() {
    int N, M;
    if (!(cin >> N)) return 0;

    // 处理逗号分隔的输入
    string line;

    if (line.empty() || line.find(',') == string::npos) {
        cin >> line;
    }

    vector<int> heights;
    stringstream ss(line);
    string temp;
    while (getline(ss, temp, ',')) {
        heights.push_back(stoi(temp));
    }
    
    if (heights.size() != M) {
        // 如果输入流分割有问题，做容错处理（视实际机考环境而定）
    }

    if (N > M) {
        cout << 0 << endl;
        return 0;
    }

    // dp[i] 表示前 i 个格子能放的最大值
    vector<int> dp(M + 1, 0);

    for (int i = 1; i <= M; i++) {
        // 继承前一个状态
        dp[i] = dp[i-1];
        
        // 尝试放置
        if (i >= N) {
            int maxH = INT_MIN;
            // 检查区间 [i-N, i-1]

                maxH = max(maxH, heights[k]);

            if (maxH < 0) {
                int count = -maxH;
                dp[i] = max(dp[i], dp[i-N] + count);
