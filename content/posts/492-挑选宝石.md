## 一、 题目描述

游乐园有一款互动游戏，游戏开始时会提供 $n$ 个宝石，每个宝石都有一个属性值 $a_1, a_2, \dots, a_n$。
玩家在游戏前可以挑选 $x$ 颗宝石，将这些宝石的属性值相乘组成玩家的属性值。
游戏玩家需要 $y$ 点属性值（注：即属性值乘积 $\ge y$），请帮助游戏玩家计算有多少种计算方式。

## 二、输入描述

- **第一行**：三个整数 $n, x, y$。
    - $n$ ($0 < n < 20$)：表示宝石总数量。
    - $x$ ($0 < x \le n$)：表示可以选择宝石个数。
    - $y$：表示通过游戏需要的属性值。
- **第二行**：$n$ 个整数 $a_1, a_2, \dots, a_n$ ($-100 < a_i < 100$)，表示每个宝石的属性值。

## 三、输出描述

- 输出一个整数，表示玩家可以通过游戏的挑选方式的数量。

## 四、备注

1. **组合不区分顺序**：即 $\{a_1, a_2\}$ 和 $\{a_2, a_1\}$ 视为同一种方式。
2. **数据范围**：$n < 20$ 且 $x \le n$，数据量极小，适合使用回溯法（Backtracking）或 DFS。
3. **数值处理**：虽然 $n$ 小，但 $20$ 个整数相乘可能溢出 `long` 范围。不过在机考常见用例中 `long` 通常足够；若需处理极端大数，Java 可用 `BigInteger`。
4. **判断逻辑**：根据示例输出推断，“需要 $y$ 点属性值”意味着 **乘积 $\ge y$**。

## 五、用例

**示例 1**

* **输入**
```text
4 2 8
2 -3 4 5
```

* **输出**
```text
3
```

* **说明**
从 4 个数中选 2 个，共有 $C(4,2) = 6$ 种组合：
1. $2 \times (-3) = -6$ (< 8)
2. $2 \times 4 = 8$ ($\ge 8$) -> **符合**
3. $2 \times 5 = 10$ ($\ge 8$) -> **符合**
4. $(-3) \times 4 = -12$ (< 8)
5. $(-3) \times 5 = -15$ (< 8)
6. $4 \times 5 = 20$ ($\ge 8$) -> **符合**
共 3 种方式。

## 六、题目解析

### 1. 解题思路
这道题是典型的**组合枚举问题**。我们需要从 $n$ 个元素中选出 $x$ 个元素，并判断其乘积是否满足条件。

* **搜索空间分析**：$n$ 最大为 20。$C(20, 10) \approx 1.8 \times 10^5$。这个量级对于计算机来说非常小（通常 1 秒可处理 $10^8$ 次操作），因此暴力搜索（DFS/回溯）完全可行。
* **算法选择**：使用递归（回溯法）遍历所有可能的组合。

### 2. 算法流程
1.  **状态定义**：`dfs(index, count, currentProduct)`
    * `index`: 当前遍历到的数组下标（避免重复选择）。
    * `count`: 当前已选取的宝石数量。
    * `currentProduct`: 当前已选宝石的乘积。
2.  **递归逻辑**：
    * **终止条件**：当 `count == x` 时，检查 `currentProduct >= y`。若满足，计数加 1，返回。
    * **剪枝优化**：如果 `(n - index) < (x - count)`，说明剩余的宝石全选也不够凑齐 $x$ 个，直接返回。
    * **分支选择**：
        * **选当前宝石**：`dfs(index + 1, count + 1, currentProduct * nums[index])`
        * **不选当前宝石**：`dfs(index + 1, count, currentProduct)`

### 3. Java 代码实现

```java
import java.util.Scanner;

public class Main {
    static int n, x;
    static long y;
    static int[] nums;
    static int result = 0;

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (sc.hasNextInt()) {
            n = sc.nextInt();
            x = sc.nextInt();
            y = sc.nextLong(); // 注意y可能较大
            
            nums = new int[n];
            for (int i = 0; i < n; i++) {
                nums[i] = sc.nextInt();
            }
            
            // 从第0个元素开始，当前已选0个，积为1
            dfs(0, 0, 1L);
            
            System.out.println(result);
        }
        sc.close();
    }

    /**
     * 回溯搜索
     * @param index 当前遍历到的宝石索引
     * @param count 当前已选宝石数量
     * @param currentProduct 当前累乘积
     */
    private static void dfs(int index, int count, long currentProduct) {
        // 1. 终止条件：选够了 x 个
        if (count == x) {
            if (currentProduct >= y) {
                result++;
            }
            return;
        }

        // 2. 剪枝：剩余宝石不足以凑齐 x 个
        // 剩余可用宝石数: n - index
        // 还需要宝石数: x - count
        if (n - index < x - count) {
            return;
        }

        // 3. 选择当前宝石 (注意可能溢出，题目未明确说明，一般long够用)
        dfs(index + 1, count + 1, currentProduct * nums[index]);

        // 4. 不选择当前宝石
        dfs(index + 1, count, currentProduct);
    }
}
```
