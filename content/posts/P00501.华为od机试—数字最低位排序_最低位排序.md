## 一、 题目描述

给定一个非空数组（列表），其元素数据类型为整型。请按照数组元素十进制最低位从小到大进行排序。
1. 十进制最低位相同的元素，**相对位置保持不变**（即要求稳定排序）。
2. 当数组元素为负值时，十进制最低位等同于去除符号位后对应十进制值的最低位（即取绝对值的最低位）。

## 二、输入描述

给定一个非空数组（列表），其元素数据类型为 32 位有符号整数。
数组长度范围为 `[1, 1000]`。
输入形式通常为逗号分隔的字符串。

## 三、输出描述

排序后的数组，元素之间以逗号分隔。

## 四、用例

### 用例1

**输入**：
```text
1,2,5,-21,22,11,55,-101,42,8,7,32
```

**输出**：
```text
1,-21,11,-101,2,22,42,32,5,55,7,8
```

**说明**：
- 最低位为 1 的元素：1, -21, 11, -101 (保持原相对顺序)
- 最低位为 2 的元素：2, 22, 42, 32
- 最低位为 5 的元素：5, 55
- 最低位为 7 的元素：7
- 最低位为 8 的元素：8
合并后结果如上。

### 用例2

**输入**：
```text
10,20,30,5,15
```

**输出**：
```text
10,20,30,5,15
```

**说明**：
10, 20, 30 的最低位都是 0，保持原顺序；5, 15 的最低位都是 5，保持原顺序。

## 五、备注

1. 必须使用**稳定排序**算法，或者利用编程语言内置的稳定排序函数。
2. 负数处理：例如 `-21` 的最低位是 `1`。可以使用 `Math.abs(n) % 10` 来获取。

## 六、题目解析

### 1. 题目分析
本题的核心考察点是**自定义排序规则**以及**排序的稳定性**。
- **规则**：比较两个数 $a$ 和 $b$ 时，比较的是 $|a| \pmod{10}$ 和 $|b| \pmod{10}$。
- **稳定性**：如果两个数的最低位相同，它们在结果中的先后顺序必须与输入时一致。

### 2. 解题思路
1. **读取输入**：将输入的字符串按逗号分割，并转换为整数列表（或对象数组）。
2. **定义比较器**：
   - 设 $valA = \text{abs}(a) \pmod{10}$
   - 设 $valB = \text{abs}(b) \pmod{10}$
   - 比较 $valA$ 和 $valB$ 的大小。
3. **执行排序**：
   - 在 Java 中，`Collections.sort()` 或 `Arrays.sort(T[] a)` 对对象进行排序时使用的是 **TimSort**，这是一种稳定的排序算法。
   - 使用 Java 8 的 `Stream.sorted()` 也是稳定的。
4. **输出结果**：将排序后的列表重新拼接成逗号分隔的字符串。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;
import java.util.stream.Collectors;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextLine()) return;
        
        String input = sc.nextLine();
        String[] parts = input.split(",");
        
        // 使用 List 存储，方便使用 Java 8 Stream
        List<Integer> nums = Arrays.stream(parts)
                                   .map(Integer::parseInt)
                                   .collect(Collectors.toList());
        
        // 稳定排序：Java 的 Collections.sort 和 Stream.sorted 默认都是稳定的
        List<Integer> result = nums.stream()
                .sorted(Comparator.comparingInt(n -> Math.abs(n) % 10))
                .collect(Collectors.toList());
        
        // 输出格式化
        String output = result.stream()
                              .map(String::valueOf)
                              .collect(Collectors.joining(","));
        
        System.out.println(output);
    }
}
```

### 4. 复杂度分析

- **时间复杂度**：$O(N \log N)$。其中 $N$ 是数组长度（最大 1000）。排序算法的时间开销占主导。
- **空间复杂度**：$O(N)$。需要额外的空间存储输入的列表以及排序后的结果。
