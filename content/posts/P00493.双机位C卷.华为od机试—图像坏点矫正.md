## 一、 题目描述

在图像处理中，坏点（异常值）会影响后续分析，需进行矫正。本题要求对 3x3 矩阵的中心元素进行坏点检测与矫正，规则如下：

1.  **坏点判断依据**：
    计算中心元素（位置 1, 1）与周围 8 个元素的均值的差值绝对值 `diff = |中心元素 - 周围均值|`。

2.  **矫正规则**：
    *   若 `diff > 50`：用周围均值（周围 8 个元素的平均值，四舍五入取整）替代中心元素；
    *   若 `30 <= diff <= 50`：用 3x3 矩阵的整体均值（9 个元素的平均值，四舍五入取整）替代中心元素；
    *   若 `diff < 30`：中心元素不变。

3.  **输出**：矫正后的 3x3 矩阵（每行元素用空格分隔）。

## 二、输入描述

输入共 3 行，每行 3 个整数（取值范围 0-255），表示 3x3 矩阵（每行元素用空格分隔）。

## 三、输出描述

输出共 3 行，表示矫正后的 3x3 矩阵（每行元素用空格分隔）。

## 四、用例

### 用例1
**输入：**
```
10 10 10
10 200 10
10 10 10
```
**输出：**
```
10 10 10
10 10 10
10 10 10
```

### 用例2
**输入：**
```
100 100 100
```

**输出：**
```

100 100 100
```

## 五、备注
1. 四舍五入取整可以使用标准的数学舍入规则（如 Java 的 `Math.round`）。
2. 输入的整数范围为 0 到 255。

## 六、题目解析

### 1. 题目分析
本题是一道标准的模拟题。核心逻辑在于对一个 3x3 矩阵的中心点（坐标 [1][1]）进行判断和修改。
*   我们需要提取中心点 $C$。
*   我们需要提取中心点以外的 8 个周围点。
*   计算周围均值 $Mean_{surr} = \text{round}(\frac{\sum \text{周围点}}{8})$。
*   计算整体均值 $Mean_{total} = \text{round}(\frac{\sum \text{所有点}}{9})$。
*   根据 $diff = |C - Mean_{surr}|$ 的区间，决定 $C$ 是否需要被替换。

### 2. 解题思路
1.  读取 3x3 矩阵并存入二维数组。
2.  累加周围 8 个元素的和 `sumSurrounding`。
3.  获取中心点的值 `center`。
4.  计算周围均值：`avgSurrounding = Math.round(sumSurrounding / 8.0)`。
5.  计算整体均值：`avgTotal = Math.round((sumSurrounding + center) / 9.0)`。
6.  计算差值绝对值 `diff = Math.abs(center - avgSurrounding)`。
7.  分支判断：
    *   `diff > 50`: `matrix[1][1] = (int)avgSurrounding`
    *   `30 <= diff <= 50`: `matrix[1][1] = (int)avgTotal`
    *   `diff < 30`: 不处理。
8.  格式化输出矩阵。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int[][] matrix = new int[3][3];
        long sumSurrounding = 0;

        // 读取矩阵
        for (int i = 0; i < 3; i++) {
            for (int j = 0; j < 3; j++) {
                matrix[i][j] = sc.nextInt();
                // 累加非中心点元素
                if (!(i == 1 && j == 1)) {
                    sumSurrounding += matrix[i][j];
                }
            }
        }

        int center = matrix[1][1];
        
        // 周围 8 个元素的均值
        long avgSurrounding = Math.round(sumSurrounding / 8.0);
        
        // 9 个元素的整体均值
        long avgTotal = Math.round((sumSurrounding + center) / 9.0);
        
        // 差值绝对值
        long diff = Math.abs(center - avgSurrounding);

        // 矫正逻辑
        if (diff > 50) {
            matrix[1][1] = (int) avgSurrounding;
        } else if (diff >= 30) {
            matrix[1][1] = (int) avgTotal;
        }

        // 输出结果
        for (int i = 0; i < 3; i++) {
            for (int j = 0; j < 3; j++) {
                System.out.print(matrix[i][j] + (j == 2 ? "" : " "));
            }
            System.out.println();
        }
    }
}
```

### 4. 复杂度分析
*   **时间复杂度**：O(1)。矩阵大小固定为 3x3，循环次数固定。
*   **空间复杂度**：O(1)。存储 3x3 矩阵需要常数空间。
