## 一、 题目描述

小聪入职新公司，参加线上的新员工必考试共25题,依次是10个判断题(每题2分）、10个单选题(每题4分)和5个多选题(每题8分),总分100分。

考题只能顺序作答，答对题目获得相应的分数,答错题目获得0分,考试系统不提示作答是否正确，答题过程中如果累积有3题答错,直接中断考试并计算考试分数。

小聪考试结果是N分（0<=N<=100)，请根据小聪的分数,算出所有可能的答题情况的个数。

## 二、输入描述

整数，表示小聪的考试得分N，N为偶数，0<=N<=100，N不会是不可能考出来的分数。

## 三、输出描述

整数表示所有可能的答题情况的个数。

## 四、用例

### 示例 1
**输入：**
```text
94
```
**输出：**
```text
100
```
**解释：**
有1道单选和一道判断题答错，其余题都对，所有可能的答题情况的个数为100。

### 示例 2
**输入：**
```text
100
```
**输出：**
```text
1
```
**解释：**
所有题都对，答题情况的个数为1。

## 五、备注

N不会是不可能考出来的分数。

## 六、题目解析

### 1. 题目分析
本题主要考察逻辑模拟与组合计数。题目核心在于答题规则：
1.  **题目分值分布**：
    * 第 1-10 题：2分/题
    * 第 11-20 题：4分/题
    * 第 21-25 题：8分/题
2.  **终止条件**：
    * **正常结束**：答完所有25道题，且错题数 < 3。
    * **提前终止**：一旦答错累计达到3题，考试立即结束，且第3道错题不扣分也不得分（直接结算之前的分数）。
3.  **目标**：给定最终得分 $N$，求有多少种答题路径能得到这个分数。

### 2. 解题思路
由于题目总数较少（25题），且状态空间有限（最多错3次），非常适合使用 **DFS（深度优先搜索）** 或 **递归回溯** 来解决。

我们需要定义递归函数的参数来记录当前的状态：
* `index`: 当前正在做第几题（0-24）。
* `score`: 当前累计得分。
* `wrongCount`: 当前累计错误次数。

**递归逻辑（状态转移）：**

对于每一道题，有两种情况：

1.  **答对**：
    * 分数增加当前题目的分值。
    * 进入下一题 `index + 1`。
    * 错误次数 `wrongCount` 不变。
    * *剪枝条件*：如果加分后超过目标分数 $N$，则这条路走不通，直接返回。

2.  **答错**：
    * 分数不变。
    * 错误次数 `wrongCount + 1`。
    * **判定逻辑**：
        * 如果 `wrongCount + 1` **达到 3**：考试立即触发熔断机制结束。此时检查 `score` 是否等于目标 $N$。如果相等，则找到一种情况，返回 1；否则返回 0。注意：达到3次错误时，不需要继续进入下一题。
        * 如果 `wrongCount + 1` **小于 3**：进入下一题 `index + 1`。

**边界条件（递归出口）：**

1.  如果 `index == 25`（所有题目做完）：
    * 此时肯定没有触发3次错误的熔断。
    * 检查 `score` 是否等于目标 $N$。如果相等，返回 1；否则返回 0。

### 3. 代码实现

我们将题目分值预处理为一个数组，方便根据 `index` 获取分值。

#### Java 实现

```java
import java.util.Scanner;

public class Main {
    // 定义每道题的分数
    static int[] scores = new int[25];
    static int targetScore;
    static int resultCount = 0;

    public static void main(String[] args) {
        // 初始化题目分数
        // 0-9: 2分, 10-19: 4分, 20-24: 8分
        for (int i = 0; i < 10; i++) scores[i] = 2;
        for (int i = 10; i < 20; i++) scores[i] = 4;
        for (int i = 20; i < 25; i++) scores[i] = 8;

        Scanner scanner = new Scanner(System.in);
        if (scanner.hasNextInt()) {
            targetScore = scanner.nextInt();
            // 开启DFS
            dfs(0, 0, 0);
            System.out.println(resultCount);
        }
    }

    /**
     * DFS 搜索
     * @param index 当前题号 (0-24)
     * @param currentScore 当前得分
     * @param wrongCount 当前错题数
     */
    public static void dfs(int index, int currentScore, int wrongCount) {
        // 剪枝：如果当前分数已经超过目标，直接返回
        if (currentScore > targetScore) return;

        // Base Case: 25道题全部答完
        if (index == 25) {
            if (currentScore == targetScore) {
                resultCount++;
            }
            return;
        }

        // 情况1：当前题答对
        dfs(index + 1, currentScore + scores[index], wrongCount);

        // 情况2：当前题答错
        if (wrongCount + 1 == 3) {
            // 如果这是第3个错误，考试直接结束
            // 此时分数不再增加，直接判断当前分数是否达标
            if (currentScore == targetScore) {
                resultCount++;
            }
            // 注意：这里return了，不再进行下一题
        } else {
            // 如果错误未达3次，继续下一题
            dfs(index + 1, currentScore, wrongCount + 1);
        }
    }
}
```

#### Python 实现

```python
import sys

# 初始化分数列表
scores = [2] * 10 + [4] * 10 + [8] * 5
target = 0
ans = 0

def dfs(idx, current_score, wrong_count):
    global ans
    
    # 剪枝：分数超过目标
    if current_score > target:
        return

    # Base Case: 完成所有题目
    if idx == 25:
        if current_score == target:
            ans += 1
        return

    # 选择1：答对
    # 下一题，分数增加，错误数不变
    dfs(idx + 1, current_score + scores[idx], wrong_count)

    # 选择2：答错
    new_wrong = wrong_count + 1
    if new_wrong == 3:
        # 触发熔断，考试结束
        # 检查当前分数是否符合目标
        if current_score == target:
            ans += 1
        # 此时不能继续递归下一题
    else:
        # 未触发熔断，继续下一题
        dfs(idx + 1, current_score, new_wrong)

def main():
    global target
    try:
        input_val = sys.stdin.read().strip()
        if not input_val:
            return
        target = int(input_val)
        
        dfs(0, 0, 0)
        print(ans)
    except Exception as e:
        print(0)

if __name__ == "__main__":
    main()
```

#### C++ 实现

```cpp
#include <iostream>
#include <vector>

using namespace std;

int scores[25];
int target;
int ans = 0;

void dfs(int idx, int currentScore, int wrongCount) {
    // 剪枝
    if (currentScore > target) return;

    // 完成所有题目
    if (idx == 25) {
        if (currentScore == target) {
            ans++;
        }
        return;
    }

    // 1. 答对当前题目
    dfs(idx + 1, currentScore + scores[idx], wrongCount);

    // 2. 答错当前题目
    if (wrongCount + 1 == 3) {
        // 累计3个错误，直接中断
        if (currentScore == target) {
            ans++;
        }
        // 不再递归下一题
    } else {
        // 错误未满3个，继续
        dfs(idx + 1, currentScore, wrongCount + 1);
    }
}

int main() {
    // 初始化分数
    for(int i=0; i<10; i++) scores[i] = 2;
    for(int i=10; i<20; i++) scores[i] = 4;
    for(int i=20; i<25; i++) scores[i] = 8;

    if (cin >> target) {
        dfs(0, 0, 0);
        cout << ans << endl;
    }

    return 0;
}
```
