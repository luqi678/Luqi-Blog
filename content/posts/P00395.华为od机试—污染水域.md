## 二、 题目描述

输入一行字符串，字符串可以转换为 $N \times N$ 的数组。数组可认为是一个水域，判断多少天后，水域被全部污染。
数组中只有 0 和 1，0 表示纯净，1 表示污染。每天污染源（1）只可污染其上下左右相邻的纯净水域（0）。
如果开始时全部被污染，或者永远无法全部被污染，则返回 -1。

## 二、输入描述

输入为一个由逗号分隔的字符串，表示 $N \times N$ 矩阵的平铺形式。
例如：`1,0,1,0,0,0,1,0,1` 代表一个 $3 \times 3$ 的矩阵。

## 三、输出描述

输出一个整数，表示全部水域被污染所需的天数。如果初始全为 1 或全为 0，返回 -1。

## 四、用例

### 用例1
**输入：**
```text
1,0,1,0,0,0,1,0,1
```
**输出：**
```text
2
```
**说明：**
转化为数组为：
1 0 1
0 0 0
1 0 1
第一天后变为：
1 1 1
1 0 1
1 1 1
第二天后全部变为 1，输出 2。

### 用例2
**输入：**
```text
0,0,0,0
```
**输出：**
```text
-1
```
**说明：**
全是 0，无法扩散，返回 -1。

## 五、备注

1. 矩阵必定为正方形（$N \times N$）。
2. 扩散方向为上下左右四个方向。

## 六、题目解析

### 1. 题目分析
本题属于典型的**矩阵扩散问题**。污染源（1）每一天向四周扩散一步，询问填满整个矩阵所需的时间。这本质上是求解图中节点到最近源点的最短距离的最大值。

### 2. 解题思路
1. **数据解析**：首先将输入的逗号分隔字符串转为一维数组，计算数组长度 $L$，则矩阵边长 $N = \sqrt{L}$。将一维数组重构成 $N \times N$ 的二维矩阵，或者直接在一维数组上通过下标换算处理。
2. **算法选择**：使用**多源广度优先搜索（BFS）**。
   - 将所有初始污染点（值为1的点）的坐标加入队列。
   - 记录初始污染点的个数和总格子数。
   - 若初始全为 1 或全为 0，根据题意直接返回 -1。
3. **扩散过程**：
   - 每一层 BFS 代表 1 天。
   - 遍历当前队列中的所有污染点，尝试向四个方向扩散。
   - 如果邻居是 0，则将其变为 1，并加入队列。
   - 持续扩散直到队列为空。
4. **结果输出**：BFS 运行的总层数即为天数。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNext()) return;
        String input = sc.next();
        String[] parts = input.split(",");
        int totalElements = parts.length;
        int n = (int) Math.sqrt(totalElements);

        int[][] grid = new int[n][n];
        Queue<int[]> queue = new LinkedList<>();
        int zeros = 0;

        // 1. 初始化矩阵并记录污染源
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                grid[i][j] = Integer.parseInt(parts[i * n + j]);
                if (grid[i][j] == 1) {
                    queue.offer(new int[]{i, j});
                } else {
                    zeros++;
                }
            }
        }

        // 2. 特殊情况判断：全 0 或全 1
        if (zeros == 0 || queue.isEmpty()) {
            System.out.println("-1");
            return;
        }

        int days = -1;
        int[][] directions = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};

        // 3. 多源 BFS 扩散
        while (!queue.isEmpty()) {
            int size = queue.size();
            days++; // 每进入一层，天数加1
            for (int i = 0; i < size; i++) {
                int[] curr = queue.poll();
                int r = curr[0];
                int c = curr[1];

                for (int[] d : directions) {
                    int nr = r + d[0];
                    int nc = c + d[1];

                    // 越界检查及是否为纯净水域
                    if (nr >= 0 && nr < n && nc >= 0 && nc < n && grid[nr][nc] == 0) {
                        grid[nr][nc] = 1; // 污染
                        queue.offer(new int[]{nr, nc});
                    }
                }
            }
        }

        System.out.println(days);
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N^2)$。每个格子最多进入队列一次，处理每个格子时只查看四个相邻方向。
- **空间复杂度**：$O(N^2)$。需要存储矩阵以及 BFS 使用的队列。
