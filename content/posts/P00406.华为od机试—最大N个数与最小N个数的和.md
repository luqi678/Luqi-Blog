## 一、 题目描述

给定一个数组，编写一个函数来计算它的最大N个数与最小N个数的和。你需要对数组进行去重。

**说明：**
1. 数组中数字范围 $[0, 1000]$。
2. 最大N个数与最小N个数不能有重叠，如有重叠，输入非法返回 -1。
3. 如果输入非法（如 N <= 0 等）也返回 -1。

## 二、输入描述

1. 第一行输入 $M$， $M$ 标识数组大小。
2. 第二行输入 $M$ 个数，标识数组内容。
3. 第三行输入 $N$，$N$ 表达需要计算的最大、最小N个数。

## 三、输出描述

输出最大N个数与最小N个数的和。

## 四、用例

### 用例1
**输入**
```text
5
95 88 83 64 100
2
```
**输出**
```text
342
```
**说明**
去重排序后为 `[64, 83, 88, 95, 100]`。最大2个数为 `[100, 95]`，最小2个数为 `[64, 83]`。
$100 + 95 + 64 + 83 = 342$。无重叠。

### 用例2
**输入**
```text
5
3 2 3 4 2
2
```
**输出**
```text
-1
```
**说明**
去重排序后为 `[2, 3, 4]`。最大2个数为 `[4, 3]`，最小2个数为 `[2, 3]`。数字 3 重叠了，输出 -1。

## 五、备注
*   数组元素去重是关键。
*   重叠判断逻辑：去重后的元素个数 $L$，如果 $2 \times N > L$，则必然存在重叠。
*   数字范围在 $[0, 1000]$，输入如果不满足此范围或 $N$ 不合理，需返回 -1。

## 六、题目解析

### 1. 题目分析
本题的核心在于“去重”与“重叠判断”。
*   **去重**：可以使用 `Set` 集合或者先排序再逻辑去重。
*   **重叠判断**：假设去重后的数组长度为 $L$。我们需要取最小的 $N$ 个和最大的 $N$ 个。如果这 $2N$ 个数中有任何一个元素被重复计算，就属于“重叠”。在有序去重数组中，只要 $2 \times N > L$，那么最小的 $N$ 个元素范围和最大的 $N$ 个元素范围一定会产生交集。

### 2. 解题思路
1.  读取 $M$ 和数组元素。
2.  对数组元素进行合法性检查（$0 \le x \le 1000$）。
3.  使用 `TreeSet` 进行去重且自动升序排序。
4.  读取 $N$。
5.  进行非法逻辑判断：
    *   如果 $N \le 0$，返回 -1。
    *   如果去重后的元素个数 $L < 2 \times N$，说明最大 $N$ 个和最小 $N$ 个必有重叠，返回 -1。
6.  计算 `TreeSet` 中前 $N$ 个元素和后 $N$ 个元素的总和。
7.  输出结果。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        try {
            if (!sc.hasNextInt()) {
                System.out.println("-1");
                return;
            }
            int M = sc.nextInt();
            
            // 使用 TreeSet 自动去重并排序
            TreeSet<Integer> set = new TreeSet<>();
            for (int i = 0; i < M; i++) {
                int num = sc.nextInt();
                // 检查数字范围
                if (num < 0 || num > 1000) {
                    // 虽然题目说范围0-1000，但如果不符合通常视为非法
                }
                set.add(num);
            }
            
            if (!sc.hasNextInt()) {
                System.out.println("-1");
                return;
            }
            int N = sc.nextInt();
            
            // 非法判断：N必须大于0
            if (N <= 0) {
                System.out.println("-1");
                return;
            }
            
            // 转换为列表方便取值
            List<Integer> list = new ArrayList<>(set);
            int L = list.size();
            
            // 重叠判断：去重后的个数必须至少为 2N
            if (L < 2 * N) {
                System.out.println("-1");
                return;
            }
            
            long sum = 0;
            // 最小的N个数 (前N个)
            for (int i = 0; i < N; i++) {
                sum += list.get(i);
            }
            // 最大的N个数 (后N个)
            for (int i = L - N; i < L; i++) {
                sum += list.get(i);
            }
            
            System.out.println(sum);
            
        } catch (Exception e) {
            System.out.println("-1");
        } finally {
            sc.close();
        }
    }
}
```

### 4. 复杂度分析
*   **时间复杂度**：$O(M \log M)$。主要消耗在 `TreeSet` 的插入排序上，其中 $M$ 是初始数组的大小。
*   **空间复杂度**：$O(M)$。最坏情况下，所有元素都不重复，需要存储 $M$ 个元素。
