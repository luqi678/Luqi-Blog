## 一、 题目描述

小明负责公司年会，想出一个趣味游戏：屏幕给出 1~9 中任意 3 个不重复的数字，大家以最快时间给出这几个数字可拼成的数字从小到大排列位于第 N 位置的数字，其中 N 为给出的数字中最大的（如果不到这么多个数字则给出最后一个即可），谁最快给出谁得奖。

**注意：**

1. 屏幕如果给出的是 “2”，大家可把它当作 “2”，也可把它当作 “5” 来拼接数字；同理，如果屏幕给的是 “5”，大家可把它当作 “5”，也可以把它当作 “2” 来拼接数字，但屏幕不能同时给出 “2” 和 “5”。
2. 屏幕如果给出的是 “6”，大家可把它当作 “6”，也可把它当作 “9” 来拼接数字；同理，如果屏幕给的是 “9”，大家可把它当作 “9”，也可以把它当作 “6” 来拼接数字，但屏幕不能同时给出 “6” 和 “9”。
3. 现在需要编写一个小程序，根据给出的数字计算出能组合的所有数字以及最终的正确答案。

**示例：**
给出：`1, 4, 8`
则可以拼成的数字为：`1, 4, 8, 14, 18, 41, 48, 81, 84, 148, 184, 418, 481, 814, 841`
那么第 N（即 8）个数字为 `81`。

## 二、输入描述

以逗号为分隔，描述 3 个 int 类型整数的字符串。

## 三、输出描述

这几个数字可拼成的数字从小到大排列位于第 N（N 为输入数字中最大的数字）位置的数字。
如果输入的数字为负数或者不是合法的字符串或者有重复，返回 -1。

## 四、用例

### 用例1
输入：
`1,4,8`
输出：
`81`

### 用例2
输入：
`1,2,4`
输出：
`5`
*解析：1位数字有 1, 2, 4, 5。第4个（max(1,2,4)=4）是5。*

## 五、备注
1. 输入固定为3个数字。
2. 数字范围1-9。
3. 2和5互换，6和9互换。

## 六、题目解析

### 1. 题目分析
本题的核心在于“排列组合”以及“特殊字符映射”。
- **合法性检查**：必须是3个数字，1-9之间，不能重复，不能同时出现2和5或6和9（虽然题目说屏幕不会给出，但逻辑上要严谨）。
- **数字映射**：如果输入包含2，则在该位置可以填2或5；同理5可填5或2。6和9同理。
- **生成规则**：可以组成 1 位数、2 位数、3 位数。
- **目标**：找排序后的第 $N$ 位，其中 $N = \max(input)$。

### 2. 解题思路
1. **数据准备**：将输入的3个数字存入列表。找到其中的最大值 $N$。
2. **构建备选集**：对于每一个输入的数字 $d$，建立一个可选列表。例如输入 2，备选集为 `[2, 5]`；输入 1，备选集为 `[1]`。
3. **递归生成（DFS）**：
   - 选取 1 个位置、2 个位置、3 个位置进行排列。
   - 在排列选定原始数字索引后，再针对每个索引从其对应的“可选集”中挑选数字。
   - 将生成的数字放入 `TreeSet` 进行去重和自动排序。
4. **输出结果**：从 `TreeSet` 中转换成列表，取第 $N$ 个。如果列表长度不足 $N$，取最后一个。

### 3. 代码实现

#### Java实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextLine()) {
            System.out.println("-1");
            return;
        }
        String input = sc.nextLine();
        String[] parts = input.split(",");

        // 1. 合法性检查：必须是3个数字
        if (parts.length != 3) {
            System.out.println("-1");
            return;
        }

        int[] nums = new int[3];
        Set<Integer> inputSet = new HashSet<>();
        int maxN = -1;

        try {
            for (int i = 0; i < 3; i++) {
                nums[i] = Integer.parseInt(parts[i].trim());
                // 2. 合法性检查：1-9 之间，且不重复
                if (nums[i] < 1 || nums[i] > 9 || inputSet.contains(nums[i])) {
                    System.out.println("-1");
                    return;
                }
                inputSet.add(nums[i]);
                if (nums[i] > maxN) maxN = nums[i];
            }
        } catch (NumberFormatException e) {
            System.out.println("-1");
            return;
        }

        // 检查 2,5 和 6,9 是否同时出现
        if ((inputSet.contains(2) && inputSet.contains(5)) || (inputSet.contains(6) && inputSet.contains(9))) {
            System.out.println("-1");
            return;
        }

        // 3. 构建每个输入位置的可选数字
        List<List<Integer>> candidateLists = new ArrayList<>();
        for (int num : nums) {
            List<Integer> options = new ArrayList<>();
            options.add(num);
            if (num == 2) options.add(5);
            else if (num == 5) options.add(2);
            else if (num == 6) options.add(9);
            else if (num == 9) options.add(6);
            candidateLists.add(options);
        }

        TreeSet<Integer> resultSet = new TreeSet<>();

        // 4. 生成所有排列（1位数, 2位数, 3位数）
        // 选择原始数字的索引组合
        for (int len = 1; len <= 3; len++) {
            generatePermutations(candidateLists, new boolean[3], 0, len, 0, resultSet);
        }

        // 5. 按照题目要求输出
        List<Integer> sortedList = new ArrayList<>(resultSet);
        if (maxN > sortedList.size()) {
            System.out.println(sortedList.get(sortedList.size() - 1));
        } else {
            System.out.println(sortedList.get(maxN - 1));
        }
    }

    /**
     * @param candidates 每个原始输入数字对应的变换可能
     * @param used 标记原始输入的哪几个数字被使用了
     * @param currentLen 当前数字位数
     * @param targetLen 目标数字位数
     * @param currentVal 当前拼接的数字值
     * @param resultSet 结果集
     */
    private static void generatePermutations(List<List<Integer>> candidates, boolean[] used, int currentLen, int targetLen, int currentVal, TreeSet<Integer> resultSet) {
        if (currentLen == targetLen) {
            resultSet.add(currentVal);
            return;
        }

        for (int i = 0; i < 3; i++) {
            if (!used[i]) {
                used[i] = true;
                // 对于原始输入的第 i 个数字，尝试它所有的变换可能（如 2 变 5）
                for (int option : candidates.get(i)) {
                    generatePermutations(candidates, used, currentLen + 1, targetLen, currentVal * 10 + option, resultSet);
                }
                used[i] = false;
            }
        }
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：输入规模固定为3。生成的最大排列数为 $P(3,1) \times 2^1 + P(3,2) \times 2^2 + P(3,3) \times 2^3 = 3 \times 2 + 6 \times 4 + 6 \times 8 = 6 + 24 + 48 = 78$。复杂度为常数级 $O(1)$。
- **空间复杂度**：存储生成的数字，空间开销极小，为 $O(1)$。
