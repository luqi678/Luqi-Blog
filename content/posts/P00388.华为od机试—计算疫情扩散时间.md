## 二、 题目描述

请根据给定的地图计算，多少天以后，全部区域都会被感染。

- 如果初始地图上所有区域全部都被感染，或者没有被感染区域，返回 -1。
- 0 表示未感染区域，1 表示已经感染区域。
- 每过一天，每一个感染区域（1）会向它上、下、左、右四个相邻的未感染区域（0）扩散。

## 二、 输入描述

一行 $N \times N$ 个数字（只包含 0, 1，不会有其他数字）表示一个地图，数字间用 `,` 分割。
每 $N$ 个数字表示地图中一行，输入数据共表示 $N$ 行 $N$ 列的区域地图。

例如输入 `1,0,1,0,0,0,1,0,1`，表示地图：
```
1, 0, 1
0, 0, 0
1, 0, 1
```

## 三、 输出描述

一个整数，表示经过多少天以后，全部区域都被感染。

## 四、 用例

### 用例1
**输入**
`1,0,1,0,0,0,1,0,1`
**输出**
`2`
**说明**
1天以后，地图中仅剩余中心点未被感染；2天以后，全部被感染。

### 用例2
**输入**
`0,0,0,0`
**输出**
`-1`
**说明**
无感染区域。

### 用例3
**输入**
`1,1,1,1,1,1,1,1,1`
**输出**
`-1`
**说明**
全部都感染。

## 五、 备注
$1 \le N < 200$

## 六、 题目解析

### 1. 题目分析
本题属于典型的**网格扩散问题**。给定一个二维矩阵，初始时有若干个点已被感染（值为 1），每天感染会向四周（上下左右）扩散一步。我们需要求出让所有格子都变成 1 所需的时间。

### 2. 解题思路
此类“扩散”或“最短路径”问题，最优解法是 **广度优先搜索（BFS）**。
- **多源 BFS**：由于初始时可能有多个感染源（1），我们可以将所有的感染源同时放入队列中，作为 BFS 的起始节点。
- **天数统计**：通过按层遍历（Level-order Traversal），每一层代表一天。
- **边界条件**：
    1. 如果初始状态下全是 0（没有感染源）或全是 1（已经全部感染），按题目要求返回 -1。
    2. 计算 $N$：输入是一个平铺的一维数组，长度为 $L$，则 $N = \sqrt{L}$。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNext()) return;
        
        String input = sc.next();
        String[] parts = input.split(",");
        int totalLen = parts.length;
        int n = (int) Math.sqrt(totalLen);
        
        int[][] grid = new int[n][n];
        Queue<int[]> queue = new LinkedList<>();
        int countZero = 0;
        
        // 初始化地图并将所有初始感染点入队
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                grid[i][j] = Integer.parseInt(parts[i * n + j]);
                if (grid[i][j] == 1) {
                    queue.offer(new int[]{i, j});
                } else {
                    countZero++;
                }
            }
        }
        
        // 异常情况：全是0或全是1
        if (countZero == 0 || queue.isEmpty()) {
            System.out.println("-1");
            return;
        }
        
        int days = 0;
        int[][] directions = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};
        
        // 多源BFS开始扩散
        while (!queue.isEmpty() && countZero > 0) {
            int size = queue.size();
            days++; // 每一层代表一天
            for (int i = 0; i < size; i++) {
                int[] curr = queue.poll();
                int r = curr[0];
                int c = curr[1];
                
                for (int[] dir : directions) {
                    int nr = r + dir[0];
                    int nc = c + dir[1];
                    
                    // 如果在边界内且未被感染
                    if (nr >= 0 && nr < n && nc >= 0 && nc < n && grid[nr][nc] == 0) {
                        grid[nr][nc] = 1;
                        countZero--;
                        queue.offer(new int[]{nr, nc});
                    }
                }
            }
        }
        
        System.out.println(days);
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N^2)$。每个格子最多入队和出队一次，且每个格子只会遍历 4 个方向。
- **空间复杂度**：$O(N^2)$。最坏情况下，队列中可能存储所有的格子。
