## 二、题目描述

去除文本中的多余空格，但如果多余的空格位于一对单引号（`'`）之间，则不需要去除。
在去除多余空格后，需要根据原始给出的关键词起始和结束下标，计算并输出这些关键词在处理后文本中的新起始和结束下标。

**规则说明：**
1. 多余空格定义：非单引号包裹的区域，如果出现连续的多个空格，只保留一个。
2. 单引号成对出现，可能有多对。
3. 关键词的下标保证涵盖一个完整的单词，且关键词内部（起始到结束）不会有多余空格（除非在单引号内）。
4. 关键词起始和结束位置本身不会是空格。

## 二、输入描述

输入为两行字符串：
1. 第一行：待处理的文本。
2. 第二行：关键词的原始开始和结束下标对，下标对之间以逗号（`,`）分隔，每一对的开始和结束位置以单个空格分隔。

## 三、输出描述

输出为两行字符串：
1. 第一行：处理（去除多余空格）后的文本。
2. 第二行：处理后关键词的新坐标对，格式为 `[start, end]`。

## 四、用例

### 用例1

**输入：**
```text
Life is painting a  picture, not doing 'a sum'
8 15,20 26,43 45
```

**输出：**
```text
Life is painting a picture, not doing 'a sum'
[8, 15][19, 25][42, 44]
```

**说明：**
`a` 和 `picture` 之间原本有两个空格，删除了一个。因此 `picture` 的起始坐标从 20 变为 19，结束坐标从 26 变为 25。单引号内的空格保留。

### 用例2

**输入：**
```text
Life is painting a picture, not doing 'a sum'
8 15,19 25,42 44
```

**输出：**
```text
Life is painting a picture, not doing 'a sum'
[8, 15][19, 25][42, 44]
```

## 五、备注

- 文本长度范围：[0, 100,000]。
- 关键词可能重复。
- 坐标更新需严格对应处理后的字符流位置。

## 六、题目解析

### 1. 题目分析

本题的核心挑战在于：
- **条件删除**：连续空格缩减为一，但要识别是否在成对单引号内部。
- **坐标映射**：由于字符被删除，原有的索引会“塌缩”。我们需要一种方法记录原字符串中每一个字符在处理后新字符串中的位置。

### 2. 解题思路

1.  **建立索引映射表**：创建一个整型数组 `oldToNew`，长度与原始字符串相同。`oldToNew[i]` 存储原字符串第 `i` 个字符在新字符串中的位置。
2.  **处理字符串**：
    - 遍历原字符串。
    - 使用布尔变量 `inQuote` 记录当前是否处于单引号内部。
    - 如果当前字符是单引号，取反 `inQuote`。
    - 如果当前字符是空格：
        - 若 `inQuote` 为 `true`，保留空格。
        - 若 `inQuote` 为 `false`，且前一个字符也是非引号内的空格，则视为“多余”，不放入结果字符串中，且映射表标记为无效。
        - 否则，保留该空格。
3.  **坐标转换**：
    - 解析输入的坐标对。
    - 利用 `oldToNew` 数组，将原起始 `start` 映射为 `oldToNew[start]`，原结束 `end` 映射为 `oldToNew[end]`。
4.  **格式化输出**：按要求输出处理后的字符串及新坐标序列。

### 3. 代码实现

#### Java 实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System. someIn);
        if (!sc.hasNextLine()) return;
        String text = sc.nextLine();
        if (!sc.hasNextLine()) return;
        String coordsStr = sc.nextLine();

        StringBuilder resultText = new StringBuilder();
        int[] oldToNew = new int[text.length()];
        boolean inQuote = false;

        for (int i = 0; i < text.length(); i++) {
            char c = text.charAt(i);
            
            // 处理单引号状态切换
            if (c == '\'') {
                inQuote = !inQuote;
            }

            // 判断是否为需要去除的多余空格
            boolean isExtraSpace = false;
            if (c == ' ' && !inQuote) {
                // 如果当前是空格且不在引号内，检查前一个字符
                if (i > 0 && text.charAt(i - 1) == ' ' && !isBeforeInQuote(text, i-1)) {
                    isExtraSpace = true;
                }
            }

            if (!isExtraSpace) {
                oldToNew[i] = resultText.length();
                resultText.append(c);
            } else {
                // 多余空格，在映射表中可以记为-1或前一个有效位置
                oldToNew[i] = -1; 
            }
        }

        // 输出处理后的文本
        System.out.println(resultText.toString());

        // 处理坐标
        String[] pairs = coordsStr.split(",");
        StringBuilder coordsResult = new StringBuilder();
        for (String pair : pairs) {
            String[] parts = pair.trim().split("\\s+");
            int start = Integer.parseInt(parts[0]);
            int end = Integer.parseInt(parts[1]);
            
            // 根据映射表找到新坐标
            int newStart = oldToNew[start];
            int newEnd = oldToNew[end];
            coordsResult.append("[").append(newStart).append(", ").append(newEnd).append("]");
        }
        System.out.println(coordsResult.toString());
    }

    // 辅助函数：判断原字符串中i位置的空格是否是在引号内（逻辑复用）
    private static boolean isBeforeInQuote(String text, int index) {
        boolean in = false;
        for (int i = 0; i <= index; i++) {
            if (text.charAt(i) == '\'') in = !in;
        }
        return in;
    }
    
    // 优化后的逻辑（单次遍历即可完成映射和引号判定）
}
```

**重写更简洁的逻辑：**

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String text = sc.nextLine();
        String coordsStr = sc.nextLine();

        StringBuilder sb = new StringBuilder();
        int n = text.length();
        int[] oldToNew = new int[n];
        boolean inQuote = false;
        
        for (int i = 0; i < n; i++) {
            char c = text.charAt(i);
            if (c == '\'') {
                oldToNew[i] = sb.length();
                sb.append(c);
                inQuote = !inQuote;
            } else if (c == ' ') {
                if (inQuote) {
                    oldToNew[i] = sb.length();
                    sb.append(c);
                } else {
                    // 非引号内空格，检查是否连续
                    if (i > 0 && text.charAt(i-1) == ' ' && !isPrevInQuote(text, i-1)) {
                        oldToNew[i] = -1; // 标记为被删除
                    } else {
                        oldToNew[i] = sb.length();
                        sb.append(c);
                    }
                }
            } else {
                oldToNew[i] = sb.length();
                sb.append(c);
            }
        }
        
        // 由于 isPrevInQuote 效率低，我们直接在主循环里用 inQuote 状态
        // 重新优化主循环：
        sb = new StringBuilder();
        inQuote = false;
        for (int i = 0; i < n; i++) {
            char c = text.charAt(i);
            boolean currentInQuote = inQuote;
            if (c == '\'') inQuote = !inQuote;

            if (c == ' ' && !currentInQuote) {
                if (i > 0 && text.charAt(i-1) == ' ' && !isCharInQuote(text, i-1)) {
                    oldToNew[i] = -1;
                    continue;
                }
            }
            oldToNew[i] = sb.length();
            sb.append(c);
        }

        System.out.println(sb.toString());
        // 解析坐标同上...
    }
    
    // 判定某个位置是否在引号内的辅助逻辑需要非常严谨
    // 最稳妥的方法是预处理出一个 boolean[] isInQuoteMap
}
```

### 4. 复杂度分析

-   **时间复杂度**：$O(N + M)$，其中 $N$ 是字符串长度，$M$ 是关键词坐标的个数。我们只需对字符串进行一次线性扫描建立映射。
-   **空间复杂度**：$O(N)$，需要一个额外的映射数组来记录每个字符的新旧索引关系。
