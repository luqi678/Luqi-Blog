## 一、 题目描述

在一个网络文件存储系统中，有众多的文件按照不同的文件夹进行分类存储。现在要对这些文件进行整理排序，以便便于快速地查找特定文件。每个文件都有一个创建的时间戳（用整数标识，代表从特定时刻到文件创建时所经历的秒数），并且文件夹之间存在层级关系。

现要求实现一个函数，对给定的某个文件夹下的所有文件（包含子文件夹中的文件）按照创建时间戳进行排序。

## 二、输入描述

1. 首先输入一个整数 `n`，表示接下来要输入的文件及文件夹信息的行数。
2. 然后输入一个指定文件夹的名称，用于说明获取哪个文件夹下的全部文件并进行排序。
3. 最后逐行输入文件或文件夹信息，行数为 `n`。
    *   对于文件，每行格式为：`文件名 创建时间戳`
    *   对于文件夹，每行格式为：`文件夹名 -1`（-1表示这是一个文件夹）。
    *   层级关系通过缩进（四个短横线 `----`）表示。例如，子文件或子文件夹的信息相比其父文件夹缩进 4 个短横线。

## 三、输出描述

输出排序后的文件列表，每行格式为：`文件名 创建时间戳`。
按照时间戳从小到大进行排序。
如果文件系统中不存在该文件夹，或者该文件夹下没有任何文件，则输出字符串 "No file"。

**补充说明：**
*   文件和文件夹名称只会出现 A-Z, a-z, 0-9 和字符 `.`。
*   不会出现特殊字符。
*   不会出现相同时间戳的文件。
*   不会出现重名的文件或文件夹。

## 四、用例

### 用例1

**输入：**
```text
5
Documents
Documents -1
----file1.txt 1600000000
----file2.txt 1600000100
----SubFolder1 -1
--------file3.txt 1600002000
```

**输出：**
```text
file1.txt 1600000000
file2.txt 1600000100
file3.txt 1600002000
```

### 用例2

**输入：**
```text
5
Video
Video -1
----SubFolder0 -1
----SubFolder1 -1
----SubFolder3 -1
--------SubFolder4 -1
```

**输出：**
```text
No file
```

## 五、备注
1. 需要正确处理前缀 `----` 带来的层级深度。
2. 需要递归或者使用栈来建立文件夹的父子关系。
3. 最后在指定目录下递归搜寻所有文件。

## 六、题目解析

### 1. 题目分析
本题的核心在于**解析层级化的文本输入并重建树形结构**。
*   **层级判定**：前缀每有 4 个 `-` 代表深度增加一级。
*   **结构识别**：`-1` 结尾的是文件夹，正整数结尾的是文件及其时间戳。
*   **查找范围**：题目要求的是指定文件夹下的**所有**文件（递归包含子目录）。

### 2. 解题思路
1.  **定义节点结构**：创建一个 `Node` 类，包含名称、类型（文件/文件夹）、时间戳、以及子节点列表。
2.  **构建树**：
    *   使用一个栈（Stack）来记录当前解析到的路径。
    *   遍历 `n` 行输入。对每一行，计算其深度 `level = (横杠数量 / 4)`。
    *   如果当前栈的大小大于 `level`，说明当前行不再是栈顶节点的子节点，需要不断弹出栈顶，直到栈的大小等于 `level`。
    *   解析当前行的名称和值（时间戳）。
    *   将当前节点加入到此时栈顶节点的子节点列表中。
    *   如果是文件夹，将其压入栈中。
3.  **检索与排序**：
    *   在构建好的树中找到名为 `targetFolder` 的节点。
    *   使用深度优先搜索（DFS）遍历该节点，收集所有文件。
    *   对收集到的文件列表按时间戳进行升序排序。
    *   按格式输出，若列表为空则输出 `No file`。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    static class Node {
        String name;
        long timestamp;
        boolean isFolder;
        List<Node> children = new ArrayList<>();

        Node(String name, long timestamp, boolean isFolder) {
            this.name = name;
            this.timestamp = timestamp;
            this.isFolder = isFolder;
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) return;
        int n = Integer.parseInt(sc.nextLine());
        String targetFolderName = sc.nextLine();

        Node rootPlaceholder = new Node("root", -1, true);
        Stack<Node> stack = new Stack<>();
        stack.push(rootPlaceholder);

        Node targetNode = null;

        for (int i = 0; i < n; i++) {
            String line = sc.nextLine();
            int dashCount = 0;
            while (dashCount < line.length() && line.charAt(dashCount) == '-') {
                dashCount++;
            }
            int level = dashCount / 4;
            String content = line.substring(dashCount);
            String[] parts = content.split(" ");
            String name = parts[0];
            long val = Long.parseLong(parts[1]);

            Node newNode = new Node(name, val, val == -1);

            // 维护栈，使得栈顶始终是当前节点的父节点
            // rootPlaceholder level是0，第一层节点level也是0? 
            // 修正逻辑：根据dashCount/4，第一行Documents没有杠，level=0。
            // 此时栈里有root(假)，我们需要把Documents挂在root下。
            while (stack.size() > level + 1) {
                stack.pop();
            }

            stack.peek().children.add(newNode);
            
            if (newNode.isFolder) {
                stack.push(newNode);
            }
            
            // 记录目标文件夹节点
            if (newNode.isFolder && newNode.name.equals(targetFolderName)) {
                targetNode = newNode;
            }
        }

        if (targetNode == null) {
            System.out.println("No file");
            return;
        }

        List<Node> allFiles = new ArrayList<>();
        collectFiles(targetNode, allFiles);

        if (allFiles.isEmpty()) {
            System.out.println("No file");
        } else {
            allFiles.sort(Comparator.comparingLong(o -> o.timestamp));
            for (Node f : allFiles) {
                System.out.println(f.name + " " + f.timestamp);
            }
        }
    }

    private static void collectFiles(Node node, List<Node> allFiles) {
        for (Node child : node.children) {
            if (child.isFolder) {
                collectFiles(child, allFiles);
            } else {
                allFiles.add(child);
            }
        }
    }
}
```

### 4. 复杂度分析
*   **时间复杂度**：$O(N \log N)$。解析输入并建树需要 $O(N)$，DFS 遍历需要 $O(N)$，最后的排序在最坏情况下（全是文件）需要 $O(N \log N)$。
*   **空间复杂度**：$O(N)$。主要用于存储树结构和栈。
