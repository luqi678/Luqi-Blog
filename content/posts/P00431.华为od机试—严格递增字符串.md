## 二、 题目描述

定义字符串完全由 "A" 和 "B" 组成，当然也可以全是 "A" 或全是 "B"。如果字符串从前往后都是以字典序排列的，那么我们称之为严格递增字符串。

给出一个字符串 $s$，允许修改字符串中的任意字符，即可以将任何的 "A" 修改成 "B"，也可以将任何的 "B" 修改成 "A"。求可以使 $s$ 满足严格递增的最小修改次数。

$0 < s.length < 100,000$。

## 三、 输入描述

输入一个由 'A' 和 'B' 组成的字符串。

## 四、 输出描述

输出一个整数，表示使字符串满足严格递增条件的最小修改次数。

## 五、 用例

### 用例1

**输入：**
```text
AABBA
```

**输出：**
```text
1
```

**说明：**
修改最后一位的 'A' 为 'B'，得到 "AABBB"，满足严格递增（所有 A 都在 B 前面）。

### 用例2

**输入：**
```text
BAAAB
```

**输出：**
```text
1
```

**说明：**
修改开头的 'B' 为 'A'，得到 "AAAAB"，满足要求。

## 六、 题目解析

### 1. 题目分析

题目要求将一个只包含 'A' 和 'B' 的字符串变成“严格递增”的状态。在字典序中，'A' < 'B'。
因此，一个满足条件的字符串必然呈现如下结构：
- 全是 'A'（如 `AAAAA`）
- 全是 'B'（如 `BBBBB`）
- 一部分 'A' 后面跟着一部分 'B'（如 `AAABBB`）

本质上，我们需要找到一个“分割点”，使得分割点左侧（包括分割点）全部是 'A'，分割点右侧全部是 'B'。

### 2. 解题思路

我们可以使用**动态规划**或者**前后缀统计**的思想。

**方法一：动态规划**
设 $dp[i][0]$ 表示处理完前 $i$ 个字符，且第 $i$ 个字符最终为 'A' 时的最小修改次数。
设 $dp[i][1]$ 表示处理完前 $i$ 个字符，且第 $i$ 个字符最终为 'B' 时的最小修改次数。

转移方程：
1. 如果第 $i$ 位要变成 'A'，那么前一位 $i-1$ 必须也是 'A'（否则不满足 A 在 B 前）：
   - $dp[i][0] = dp[i-1][0] + (s[i] == 'B' ? 1 : 0)$
2. 如果第 $i$ 位要变成 'B'，那么前一位 $i-1$ 既可以是 'A' 也可以是 'B'，取最小值：
   - $dp[i][1] = \min(dp[i-1][0], dp[i-1][1]) + (s[i] == 'A' ? 1 : 0)$

**方法二：统计前缀B和后缀A**
假设我们在位置 $i$ 划线，左侧全部变 A，右侧全部变 B。
总修改次数 = 左侧 'B' 的数量 + 右侧 'A' 的数量。
通过遍历所有可能的分割点（共 $n+1$ 个位置），找这个和的最小值。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNext()) return;
        String s = sc.next();
        int n = s.length();

        // dp0[i] 表示前 i 个字符变为严格递增且以 A 结尾的最小代价
        // dp1[i] 表示前 i 个字符变为严格递增且以 B 结尾的最小代价
        int dp0 = 0;
        int dp1 = 0;

        for (int i = 0; i < n; i++) {
            char c = s.charAt(i);
            
            // 计算当前位如果是 B 的代价
            // 当前位是 B，前一位可以是 A 也可以是 B，取其小者
            int nextDp1 = Math.min(dp0, dp1) + (c == 'A' ? 1 : 0);
            
            // 计算当前位如果是 A 的代价
            // 当前位是 A，前一位只能是 A
            int nextDp0 = dp0 + (c == 'B' ? 1 : 0);
            
            dp0 = nextDp0;
            dp1 = nextDp1;
        }

        System.out.println(Math.min(dp0, dp1));
    }
}
```

#### 另一种思路：前缀统计 (更直观)

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String s = sc.next();
        int n = s.length();

        // 统计总共的 A 的个数
        int totalA = 0;
        for (char c : s.toCharArray()) {
            if (c == 'A') totalA++;
        }

        // 初始状态：全部改为 B，则修改次数为总的 A 的个数
        int minChanges = totalA; 
        
        int currentB = 0;
        int currentA = 0;
        
        // 遍历每一个分割点，分割点 i 表示前 i 个字符属于 A 区
        for (int i = 0; i < n; i++) {
            if (s.charAt(i) == 'A') {
                currentA++;
            } else {
                currentB++;
            }
            // 需要修改的次数 = 左边的B变A (currentB) + 右边的A变B (totalA - currentA)
            int changes = currentB + (totalA - currentA);
            minChanges = Math.min(minChanges, changes);
        }

        System.out.println(minChanges);
    }
}
```

### 4. 复杂度分析

- **时间复杂度**：$O(n)$，其中 $n$ 是字符串的长度。我们只需要遍历一遍字符串。
- **空间复杂度**：$O(1)$ 或 $O(n)$。如果使用动态规划并进行空间优化（如上面的实现），空间复杂度为 $O(1)$（不计输入字符串占用的空间）。
