## 二、题目描述

给定一个可存储若干单词的字典，找出指定单词的所有相似单词，并且按照单词名称从小到大排序输出。

单词仅包括字母，但可能大小写并存 (大写不一定只出现在首字母)。

**相似单词说明：**
给定一个单词 X，如果通过任意交换单词中字母的位置得到不同的单词 Y，那么定义 Y 是 X 的相似单词。在本题的实际执行中，如果两个单词包含的字母及其出现的次数完全相同（即为“变位词”或“异构词”），则认为它们是相似单词（包括单词本身）。

**字符排序规则：**
大小写是不同的字母，且遵循：大写字母 < 小写字母。
同样大小写的字母，遵循 26 字母顺序。
即：`A < B < C < ... < X < Y < Z < a < b < c < ... < x < y < z`。
例如：`Bac < aBc < acB < cBa`。

## 三、输入描述

1. 第一行为给定的单词个数 $N$ ($N$ 为非负整数)。
2. 从第二行到第 $N+1$ 行是具体的单词（每行一个单词）。
3. 最后一行是指定的待检测单词（用于检测上述字典中哪些是该指定单词的相似单词）。

## 四、输出描述

从给定的单词组中，找出指定单词的相似单词，并且按照从小到大字典序排列输出，中间以空格隔开。如果不存在，则输出字符串 `null`。

## 五、用例

### 用例1
**输入：**
```text
4
abc
dasd
tad
bca
abc
```
**输出：**
```text
abc bca
```

### 用例2
**输入：**
```text
4
abc
dasd
tad
bca
abd
```
**输出：**
```text
null
```

## 六、题目解析

### 1. 题目分析
本题的核心要求是：
1. **相似判定**：两个单词如果包含的字符种类及其对应频率完全一致，则互为相似单词。这在算法中通常被称为“变位词（Anagram）”。
2. **区分大小写**：'a' 和 'A' 是两个不同的字符。
3. **排序规则**：题目定义的排序规则与 ASCII 码表的顺序一致（'A'-'Z' 为 65-90，'a'-'z' 为 97-122）。因此，直接使用编程语言内置的字符串排序即可。
4. **输出处理**：如果找到结果，按字典序排序后输出，空格分隔；否则输出 `null`。

### 2. 解题思路
1. 读取字典单词数量 $N$。
2. 循环读取 $N$ 个单词存入列表中。
3. 读取目标单词 $Target$。
4. 为了快速判断是否相似：
   - 将 $Target$ 转换为字符数组并排序，得到一个“标准形式”的字符串。
5. 遍历字典中的每一个单词 $W$：
   - 如果 $W$ 的长度不等于 $Target$ 的长度，跳过。
   - 将 $W$ 转换为字符数组并排序，对比其排序后的结果是否与 $Target$ 的标准形式一致。
   - 如果一致，将 $W$ 加入结果列表。
6. 对结果列表按字典序排序。
7. 如果结果列表为空，输出 `null`；否则输出列表内容。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 处理输入：单词个数 N
        if (!sc.hasNextInt()) return;
        int n = sc.nextInt();
        
        // 存储字典单词
        List<String> dictionary = new ArrayList<>();
        for (int i = 0; i < n; i++) {
            dictionary.add(sc.next());
        }

        // 读取待检测单词
        if (!sc.hasNext()) return;
        String target = sc.next();

        // 获取目标单词的标准形式（字符排序后）
        String sortedTarget = sortString(target);

        List<String> result = new ArrayList<>();
        for (String word : dictionary) {
            // 长度不等直接排除
            if (word.length() != target.length()) {
                continue;
            }
            // 排序后比较是否相等
            if (sortString(word).equals(sortedTarget)) {
                result.add(word);
            }
        }

        // 结果处理
        if (result.isEmpty()) {
            System.out.println("null");
        } else {
            // 默认排序即符合 A-Z < a-z
            Collections.sort(result);
            StringBuilder sb = new StringBuilder();
            for (int i = 0; i < result.size(); i++) {
                sb.append(result.get(i));
                if (i < result.size() - 1) {
                    sb.append(" ");
                }
            }
            System.out.println(sb.toString());
        }
    }

    /**
     * 将字符串中的字符按照 ASCII 序排序
     */
    private static String sortString(String s) {
        char[] chars = s.toCharArray();
        Arrays.sort(chars);
        return new String(chars);
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N \cdot L \log L)$，其中 $N$ 是单词数量，$L$ 是单词的最大长度。每个单词都需要排序进行相似性校验。最后的输出排序复杂度为 $O(M \log M \cdot L)$，其中 $M$ 是相似单词的数量。
- **空间复杂度**：$O(N \cdot L)$，用于存储字典单词及结果。
