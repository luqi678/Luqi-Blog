## 一、 题目描述

某部门计划通过结队编程来进行项目开发，已知该部门有 `N` 名员工，每个员工有独一无二的职级，每三个员工形成一个小组进行结队编程，结队分组规则如下：

从部门中选出序号分别为 `i`、`j`、`k` 的 3 名员工，他们的职级分别为 `level[i]`，`level[j]`，`level[k]`。
结队小组需满足以下两种情况之一：
1.  **递增**：`level[i] < level[j] < level[k]`
2.  **递减**：`level[i] > level[j] > level[k]`

其中索引满足 `0 <= i < j < k < n`。

请你按上述条件计算可能组合的小组数量。同一员工可以参加多个小组。

## 二、输入描述

1.  第一行输入：员工总数 `n`。
2.  第二行输入：按序号依次排列的员工的职级 `level`，中间用空格隔开。

**约束：**
- $1 \le n \le 6000$
- $1 \le level[i] \le 10^5$

## 三、输出描述

输出可能结队的小组数量。

## 四、备注

- 题目中的核心难点在于时间复杂度的控制。
- 暴力解法是三层循环枚举 `i, j, k`，复杂度为 $O(N^3)$。由于 $N$ 最大为 6000，$N^3 \approx 2.16 \times 10^{11}$，这在一般的机考限时（1秒/10^8次运算）内会**超时**。
- 优化思路是**枚举中间节点 `j`**。对于固定的 `j`，我们只需要统计它左边有多少个比它小的/大的，右边有多少个比它大的/小的，然后利用乘法原理计算组合数。该方法复杂度为 $O(N^2)$，$6000^2 = 3.6 \times 10^7$，完全符合要求。
- **注意**：结果数量可能很大，建议使用 `long` 类型存储统计结果。

## 五、用例

| 输入 | 输出 | 说明 |
| :--- | :--- | :--- |
| `4`<br>`1 2 3 4` | `4` | 满足条件的组合有：<br>1. (1,2,3)<br>2. (1,2,4)<br>3. (1,3,4)<br>4. (2,3,4)<br>全部为递增序列。 |
| `3`<br>`5 4 7` | `0` | 只有3个人，顺序是 [5, 4, 7]。<br>递增？5<4 False。<br>递减？5>4 True, 但 4>7 False。<br>无法组成任何满足条件的小组。 |

## 六、题目解析

### 核心思路 (中心扩展法)

我们需要找到满足 $L[i] < L[j] < L[k]$ 或 $L[i] > L[j] > L[k]$ 的三元组。

如果我们遍历中间的元素 `j`（索引从 `1` 到 `n-2`），问题就转化为了：
1.  在 `j` 的**左边** ($0 \dots j-1$)，有多少个元素比 `level[j]` 小？记为 `leftSmall`。
2.  在 `j` 的**左边** ($0 \dots j-1$)，有多少个元素比 `level[j]` 大？记为 `leftBig`。
3.  在 `j` 的**右边** ($j+1 \dots n-1$)，有多少个元素比 `level[j]` 大？记为 `rightBig`。
4.  在 `j` 的**右边** ($j+1 \dots n-1$)，有多少个元素比 `level[j]` 小？记为 `rightSmall`。

对于当前的 `j`，能组成的组合数为：
* **递增组合数**：`leftSmall * rightBig`
* **递减组合数**：`leftBig * rightSmall`

将所有位置 `j` 的这两个积累加，即为最终结果。

### 算法流程
1.  读取 `n` 和 `level` 数组。
2.  初始化 `count = 0` (使用 long)。
3.  遍历 `j` 从 `1` 到 `n-2`：
    * 初始化四个计数器为0。
    * 内层循环1：遍历 `i` 从 `0` 到 `j-1`，统计 `leftSmall` 和 `leftBig`。
    * 内层循环2：遍历 `k` 从 `j+1` 到 `n-1`，统计 `rightSmall` 和 `rightBig`。
    * `count += (leftSmall * rightBig) + (leftBig * rightSmall)`。
4.  输出 `count`。

### 算法实现 (Java)

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // 读取员工总数
        if (!sc.hasNextInt()) return;
        int n = sc.nextInt();
        
        // 读取职级数组
        int[] levels = new int[n];
        for (int i = 0; i < n; i++) {
            levels[i] = sc.nextInt();
        }
        
        // N < 3 无法组成三人组
        if (n < 3) {
            System.out.println(0);
            return;
        }

        long count = 0;

        // 遍历中间节点 j
        // j 至少要是第2个元素 (index 1)，至多是倒数第2个元素 (index n-2)
        for (int j = 1; j < n - 1; j++) {
            long leftSmall = 0;
            long leftBig = 0;
            long rightSmall = 0;
            long rightBig = 0;
            
            int currentVal = levels[j];

            // 统计左边 (0 ~ j-1)
            for (int i = 0; i < j; i++) {
                if (levels[i] < currentVal) {
                    leftSmall++;
                } else if (levels[i] > currentVal) {
                    leftBig++;
                }
            }

            // 统计右边 (j+1 ~ n-1)
            for (int k = j + 1; k < n; k++) {
                if (levels[k] < currentVal) {
                    rightSmall++;
                } else if (levels[k] > currentVal) {
                    rightBig++;
                }
            }

            // 累加组合数
            // 情况1：小 中 大 (递增) -> 左小 * 右大
            // 情况2：大 中 小 (递减) -> 左大 * 右小
            count += (leftSmall * rightBig) + (leftBig * rightSmall);
        }

        System.out.println(count);
    }
}
```
