## 一、 题目描述

给定一个 $n$ 行 $m$ 列的矩阵，每个位置上有一个整数元素。
你可以从当前位置移动到上下左右相邻的位置，移动的代价是前后两个位置元素值差的绝对值。
另外，你最多可以使用一次传送阵：即你可以从当前位置直接跳到矩阵中另一个具有**相同数值**的元素位置，这次传送的代价为 0。
求从矩阵左上角 $(0, 0)$ 走到右下角 $(n-1, m-1)$ 所需要的最小代价（时间）。

## 二、 输入描述

第一行两个整数 $n, m$，分别代表矩阵的行数和列数。
接下来 $n$ 行，每行 $m$ 个整数，表示矩阵中的元素值。

## 三、 输出描述

输出一个整数，表示从左上角到右下角的最小代价。

## 四、 用例

### 示例 1
**输入：**
```text
3 3
1 2 3
4 5 6
7 8 9
```
**输出：**
```text
8
```
**说明：**
没有重复数字，无法使用传送阵。最短路径为 $1 \to 2 \to 3 \to 6 \to 9$，代价为 $|1-2| + |2-3| + |3-6| + |6-9| = 1 + 1 + 3 + 3 = 8$。

### 示例 2
**输入：**
```text
3 3
1 2 3
4 5 6
7 1 4
```
**输出：**
```text
3
```
**说明：**
从 $(0,0)$ 的元素 1 传送到 $(2,1)$ 的元素 1，代价为 0。再从 $(2,1)$ 移动到 $(2,2)$ 的元素 4，代价为 $|1-4|=3$。总代价为 $0 + 3 = 3$。

## 五、 备注
- $1 \le n, m \le 500$
- 矩阵元素值为整数，范围通常在 $[0, 10^9]$。
- 传送阵最多只能使用一次。

## 六、 题目解析

本题是一个经典的**最短路径问题**。由于移动代价是非负的（绝对值），且存在特定规则的特殊跳转（传送阵），我们可以使用 **Dijkstra 算法**。

### 核心思路：
1.  **分层图思想（或状态扩展）**：
    因为传送阵只能使用一次，我们需要在 Dijkstra 的状态中记录“是否已使用过传送阵”。
    定义状态为 `(r, c, used)`，其中：
    - `r, c`: 当前所在的行和列。
    - `used`: 0 表示未使用过传送阵，1 表示已使用过。
    总状态数为 $N \times M \times 2$。

2.  **转移方程**：
    从状态 `(r, c, used)` 出发：
    - **常规移动**：移动到相邻的 `(nr, nc)`，代价为 `abs(matrix[r][c] - matrix[nr][nc])`。状态变为 `(nr, nc, used)`。
    - **传送移动**（仅当 `used == 0` 时）：可以跳到任何 `(tr, tc)` 满足 `matrix[tr][tc] == matrix[r][c]`。代价为 0。状态变为 `(tr, tc, 1)`。

3.  **效率优化**：
    - 传送跳转如果简单遍历所有相同数值的点，复杂度会退化。
    - **优化点**：对于数值 $V$，一旦我们从某个状态 `(r, c, 0)` 触发了数值 $V$ 的传送，后续再遇到数值 $V$ 的其他点时，不需要再次触发传送。因为 Dijkstra 保证了第一次到达数值 $V$ 的点时，代价已经是最小的，后续触发不会得到更优解。
    - 预处理：使用 `HashMap<Integer, List<int[]>>` 存储每个数值对应的所有坐标。

### 代码实现 (Java)：

```java
import java.util.*;

public class Main {
    static class Node implements Comparable<Node> {
        int r, c, used;
        long dist;

        Node(int r, int c, int used, long dist) {
            this.r = r;
            this.c = c;
            this.used = used;
            this.dist = dist;
        }

        @Override
        public int compareTo(Node o) {
            return Long.compare(this.dist, o.dist);
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNext()) return;
        int n = sc.nextInt();
        int m = sc.nextInt();
        int[][] matrix = new int[n][m];
        Map<Integer, List<int[]>> posMap = new HashMap<>();

        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                matrix[i][j] = sc.nextInt();
                posMap.computeIfAbsent(matrix[i][j], k -> new ArrayList<>()).add(new int[]{i, j});
            }
        }

        long[][][] dist = new long[n][m][2];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                Arrays.fill(dist[i][j], Long.MAX_VALUE);
            }
        }

        PriorityQueue<Node> pq = new PriorityQueue<>();
        dist[0][0][0] = 0;
        pq.offer(new Node(0, 0, 0, 0));

        int[][] dirs = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};
        Set<Integer> teleportedValues = new HashSet<>();

        while (!pq.isEmpty()) {
            Node curr = pq.poll();

            if (curr.dist > dist[curr.r][curr.c][curr.used]) continue;
            if (curr.r == n - 1 && curr.c == m - 1) {
                System.out.println(curr.dist);
                return;
            }

            // 1. 常规移动
            for (int[] d : dirs) {
                int nr = curr.r + d[0];
                int nc = curr.c + d[1];
                if (nr >= 0 && nr < n && nc >= 0 && nc < m) {
                    long cost = Math.abs(matrix[curr.r][curr.c] - matrix[nr][nc]);
                    if (dist[nr][nc][curr.used] > curr.dist + cost) {
                        dist[nr][nc][curr.used] = curr.dist + cost;
                        pq.offer(new Node(nr, nc, curr.used, dist[nr][nc][curr.used]));
                    }
                }
            }

            // 2. 传送移动 (仅限从 used=0 状态发起一次)
            if (curr.used == 0) {
                int val = matrix[curr.r][curr.c];
                if (!teleportedValues.contains(val)) {
                    teleportedValues.add(val);
                    List<int[]> sameValPoints = posMap.get(val);
                    if (sameValPoints != null) {
                        for (int[] p : sameValPoints) {
                            if (dist[p[0]][p[1]][1] > curr.dist) {
                                dist[p[0]][p[1]][1] = curr.dist;
                                pq.offer(new Node(p[0], p[1], 1, dist[p[0]][p[1]][1]));
                            }
                        }
                    }
                }
            }
        }
    }
}
```

## 七、 复杂度分析
- **时间复杂度**：$O(V \log V + E)$，其中 $V = 2NM$。由于我们对每个数值的传送列表只遍历一次，传送相关的边总数也是 $O(NM)$ 级别的，因此整体效率很高。
- **空间复杂度**：$O(NM)$，用于存储矩阵、距离数组及数值坐标映射。
