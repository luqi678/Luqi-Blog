## 一、 题目描述

给定一个射击比赛成绩单，包含多个选手若干次射击的成绩分数，请对每个选手按其最高3个分数之和进行降序排名，输出降序排名后的选手ID序列。条件如下：

1. 一个选手可以有多个射击成绩的分数，且次序不固定。
2. 如果一个选手成绩少于3个，则认为选手的所有成绩无效，排名忽略该选手。
3. 如果选手的成绩之和相等，则成绩之和相等的选手按照其ID降序排列。

## 二、输入描述

- 输入第一行，一个整数N，表示该场比赛总共进行了N次射击，产生N个成绩分数（2 <= N <= 100）。
- 输入第二行，一个长度为N的整数序列（以逗号分隔），表示参与每次射击的选手ID（0 <= ID <= 99）。
- 输入第三行，一个长度为N的整数序列（以逗号分隔），表示参与每次射击的选手对应的成绩（0 <= 成绩 <= 100）。

## 四、用例

### 用例1

**输入**
```text
13
3,3,7,4,4,4,4,7,7,3,5,5,5
53,80,68,24,39,76,66,16,100,55,53,80,55
```

**输出**
```text
5,3,7,4
```

**说明**
- 3号选手成绩：53, 80, 55，最高3个和：80 + 55 + 53 = 188
- 4号选手成绩：24, 39, 76, 66，最高3个和：76 + 66 + 39 = 181
- 5号选手成绩：53, 80, 55，最高3个和：80 + 55 + 53 = 188
- 7号选手成绩：68, 16, 100，最高3个和：100 + 68 + 16 = 184
- 排名逻辑：(5号:188, 3号:188) > (7号:184) > (4号:181)。由于5号和3号总分相同，按ID降序，5在3前。

## 五、备注
- $2 \le N \le 100$
- $0 \le ID \le 99$
- $0 \le 成绩 \le 100$

## 六、题目解析

### 1. 题目分析
本题是一道典型的“数据处理+多条件排序”题目。
- 需要将输入的 ID 和分数进行关联。
- 需要对同一个 ID 的分数进行聚合，并筛选出次数 $\ge 3$ 的选手。
- 排序规则：第一关键字是前三名成绩之和（降序），第二关键字是选手 ID（降序）。

## 三、输出描述

符合题设条件的降序排名后的选手ID序列，以逗号分隔。

### 2. 解题思路
1. **数据结构选择**：使用 `HashMap<Integer, List<Integer>>` 来存储每个选手的成绩，Key 为选手 ID，Value 为其分数的列表。
2. **成绩处理**：
   - 遍历输入，将分数存入对应 ID 的 List 中。
   - 遍历 Map，剔除 List 长度小于 3 的选手。
   - 对每个选手的 List 进行降序排序，取前 3 个值求和。
3. **多条件排序**：
   - 创建一个存储 `(ID, Sum)` 的类或对象列表。
   - 实现自定义排序：
     - 若 `sum1 != sum2`，返回 `sum2 - sum1`。
     - 若 `sum1 == sum2`，返回 `id2 - id1`。
4. **输出格式**：将排序后的 ID 转换成以逗号分隔的字符串。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;
import java.util.stream.Collectors;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 1. 读取输入
        if (!sc.hasNextInt()) return;
        int n = Integer.parseInt(sc.nextLine());
        String[] idsStr = sc.nextLine().split(",");
        String[] scoresStr = sc.nextLine().split(",");

        // 2. 聚合数据
        Map<Integer, List<Integer>> playerScores = new HashMap<>();
        for (int i = 0; i < n; i++) {
            int id = Integer.parseInt(idsStr[i]);
            int score = Integer.parseInt(scoresStr[i]);
            playerScores.computeIfAbsent(id, k -> new ArrayList<>()).add(score);
        }

        // 3. 计算符合条件的选手总分
        List<Player> players = new ArrayList<>();
        for (Map.Entry<Integer, List<Integer>> entry : playerScores.entrySet()) {
            List<Integer> scores = entry.getValue();
            if (scores.size() >= 3) {
                // 降序排序
                scores.sort(Collections.reverseOrder());
                // 取前三名求和
                int sum = scores.get(0) + scores.get(1) + scores.get(2);
                players.add(new Player(entry.getKey(), sum));
            }
        }

        // 4. 多条件排序
        players.sort((a, b) -> {
            if (a.totalSum != b.totalSum) {
                return b.totalSum - a.totalSum; // 总分降序
            }
            return b.id - a.id; // ID降序
        });

        // 5. 输出结果
        String result = players.stream()
                .map(p -> String.valueOf(p.id))
                .collect(Collectors.joining(","));
        System.out.println(result);
    }

    static class Player {
        int id;
        int totalSum;

        Player(int id, int totalSum) {
            this.id = id;
            this.totalSum = totalSum;
        }
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N \log N)$。其中聚合数据为 $O(N)$，对每个选手的成绩进行内部排序最坏情况下也是 $O(N \log N)$，最后对选手列表进行排序由于选手数量最多为 $N/3$，复杂度也为 $O(N \log N)$。
- **空间复杂度**：$O(N)$。使用了 Map 和 List 存储所有的射击成绩。
