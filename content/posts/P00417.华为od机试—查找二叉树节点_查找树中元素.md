## 一、 题目描述

已知树形结构的所有节点信息，现要求根据输入坐标 $(x, y)$ 找到该节点保存的内容值。

其中：
- $x$ 表示节点所在的层数，根节点位于第 0 层，根节点的子节点位于第 1 层，依此类推。
- $y$ 表示节点在该层内的相对偏移，从左至右，第一个节点偏移 0，第二个节点偏移 1，依此类推。

**举例：**
在上图中，假定圆圈内的数字表示节点保存的内容值，则根据坐标 $(1, 1)$ 查到的内容值是 23。

## 二、输入描述

1. 第一行输入一个整数 $N$，表示节点的总数量。
2. 接下来 $N$ 行，每行表示一个节点的信息。每行第一个数字是该节点保存的**内容值**，后续数字是该节点的**子节点在二维数组中的索引位置**（索引从 0 开始）。
   - 二维数组的 0 号索引位置固定存放根节点。
3. 最后一行输入两个整数 $x$ 和 $y$，表示查询的坐标。

## 三、输出描述

- 查询到内容时，输出 `{内容值}`。
- 查询不到内容时，输出 `{}`。

## 四、用例

### 用例1

**输入：**
```text
6
10 1 2
-21 3 4
23 5
14
35
66
1 1
```

**输出：**
```text
{23}
```

### 用例2

**输入：**
```text
6
10 1 2
-21 3 4
23 5
14
35
66
1 2
```

**输出：**
```text
{}
```

## 五、备注

1. 树的节点数 $N \ge 1$。
2. 根节点始终是输入列表中的第一个元素（索引 0）。
3. 如果查询的层数 $x$ 或偏移量 $y$ 超出了树的实际范围，应输出 `{}`。

## 六、题目解析

### 1. 题目分析
本题要求在给定的通用树结构中，根据“层级”和“层内偏移量”定位节点。这是一个典型的**层序遍历（Level Order Traversal）**问题。

- **节点表示**：每个节点包含一个值和若干个指向子节点的索引。
- **层级 (x)**：代表 BFS 遍历时的深度。
- **偏移量 (y)**：代表在特定深度的 BFS 队列中的第几个元素。

### 2. 解题思路
1. **构建树模型**：可以使用一个类 `Node` 或直接使用 `List<Integer>[]` 来存储每个节点的子节点索引，并用一个数组存储每个节点的值。
2. **BFS 遍历**：
   - 使用队列 `Queue` 进行层序遍历。
   - 初始化：将根节点（索引 0）入队。
   - 迭代：
     - 记录当前层的节点数量 `size`。
     - 如果当前层数等于查询层数 $x$：
       - 判断 $y$ 是否小于 `size`。
       - 如果是，则第 $y$ 个出队的节点就是目标节点，输出其值。
       - 如果不是，说明该层节点数不足 $y+1$，输出 `{}`。
     - 如果当前层数小于 $x$：
       - 将当前层的所有子节点按顺序加入队列。
       - 层数计数加 1。
     - 如果遍历结束仍未到达第 $x$ 层，说明树的深度不足，输出 `{}`。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    static class Node {
        int val;
        List<Integer> children = new ArrayList<>();

        Node(int val) {
            this.val = val;
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        if (!sc.hasNextInt()) return;
        int n = sc.nextInt();
        sc.nextLine(); // 消耗换行符

        Node[] nodes = new Node[n];
        // 第一遍先初始化所有节点对象
        // 注意：输入可能包含多行，且每行长度不固定
        for (int i = 0; i < n; i++) {
            String line = sc.nextLine().trim();
            if (line.isEmpty()) {
                i--; // 处理可能的空行
                continue;
            }
            String[] parts = line.split("\\s+");
            int val = Integer.parseInt(parts[0]);
            nodes[i] = new Node(val);
            for (int j = 1; j < parts.length; j++) {
                nodes[i].children.add(Integer.parseInt(parts[j]));
            }
        }

        if (!sc.hasNextInt()) return;
        int targetX = sc.nextInt();
        int targetY = sc.nextInt();

        // BFS 层序遍历
        Queue<Integer> queue = new LinkedList<>();
        queue.offer(0); // 根节点索引为0

        int currentLevel = 0;
        boolean found = false;

        while (!queue.isEmpty()) {
            int size = queue.size();
            
            // 如果到达了目标层
            if (currentLevel == targetX) {
                if (targetY >= 0 && targetY < size) {
                    // 找到目标偏移量
                    int targetNodeIdx = -1;
                    for (int i = 0; i <= targetY; i++) {
                        targetNodeIdx = queue.poll();
                    }
                    System.out.println("{" + nodes[targetNodeIdx].val + "}");
                    found = true;
                }
                break; // 已经处理完目标层或目标层不存在该偏移量
            }

            // 处理当前层，将下一层节点入队
            for (int i = 0; i < size; i++) {
                int currIdx = queue.poll();
                for (int childIdx : nodes[currIdx].children) {
                    queue.offer(childIdx);
                }
            }
            currentLevel++;
            
            // 剪枝：如果当前层已经超过目标层，退出
            if (currentLevel > targetX) break;
        }

        if (!found) {
            System.out.println("{}");
        }
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N)$。在最坏情况下，我们需要遍历树的所有节点。
- **空间复杂度**：$O(N)$。需要存储树的结构（邻接表）以及 BFS 过程中的队列，队列在最宽的一层可能达到 $O(N)$ 级别。
