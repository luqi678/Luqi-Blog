## 一、 题目描述
给你一个包含 $n$ 个整数的数组 `nums`，判断 `nums` 中是否存在三个元素 $a, b, c$，使得 $a + b + c = 0$ ？请你找出所有和为 $0$ 且**不重复**的三元组。

**注意：**
1. 答案中不能包含重复的三元组。
2. 答案中可以包含重复的数字，但三元组组合本身不能重复（例如 `[-1, 0, 1]` 和 `[0, -1, 1]` 被视为同一个三元组）。

## 二、输入描述
一个整数数组 `nums`。

## 三、输出描述
所有满足条件且不重复的三元组列表。

## 四、用例
### 用例1
输入：`nums = [-1,0,1,2,-1,-4]`
输出：`[[-1,-1,2],[-1,0,1]]`

### 用例2
输入：`nums = [0,1,1]`
输出：`[]`
*(注：原题干示例 2 有误，0+1+1 != 0，此处按逻辑修正)*

### 用例3
输入：`nums = [0,0,0]`
输出：`[[0,0,0]]`

## 五、备注
*   $3 \le nums.length \le 3000$
*   $-10^5 \le nums[i] \le 10^5$

## 六、题目解析

### 1. 题目分析
本题的难点不在于寻找和为 0 的三个数（暴力 $O(N^3)$ 容易实现），而在于如何**高效地去重**。如果使用哈希表记录结果，处理重复三元组会非常麻烦。

### 2. 解题思路
最优解法是 **排序 + 双指针**：
1.  **排序**：先对数组进行升序排序。排序是去重和使用双指针的前提。
2.  **固定一个数**：遍历数组，令当前元素为 `nums[i]`。
    *   若 `nums[i] > 0`：因为数组已排序，后面的数都大于 0，三数之和必然大于 0，直接结束循环。
    *   **去重**：若 `i > 0` 且 `nums[i] == nums[i-1]`，则说明该数字已经作为第一个数处理过了，直接 `continue`。
3.  **双指针寻找另外两数**：定义左指针 `L = i + 1`，右指针 `R = n - 1`。
    *   计算 `sum = nums[i] + nums[L] + nums[R]`。
    *   若 `sum == 0`：找到一组解。
        *   **去重**：在将结果加入列表后，需要判断 `L` 和 `R` 指针指向的下一个数是否与当前数相同。若相同则跳过，防止产生重复组合。
        *   收缩左右指针：`L++`, `R--`。
    *   若 `sum < 0`：说明数值太小，左指针右移 `L++`。
    *   若 `sum > 0`：说明数值太大，右指针左移 `R--`。

### 3. 代码实现

#### Java8实现
```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        int n = nums.length;
        if (nums == null || n < 3) return result;

        // 1. 排序
        Arrays.sort(nums);

        for (int i = 0; i < n; i++) {
            // 如果第一个数已经大于0，后面不可能和为0
            if (nums[i] > 0) break;

            // 2. 对第一个数进行去重
            if (i > 0 && nums[i] == nums[i - 1]) continue;

            int L = i + 1;
            int R = n - 1;

            while (L < R) {
                int sum = nums[i] + nums[L] + nums[R];
                if (sum == 0) {
                    result.add(Arrays.asList(nums[i], nums[L], nums[R]));
                    
                    // 3. 对第二个数和第三个数进行去重
                    while (L < R && nums[L] == nums[L + 1]) L++;
                    while (L < R && nums[R] == nums[R - 1]) R--;
                    
                    L++;
                    R--;
                } else if (sum < 0) {
                    L++;
                } else {
                    R--;
                }
            }
        }
        return result;
    }

    public static void main(String[] args) {
        Solution sol = new Solution();
        int[] nums = {-1, 0, 1, 2, -1, -4};
        System.out.println(sol.threeSum(nums));
    }
}
```

### 4. 复杂度分析
*   **时间复杂度**：$O(N^2)$。其中排序 $O(N \log N)$，外层遍历 $O(N)$，内层双指针 $O(N)$，综合为 $O(N^2)$。
*   **空间复杂度**：$O(\log N)$。主要取决于排序算法的空间开销（若不计算返回结果列表的存储）。
