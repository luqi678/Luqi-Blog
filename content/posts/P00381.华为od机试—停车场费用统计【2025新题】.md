## 一、题目描述

停车场统计当日总收费，计费规则如下：
1. **包月车**：不统计费用。
2. **普通车**：
   - 每半个小时收费 1 元。
   - **起步规则**：停车时间不满半小时不收费。
   - **进阶规则**：如果停车时间超过（含）半小时，零头部分不满半小时按半小时计费（即：向上取整到半小时）。
3. **免费时段**：每天 **11:30 - 13:30** 为不收费时间段。如果停车跨越该时段，需扣除该时段内的停车时长。
4. **封顶规则**：单次停车如果计费时长超过 8 小时（480分钟），超过部分不收费（即单次计费时长上限为 480 分钟）。

提供停车场当日进出车辆的统计信息，计算当日总收费。

## 二、输入描述

1. 第一行输入一个整数 $n$，表示今日进出停车场的包月车辆数。
2. 第二行输入 $n$ 个包月车的车牌号，以空格分隔。
3. 接下来若干行（直到输入结束），输出进出停车场的车辆记录，格式如下：
   - 进入：`时间 车牌号 enter`
   - 离开：`时间 车牌号 leave`

*注：时间格式为 `HH:mm`。假设所有记录均在同一天内。*

## 三、输出描述

输出今日总收费（整数）。

## 四、用例

### 用例1
**输入：**
```text
2
AAA CCC
8:00 AAA enter
10:00 AAA leave
```
**输出：**
```text
0
```
**说明：** AAA 是包月车，不收费。

### 用例2
**输入：**
```text
1
AAA
8:00 AAA enter
10:00 AAA leave
8:00 BBB enter
9:00 BBB leave
```
**输出：**
```text
2
```
**说明：** 
- AAA 包月，不收费。
- BBB 从 8:00 到 9:00，停了 60 分钟。不在免费时段，计费时长 60 分钟。
- 60 分钟 = 2 个半小时，收费 2 元。

## 六、题目解析

### 1. 题目分析
本题是一道典型的逻辑模拟题，核心难点在于**计费时长的计算**，特别是：
- 扣除 11:30 - 13:30 的重叠时长。
- 处理 8 小时封顶限制。
- 处理“不满半小时不收，超过半小时向上取整”的逻辑。

### 2. 解题思路
1. **数据存储**：
   - 使用 `Set` 存储包月车牌，方便 $O(1)$ 查询。
   - 使用 `Map<String, Integer>` 记录每辆非包月车的进入时间（转换为分钟）。
2. **时间处理**：
   - 将 `HH:mm` 统一转换为从 `00:00` 开始的分钟数。
   - 免费时段范围为：`[11.5 * 60, 13.5 * 60]` 即 `[690, 810]`。
3. **核心计算逻辑**：
   - **计算重叠时长**：计算车辆停车区间 $[Start, End]$ 与免费区间 $[690, 810]$ 的交集长度。
     - $Overlap = \max(0, \min(End, 810) - \max(Start, 690))$。
   - **有效时长**：$TotalMinutes = (End - Start) - Overlap$。
   - **封顶限制**：若 $TotalMinutes > 480$，则取 $480$。
   - **计费**：
     - 若 $TotalMinutes < 30$，费用为 0。
     - 否则，费用为 `ceil(TotalMinutes / 30.0)`。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // 1. 处理包月车
        int n = Integer.parseInt(sc.nextLine());
        Set<String> monthlyCars = new HashSet<>();
        if (n > 0) {
            String[] plates = sc.nextLine().split(" ");
            for (String p : plates) monthlyCars.add(p);
        }

        // 2. 处理进出记录
        Map<String, Integer> enterRecords = new HashMap<>();
        long totalFee = 0;

        while (sc.hasNextLine()) {
            String line = sc.nextLine();
            if (line.isEmpty()) break;
            String[] parts = line.split(" ");
            String timeStr = parts[0];
            String plate = parts[1];
            String action = parts[2];

            int timeInMinutes = timeToMinutes(timeStr);

            if (monthlyCars.contains(plate)) continue;

            if ("enter".equals(action)) {
                enterRecords.put(plate, timeInMinutes);
            } else {
                if (enterRecords.containsKey(plate)) {
                    int start = enterRecords.get(plate);
                    int end = timeInMinutes;
                    totalFee += calculateFee(start, end);
                    enterRecords.remove(plate);
                }
            }
        }
        System.out.println(totalFee);
    }

    // 时间转换：HH:mm -> minutes
    private static int timeToMinutes(String timeStr) {
        String[] parts = timeStr.split(":");
        return Integer.parseInt(parts[0]) * 60 + Integer.parseInt(parts[1]);
    }

    // 核心计费逻辑
    private static int calculateFee(int start, int end) {
        // 免费时段 [11:30, 13:30] -> [690, 810]
        int freeStart = 690;
        int freeEnd = 810;

        // 计算重叠部分
        int overlap = Math.max(0, Math.min(end, freeEnd) - Math.max(start, freeStart));
        
        // 计费时长 = 总时长 - 免费时长
        int duration = (end - start) - overlap;

        // 8小时封顶
        if (duration > 480) {
            duration = 480;
        }

        // 计费规则
        if (duration < 30) {
            return 0;
        } else {
            // 超过30分钟，零头按半小时计 (即向上取整)
            return (int) Math.ceil(duration / 30.0);
        }
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(M)$，其中 $M$ 为记录的行数。每条记录仅处理一次。
- **空间复杂度**：$O(N + K)$，其中 $N$ 是包月车辆数，$K$ 是同时在场内的非包月车数量。
