## 一、 题目描述
给定一个非空字符串 `s` 和一个不包含重复单词的单词列表 `wordDict`，判断 `s` 是否可以被空格拆分成一个或多个在 `wordDict` 中的单词。

**说明：**
- 拆分时可以重复使用字典中的单词。
- 你可以假设字典中没有重复的单词。

## 二、输入描述
- 第一行输入为目标字符串 `s`。
- 第二行输入为单词列表 `wordDict`（通常以逗号或空格分隔，具体面试时视题目给定形式而定）。

## 三、输出描述
- 输出一个布尔值：如果可以拆分则返回 `true`，否则返回 `false`。

## 四、用例
### 用例1
**输入:** 
s = "leetcode", wordDict = ["leet", "code"]
**输出:** 
true
**解释:** 返回 true 因为 "leetcode" 可以被拆分成 "leet code"。

### 用例2
**输入:** 
s = "applepenapple", wordDict = ["apple", "pen"]
**输出:** 
true
**解释:** 返回 true 因为 "applepenapple" 可以被拆分成 "apple pen apple"。注意你可以重复使用字典中的单词。

### 用例3
**输入:** 
s = "catsandog", wordDict = ["cats", "dog", "sand", "and", "cat"]
**输出:** 
false

## 五、备注
1. 字符串 `s` 的长度范围通常在 $[1, 300]$。
2. `wordDict` 的单词总数和单词长度也在合理范围内。
3. 这是一个典型的动态规划问题，考察对子问题状态转移的理解。

## 六、题目解析
### 1. 题目分析
本题要求判断字符串是否能被“切分”成字典中存在的单词。由于切分的方式多种多样，且一个大的切分方案依赖于前面的小切分方案是否成功，这符合**动态规划（Dynamic Programming）**的特征。

### 2. 解题思路
- **定义状态**：设 `dp[i]` 为一个布尔值，表示字符串 `s` 的前 `i` 个字符（下标从 $0$ 到 $i-1$）是否可以被合法拆分。
- **初始状态**：`dp[0] = true`，表示空字符串是可以被拆分的（基础情况）。
- **状态转移方程**：对于 `dp[i]`，我们需要检查在它之前的每一个位置 `j`（$0 \le j < i$）：
  - 如果 `dp[j]` 为 `true`（说明前 `j` 个字符可以被合法拆分）；
  - 并且子串 `s.substring(j, i)` 存在于 `wordDict` 中；
  - 那么 `dp[i]` 就为 `true`，此时可以直接跳出内层循环（只要找到一种切分方式即可）。
- **辅助工具**：为了提高查找效率，将 `wordDict` 存入 `HashSet` 中，使单词查询的时间复杂度降为 $O(1)$。

### 3. 代码实现
#### Java8实现
```java
import java.util.*;

public class Solution {
    public boolean wordBreak(String s, List<String> wordDict) {
        // 使用Set提高查询效率
        Set<String> wordSet = new HashSet<>(wordDict);
        int n = s.length();
        // dp[i] 表示 s 的前 i 个字符是否可以拆分
        boolean[] dp = new boolean[n + 1];
        
        // 初始状态
        dp[0] = true;
        
        // 遍历字符串的每一个截止位置
        for (int i = 1; i <= n; i++) {
            // 尝试在 j 处进行切分
            for (int j = 0; j < i; j++) {
                // 如果前 j 个字符合法，且 [j, i) 组成的子串在字典中
                if (dp[j] && wordSet.contains(s.substring(j, i))) {
                    dp[i] = true;
                    // 只要找到一种合法的切分方式，就可以停止对当前 i 的尝试
                    break;
                }
            }
        }
        
        return dp[n];
    }

    public static void main(String[] args) {
        Solution sol = new Solution();
        // 测试用例
        System.out.println(sol.wordBreak("leetcode", Arrays.asList("leet", "code"))); // true
        System.out.println(sol.wordBreak("applepenapple", Arrays.asList("apple", "pen"))); // true
        System.out.println(sol.wordBreak("catsandog", Arrays.asList("cats", "dog", "sand", "and", "cat"))); // false
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(n^2)$。其中 $n$ 为字符串 `s` 的长度。我们需要两层循环，内层循环中 `substring` 操作的时间复杂度在某些语言里是 $O(k)$，整体接近 $O(n^2)$ 或 $O(n^3)$（取决于具体实现和单词长度上限）。
- **空间复杂度**：$O(n + m)$。其中 $n$ 是字符串长度（DP数组空间），$m$ 是字典中所有单词占用的空间。
