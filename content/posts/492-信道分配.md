## 一、 题目描述

算法工程师小明面对着这样一个问题，需要将通信用的信道分配给尽量多的用户。

信道的条件及分配规则如下：
1. 所有信道都有属性：“阶”。阶为 $r$ 的信道的容量为 $2^r$ 比特；
2. 所有用户需要传输的数据量都一样：$D$ 比特；
3. 一个用户可以分配多个信道，但每个信道只能分配给一个用户；
4. 只有当分配给一个用户的所有信道的容量和 $\\ge D$，用户才能传输数据；

给出一组信道资源，最多可以为多少用户传输数据？

## 二、 输入描述

第一行，一个数字 $R$。$R$ 为最大阶数。$0 \\le R < 20$。

第二行，$R+1$ 个数字，用空格隔开。代表每种信道的数量 $N_i$。按照阶的值从小到大排列。$0 \\le i \\le R, 0 \\le N_i < 1000$。

第三行，一个数字 $D$。$D$ 为单个用户需要传输的数据量。$0 < D < 1,000,000$。

## 三、 输出描述

一个数字（代表最多可以供多少用户传输数据）。

## 四、 用例

**输入**
```
5
10 5 0 1 3 2
30
```

**输出**
```
4
```

**说明**
1. 阶为 5 的信道容量为 $2^5 = 32$，大于 30，每个信道可供一个用户使用，2 个信道可供 2 个用户。
2. 剩余信道：阶 4 (容量 16) 有 3 个，阶 3 (容量 8) 有 1 个，阶 1 (容量 2) 有 5 个，阶 0 (容量 1) 有 10 个。
3. 组合 1：1 个阶 4 (16) + 1 个阶 3 (8) + 3 个阶 1 (6) = 30，可供 1 个用户。
4. 组合 2：2 个阶 4 (32) 虽大于 30，但已用 1 个，余 1 个阶 4 (16) + 2 个阶 1 (4) + 10 个阶 0 (10) = 30，可供 1 个用户。
5. 总计：2 + 1 + 1 = 4 个用户。

## 五、 备注

* $2^R$ 可能很大，但由于 $R < 20$，最大容量为 $2^{19} = 524288$。
* $D$ 最大为 $1,000,000$。
* 需要最大化用户数，应当优先使用大容量信道满足单个用户，再通过贪心策略组合小容量信道。

## 六、 题目解析

这道题的核心在于**贪心算法**。

1. **处理大容量信道**：
 如果某个信道的容量 $2^i \\ge D$，那么这个信道本身就能满足一个用户。由于题目要求的是“最多”用户，且一个信道只能给一个用户，多余的容量不能拆分给别人。因此，这类信道每一个都直接对应一个用户。

2. **处理小容量信道（关键）**：
 对于容量 $2^i < D$ 的信道，我们需要进行组合。贪心策略如下：
 * 我们要凑出 $D$。为了尽量不浪费，我们应该尽量让组合出来的总容量接近 $D$。
 * **步骤 A**：从最高阶（小于 $D$）向最低阶遍历。尽可能多地消耗信道，但保证总和**不超过** $D$。
 * **步骤 B**：如果此时总和还未达到 $D$，说明我们需要从低阶向高阶寻找剩余的信道来“补位”，直到总和刚好 $\\ge D$。
 * 重复上述步骤，直到无法凑出一个 $D$ 为止。

这种“先高位贪心凑接近值，再地位贪心补齐”的策略能有效减少浪费。

## 七、 代码实现

### Java

```java
import java.util.Scanner;

public class Main {
 public static void main(String[] args) {
 Scanner sc = new Scanner(System.in);
 int R = sc.nextInt();
 int[] counts = new int[R + 1];
 long[] capacity = new long[R + 1];
 for (int i = 0; i <= R; i++) {
 counts[i] = sc.nextInt();
 capacity[i] = (long) Math.pow(2, i);
 }
 int D = sc.nextInt();

 int ans = 0;
 // 1. 处理本身就大于等于 D 的信道
 for (int i = 0; i <= R; i++) {
 if (capacity[i] >= D) {
 ans += counts[i];
 counts[i] = 0;
 }
 }

 // 2. 处理需要组合的信道
 while (true) {
 int remainingD = D;
 // 贪心策略：从大到小拿，但不超过 D
 for (int i = R; i >= 0; i--) {
 if (counts[i] > 0 && capacity[i] < remainingD) {
 int needed = Math.min(counts[i], remainingD / (int) capacity[i]);
 remainingD -= needed * capacity[i];
 counts[i] -= needed;
 }
 }

 // 如果还没凑够，从小到大补齐
 if (remainingD > 0) {
 for (int i = 0; i <= R; i++) {
 if (counts[i] > 0) {
 // 只需要拿一个
 remainingD -= capacity[i];
 counts[i]--;
 if (remainingD <= 0) break;
 // 如果拿了一个还没够，可能需要继续拿这个阶层的
 int needed = Math.min(counts[i], (int) Math.ceil((double)remainingD / capacity[i]));
 remainingD -= needed * capacity[i];
 counts[i] -= needed;
 if (remainingD <= 0) break;
 }
 }
 }

 if (remainingD <= 0) {
 ans++;
 } else {
 // 如果所有信道用完都凑不到 D
 break;
 }
 }

 System.out.println(ans);
 }
}
```

```python
import math

def solve():
 try:
 R = int(input())
 counts = list(map(int, input().split()))
 D = int(input())
 except EOFError:
 return

 capacities = [2**i for i in range(R + 1)]
 ans = 0

 # 1. 统计本身就满足 D 的信道
 for i in range(R + 1):
 if capacities[i] >= D:
 ans += counts[i]
 counts[i] = 0

 # 2. 贪心凑 D
 while True:
 rest = D
 # 从大到小尽可能凑，但不超过 D
 for i in range(R, -1, -1):
 if counts[i] > 0 and capacities[i] < rest:
 use = min(counts[i], rest // capacities[i])
 rest -= use * capacities[i]
 counts[i] -= use
 
 # 如果还没凑够，从小到大补一个最小的溢出
 if rest > 0:
 for i in range(R + 1):
 while counts[i] > 0 and rest > 0:
 rest -= capacities[i]
 counts[i] -= 1
 if rest <= 0:
 break
 
 if rest <= 0:
 ans += 1
 else:
 break

 print(ans)

if __name__ == \"__main__\":
 solve()
```

### C++

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <algorithm>

using namespace std;

int main() {
 int R;
 cin >> R;
 vector<int> counts(R + 1);
 vector<long long> capacity(R + 1);
 for (int i = 0; i <= R; i++) {
 cin >> counts[i];
 capacity[i] = 1LL << i;
 }
 int D;
 cin >> D;

 int ans = 0;
 // 1. 处理单体满足的
 for (int i = 0; i <= R; i++) {
 if (capacity[i] >= D) {
 ans += counts[i];
 counts[i] = 0;
 }
 }

 // 2. 组合满足
 while (true) {
 int rest = D;
 // 从大到小凑
 for (int i = R; i >= 0; i--) {
 if (counts[i] > 0 && capacity[i] < rest) {
 int take = min(counts[i], (int)(rest / capacity[i]));
 rest -= take * capacity[i];
 counts[i] -= take;
 }
 }

 // 不够则从小到大补
 if (rest > 0) {
 for (int i = 0; i <= R; i++) {
 while (counts[i] > 0 && rest > 0) {
 rest -= capacity[i];
 counts[i]--;
 }
 if (rest <= 0) break;
 }
 }

 if (rest <= 0) {
 ans++;
 } else {
 break;
 }
 }

 cout << ans << endl;
 return 0;
}
```

```javascript
const readline = require('readline');

const rl = readline.createInterface({

let lines = [];
rl.on('line', (line) => {
 lines.push(line);
}).on('close', () => {

 for (let i = 0; i <= R; i++) {
 capacities.push(Math.pow(2, i));
 }

 let ans = 0;

 // 1. 处理直接满足的
 for (let i = 0; i <= R; i++) {
 if (capacities[i] >= D) {
 ans += counts[i];
 counts[i] = 0;
 }
 }

 // 2. 循环凑数
 while (true) {
 let rest = D;
 // 从大到小填
 for (let i = R; i >= 0; i--) {
 if (counts[i] > 0 && capacities[i] < rest) {
 let take = Math.min(counts[i], Math.floor(rest / capacities[i]));
 rest -= take * capacities[i];
 counts[i] -= take;
 }
 }

 if (rest > 0) {
 for (let i = 0; i <= R; i++) {
 while (counts[i] > 0 && rest > 0) {
 rest -= capacities[i];
 counts[i]--;
 }

 }
 }

 ans++;
 } else {
 break;
 }
 }

 console.log(ans);
});
```
