## 一、 题目描述
给定一个以字符串表示的非负整数 `num`，和一个整数 `k`，请你从 `num` 中移除 `k` 位数字，使得剩下的数字最小。
注意：
1. `num` 的长度小于 10002 且 $\ge k$。
2. `num` 不包含任何前导零。
3. 如果最后剩下的数字为空，则返回 "0"。

## 二、输入描述
* `num`: 一个由数字组成的字符串，例如 "1432219"。
* `k`: 一个整数，表示需要移除的数字位数。

## 三、输出描述
* 移除 `k` 位数字后能得到的最小数字字符串。

## 四、用例
### 用例1
**输入：**
`num = "1432219", k = 3`
**输出：**
`"1219"`
**解释：** 移除 4, 3, 2 后得到 1219，这是最小的结果。

### 用例2
**输入：**
`num = "10200", k = 1`
**输出：**
`"200"`
**解释：** 移除首位的 1，剩余 0200，去掉前导零后为 200。

### 用例3
**输入：**
`num = "10", k = 2`
**输出：**
`"0"`
**解释：** 移除所有数字，剩余为空，按规则返回 "0"。

## 五、备注
* 这是一个典型的贪心算法问题，关键在于“高位数字越小，整体数值越小”。
* 需要特别处理前导零的情况。
* 结果集如果为空字符串，必须转化为 "0"。

## 六、题目解析

### 1. 题目分析
要使剩下的数字最小，我们应该尽可能让**高位**的数字变小。
例如：对于 "1432219"，我们要决定移除哪三个。
从左往右看，1 比 4 小（保持），4 比 3 大（移除 4），此时剩下 "132219"。
接下来的逻辑以此类推：如果当前数字比右边相邻的数字大，移除当前数字就能让这一位被更小的数字占据。

### 2. 解题思路
我们可以利用 **单调栈（Monotonic Stack）** 的思想：
1. **贪心策略**：遍历字符串，如果当前字符比栈顶字符小，且我们还有移除名额（$k > 0$），则弹出栈顶元素。
2. **入栈**：将当前字符压入栈中。
3. **剩余处理**：如果遍历完字符串后，$k$ 仍然大于 0（说明原字符串是升序的），则从栈顶（末尾）继续移除多余的 $k$ 位。
4. **前导零处理**：将栈中元素转化为字符串，并移除开头的 '0'。
5. **边界检查**：若结果字符串为空，返回 "0"。

### 3. 代码实现

#### Java8实现
```java
import java.util.Deque;
import java.util.LinkedList;

public class Solution {
    public String removeKdigits(String num, int k) {
        // 如果移除位数等于长度，直接返回"0"
        if (num.length() == k) {
            return "0";
        }

        Deque<Character> stack = new LinkedList<>();
        int len = num.length();

        for (int i = 0; i < len; i++) {
            char digit = num.charAt(i);
            // 贪心：如果当前数字比栈顶小，弹出栈顶，减少k
            while (!stack.isEmpty() && k > 0 && stack.peekLast() > digit) {
                stack.removeLast();
                k--;
            }
            stack.addLast(digit);
        }

        // 如果k还没用完（例如"1234"，k=1），从末尾截断
        for (int i = 0; i < k; i++) {
            stack.removeLast();
        }

        // 构建结果并处理前导零
        StringBuilder sb = new StringBuilder();
        boolean leadingZero = true;
        while (!stack.isEmpty()) {
            char digit = stack.removeFirst();
            if (leadingZero && digit == '0') {
                continue;
            }
            leadingZero = false;
            sb.append(digit);
        }

        // 如果结果为空返回"0"
        return sb.length() == 0 ? "0" : sb.toString();
    }

    public static void main(String[] args) {
        Solution sol = new Solution();
        System.out.println(sol.removeKdigits("1432219", 3)); // 输出: 1219
        System.out.println(sol.removeKdigits("10200", 1));   // 输出: 200
        System.out.println(sol.removeKdigits("10", 2));      // 输出: 0
    }
}
```

### 4. 复杂度分析
* **时间复杂度**：$O(N)$，其中 $N$ 是字符串的长度。虽然有嵌套循环，但每个字符最多入栈一次、出栈一次。
* **空间复杂度**：$O(N)$，用于存储栈中的元素。
