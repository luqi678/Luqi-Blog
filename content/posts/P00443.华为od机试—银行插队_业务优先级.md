## 二、输入描述
输入第一行是一个正整数 $n$，表示输入的序列中的事件数量。($1 \le n \le 500$)。

接下来有 $n$ 行，每行第一个字符为 `a` 或 `p`：
- 当字符为 `a` 时，后面会有两个正整数 `num` 和 `x`，表示到来的客户编号为 `num`，优先级为 `x`；
- 当字符为 `p` 时，表示当前优先级最高的客户去办理业务。

## 三、输出描述
输出包含若干行，对于每个 `p`，输出一行，仅包含一个正整数 `num`，表示办理业务的客户编号。

## 四、用例
### 用例1
**输入**
```text
4
a 1 3
a 2 2
a 3 2
p
```
**输出**
```text
2
```

### 用例2
**输入**
```text
7
a 1 5
a 2 1
a 3 5
p
a 4 1
p
p
```
**输出**
```text
2
4
1
```

## 五、备注
- 优先级数值越小，等级越高（1级最高，5级最低）。
- 如果优先级相同，则按照先来后到的顺序（FIFO）办理。
- 题目保证在执行 `p` 操作时，队列中一定有客户。

## 六、题目解析
### 1. 题目分析
本题模拟了一个典型的优先级队列场景。核心点在于“插队”逻辑：
1. **第一关键字（优先级）**：优先级高的（数值小的）先办理。
2. **第二关键字（先后顺序）**：优先级相同时，早到的先办理。

### 2. 解题思路
- **数据结构选择**：可以使用 Java 的 `PriorityQueue`（优先级队列）。该数据结构可以在 $O(\log n)$ 的时间内插入元素，并以 $O(\log n)$ 的时间取出优先级最高（或最小）的元素。
- **自定义对象**：创建一个 `Customer` 类，包含三个属性：
    - `id`: 客户编号。
    - `priority`: 客户优先级。
    - `arrivalTime`: 客户到达的时间顺序（用一个全局自增变量 `time` 表示）。
- **比较器（Comparator）逻辑**：
    - 如果 `c1.priority != c2.priority`，则 `priority` 小的排在前面。
    - 如果 `c1.priority == c2.priority`，则 `arrivalTime` 小的排在前面。

### 3. 代码实现
#### Java8实现

```java
import java.util.*;

public class Main {
    // 定义客户类
    static class Customer {
        int id;
        int priority;
        int arrivalTime;

        public Customer(int id, int priority, int arrivalTime) {
            this.id = id;
            this.priority = priority;
            this.arrivalTime = arrivalTime;
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) return;
        int n = sc.nextInt();

        // 优先级队列排序规则：优先级数值小的优先；优先级相同时，到达时间小的优先
        PriorityQueue<Customer> queue = new PriorityQueue<>((c1, c2) -> {
            if (c1.priority != c2.priority) {
                return c1.priority - c2.priority;
            }
            return c1.arrivalTime - c2.arrivalTime;
        });

        int timeCounter = 0; // 用于记录到达的先后顺序

        for (int i = 0; i < n; i++) {
            String op = sc.next();
            if ("a".equals(op)) {
                int num = sc.nextInt();
                int x = sc.nextInt();
                // 客户到来，加入队列，并记录当前的时间戳
                queue.offer(new Customer(num, x, timeCounter++));
            } else if ("p".equals(op)) {
                // 办理业务，弹出队列头部的客户
                if (!queue.isEmpty()) {
                    Customer served = queue.poll();
                    System.out.println(served.id);
                }
            }
        }
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：
    - 插入操作（`a`）：`PriorityQueue.offer()` 为 $O(\log n)$。
    - 弹出操作（`p`）：`PriorityQueue.poll()` 为 $O(\log n)$。
    - 总共有 $n$ 次操作，因此总时间复杂度为 $O(n \log n)$。在本题 $n \le 500$ 的规模下，效率极高。
- **空间复杂度**：
    - 需要存储最多 $n$ 个客户的信息，空间复杂度为 $O(n)$。
