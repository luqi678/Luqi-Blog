## 二、题目描述

工位由序列 $F_1, F_2, \dots, F_n$ 组成，$F_i$ 值为 0、1 或 2。其中：
- **0** 代表空置
- **1** 代表有人（老员工）
- **2** 代表障碍物

**友好度计算规则：**
1. 某一空位的友好度为**左右连续老员工数之和**。
2. 所谓“连续”，是指在遇到障碍物（2）或其他空位（0）时会中断。
3. 为方便新员工学习求助，优先安排友好度最高的空位。

给出工位序列，求所有空位中友好度的最大值。如果没有空位，返回 0。

## 二、输入描述

第一行为工位序列，由 $F_1, F_2, \dots, F_n$ 组成，以空格分隔。
- $1 \le n \le 10000$
- $F_i$ 值为 0、1 或 2。

## 三、输出描述

输出所有空位中友好度的最大值。

## 四、用例

### 用例1
**输入**
```text
0 1 0
```
**输出**
```text
1
```
**说明**
第1个位置（索引0）左边无员工，右边连续1个员工，友好度 0+1=1。
第3个位置（索引2）左边连续1个员工，右边无员工，友好度 1+0=1。
最大值为1。

### 用例2
**输入**
```text
1 1 0 1 2 1 0
```
**输出**
```text
3
```
**说明**
- 索引2的空位：左边连续 2 个员工（1, 1），右边连续 1 个员工（1），被障碍物 2 隔断。友好度 = 2 + 1 = 3。
- 索引6的空位：左边连续 1 个员工（1），右边无员工。友好度 = 1 + 0 = 1。
最大值为 3。

## 五、备注
- 如果没有空位（即序列中没有 0），则输出 0。
- 连续老员工的统计必须是紧邻空位的。

## 六、题目解析

### 1. 题目分析
本题要求在给定的数组中寻找所有的 `0`（空位），并计算每个 `0` 左右两边**连续**的 `1`（员工）的数量。这里的“连续”意味着一旦遇到 `0` 或 `2`，计数就停止。

### 2. 解题思路
为了高效解决该问题，我们可以使用**线性扫描**的方法：
1. **预处理左侧连续员工数**：创建一个数组 `leftCount`，其中 `leftCount[i]` 表示位置 `i` 左侧紧邻的连续员工数量。
   - 如果 `F[i-1] == 1`，则 `leftCount[i] = leftCount[i-1] + 1`。
   - 如果 `F[i-1] == 0` 或 `2`，则 `leftCount[i] = 0`。
2. **预处理右侧连续员工数**：同理，创建一个数组 `rightCount`，从右往左遍历。
   - 如果 `F[i+1] == 1`，则 `rightCount[i] = rightCount[i+1] + 1`。
   - 如果 `F[i+1] == 0` 或 `2`，则 `rightCount[i] = 0`。
3. **计算结果**：遍历原数组，当遇到 `0` 时，计算 `leftCount[i] + rightCount[i]`，记录并更新最大值。

### 3. 代码实现

#### Java8实现

```java
import java.util.Scanner;
import java.util.Arrays;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextLine()) return;
        
        String line = sc.nextLine();
        String[] parts = line.split("\\s+");
        int n = parts.length;
        int[] f = new int[n];
        for (int i = 0; i < n; i++) {
            f[i] = Integer.parseInt(parts[i]);
        }

        int[] leftCount = new int[n];
        int[] rightCount = new int[n];

        // 计算每个位置左侧连续的1的个数
        for (int i = 1; i < n; i++) {
            if (f[i - 1] == 1) {
                leftCount[i] = leftCount[i - 1] + 1;
            } else {
                leftCount[i] = 0;
            }
        }

        // 计算每个位置右侧连续的1的个数
        for (int i = n - 2; i >= 0; i--) {
            if (f[i + 1] == 1) {
                rightCount[i] = rightCount[i + 1] + 1;
            } else {
                rightCount[i] = 0;
            }
        }

        int maxFriendship = 0;
        boolean hasZero = false;
        for (int i = 0; i < n; i++) {
            if (f[i] == 0) {
                hasZero = true;
                maxFriendship = Math.max(maxFriendship, leftCount[i] + rightCount[i]);
            }
        }

        System.out.println(hasZero ? maxFriendship : 0);
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(N)$。我们需要遍历数组三次（计算左侧、计算右侧、寻找最大值），其中 $N$ 是数组长度。
- **空间复杂度**：$O(N)$。需要两个额外的辅助数组存储左右连续员工的数量。
