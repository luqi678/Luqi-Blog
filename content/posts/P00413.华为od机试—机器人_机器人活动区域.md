## 二、 题目描述

现有一个机器人，可放置于 $M \times N$ 的网格中任意位置。每个网格包含一个非负整数编号。当相邻网格的数字编号差值的绝对值小于等于 1 时，机器人可以在网格间移动。

**问题：** 求机器人可活动的最大范围对应的网格点数目。

**说明：**
1. 网格左上角坐标为 $(0,0)$，右下角坐标为 $(M-1, N-1)$。
2. 机器人只能在相邻网格间（上下左右）移动。

## 二、输入描述

第 1 行输入为 $M$ 和 $N$，$M$ 表示网格的行数，$N$ 表示网格的列数。
之后 $M$ 行表示网格数值，每行 $N$ 个数值（数值大小用 $k$ 表示），数值间用单个空格分隔。
$M, N, k$ 均为整数，且 $1 \le M, N \le 150, 0 \le k \le 50$。

## 三、输出描述

输出 1 行，包含 1 个数字，表示最大活动区域的网格点数目。

## 四、用例

### 用例1
**输入**
```
4 4
1 2 5 2
2 4 4 5
3 5 7 1
4 6 2 4
```
**输出**
```
6
```

### 用例2
**输入**
```
2 3
1 3 5
4 1 3
```
**输出**
```
1
```

## 五、备注
网格的大小最大为 $150 \times 150$，数值范围较小，适合使用搜索算法。

## 六、题目解析

### 1. 题目分析
本题要求在给定的二维矩阵中，寻找一个“最大连通区域”。连通的定义是：
1. 两个网格必须是相邻的（上下左右）。
2. 两个网格内的数值之差的绝对值 $\le 1$。

这实际上是一个典型的**连通分量**问题。我们需要遍历矩阵中的每一个点，如果该点未被访问，则从该点开始进行搜索（BFS 或 DFS），统计与其连通的所有点的数量，并不断更新最大值。

### 2. 解题思路
1. **输入处理**：读取 $M$ 和 $N$，并读取整个 $M \times N$ 的矩阵存储到二维数组 `grid` 中。
2. **初始化**：创建一个同等大小的布尔矩阵 `visited` 用于标记该位置是否已经被处理。
3. **遍历与搜索**：
   - 遍历 `grid` 的每一个单元格 $(i, j)$。
   - 如果 `visited[i][j]` 为 `false`，则以该点为起点启动 **广度优先搜索 (BFS)** 或 **深度优先搜索 (DFS)**。
   - 在搜索过程中，检查当前点相邻的四个方向 $(nr, nc)$。
   - **移动条件**：满足 `0 <= nr < M` 且 `0 <= nc < N`，且 `!visited[nr][nc]`，且 `abs(grid[nr][nc] - grid[r][c]) <= 1`。
   - 记录该次搜索访问到的节点总数。
4. **更新最大值**：在每次搜索结束后，更新 `max_area = max(max_area, current_count)`。
5. **输出结果**：遍历结束后输出 `max_area`。

### 3. 代码实现

#### Java8实现

```java
import java.util.*;

public class Main {
    private static int M, N;
    private static int[][] grid;
    private static boolean[][] visited;
    private static int[][] directions = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // 处理输入
        if (!sc.hasNextInt()) return;
        M = sc.nextInt();
        N = sc.nextInt();
        
        grid = new int[M][N];
        for (int i = 0; i < M; i++) {
            for (int j = 0; j < N; j++) {
                grid[i][j] = sc.nextInt();
            }
        }

        int maxArea = 0;
        visited = new boolean[M][N];

        // 遍历每个点
        for (int i = 0; i < M; i++) {
            for (int j = 0; j < N; j++) {
                // 如果该点没有被访问过，则开始搜索一个新的连通分量
                if (!visited[i][j]) {
                    maxArea = Math.max(maxArea, bfs(i, j));
                }
            }
        }

        System.out.println(maxArea);
    }

    /**
     * 使用BFS计算连通区域的大小
     */
    private static int bfs(int r, int c) {
        Queue<int[]> queue = new LinkedList<>();
        queue.offer(new int[]{r, c});
        visited[r][c] = true;
        int count = 0;

        while (!queue.isEmpty()) {
            int[] curr = queue.poll();
            count++;

            for (int[] d : directions) {
                int nr = curr[0] + d[0];
                int nc = curr[1] + d[1];

                // 边界检查 & 访问检查
                if (nr >= 0 && nr < M && nc >= 0 && nc < N && !visited[nr][nc]) {
                    // 移动条件：差值绝对值 <= 1
                    if (Math.abs(grid[nr][nc] - grid[curr[0]][curr[1]]) <= 1) {
                        visited[nr][nc] = true;
                        queue.offer(new int[]{nr, nc});
                    }
                }
            }
        }
        return count;
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(M \times N)$。矩阵中的每个单元格最多被访问一次，每个单元格检查四个方向，因此总复杂度与网格大小成正比。
- **空间复杂度**：$O(M \times N)$。需要一个额外的 `visited` 数组来存储访问状态，以及一个队列用于 BFS 搜索，最坏情况下队列的大小也会达到 $O(M \times N)$。
