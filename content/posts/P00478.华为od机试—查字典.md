## 二、 题目描述

输入一个单词前缀和一个有序的单词字典。请从字典中找出所有包含该前缀（即以该前缀开头）的单词，并按字典顺序输出。

如果字典中没有任何单词包含该前缀，则输出 -1。

## 二、输入描述

输入在一行中给出：
1. 单词前缀（字符串）
2. 字典长度 $N$（整数）
3. 字典中的 $N$ 个单词（字符串数组，已按升序排列）

输入的所有字符均为小写英文字母。

## 三、输出描述

输出所有包含该前缀的单词，每个单词占据一行。
如果没有匹配的单词，则输出 -1。

## 四、用例

### 用例1
**输入：**
```text
b 3 a b c
```
**输出：**
```text
b
```

### 用例2
**输入：**
```text
abc 4 a ab abc abcd
```
**输出：**
```text
abc
abcd
```

### 用例3
**输入：**
```text
a 3 b c d
```
**输出：**
```text
-1
```

## 五、备注
1. 字典中的单词是有序的。
2. 输入输出均为小写。
3. 如果匹配到多个单词，需要按序（即输入顺序）输出。

## 六、题目解析

### 1. 题目分析
本题属于基础的字符串匹配问题。给定一个目标前缀，要求在给定的有序集合中筛选出所有满足条件的字符串。

### 2. 解题思路
1.  **读取输入**：首先读取前缀，然后读取字典的长度 $N$，接着读取 $N$ 个单词存入数组或列表。
2.  **前缀匹配**：遍历单词列表，利用 Java 中字符串的 `startsWith()` 方法检查每个单词是否以给定前缀开头。
3.  **结果存储**：将所有匹配成功的单词存入一个结果集合（如 `ArrayList`）。
4.  **结果输出**：
    *   如果结果集合为空，说明没有单词匹配，输出 -1。
    *   如果结果集合不为空，依次输出集合中的每一个单词，每个单词换行。
5.  **优化建议**（选读）：由于字典是有序的，理论上可以使用二分查找找到第一个匹配的单词位置，然后向后遍历直到不再匹配。但在机考常规数据规模下，线性遍历 $O(N \times L)$（$L$ 为前缀长度）已足够。

### 3. 代码实现

#### Java8实现

```java
import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 判断是否有输入
        if (!sc.hasNext()) {
            return;
        }

        // 1. 读取单词前缀
        String prefix = sc.next();
        
        // 2. 读取字典长度
        if (!sc.hasNextInt()) {
            return;
        }
        int n = sc.nextInt();

        // 3. 读取字典单词并进行匹配
        List<String> result = new ArrayList<>();
        for (int i = 0; i < n; i++) {
            if (sc.hasNext()) {
                String word = sc.next();
                // 检查是否以前缀开头
                if (word.startsWith(prefix)) {
                    result.add(word);
                }
            }
        }

        // 4. 输出结果
        if (result.isEmpty()) {
            System.out.println("-1");
        } else {
            for (String res : result) {
                System.out.println(res);
            }
        }

        sc.close();
    }
}
```

### 4. 复杂度分析
-   **时间复杂度**：$O(N \times L)$，其中 $N$ 是单词个数，$L$ 是前缀的长度。我们需要遍历每个单词并进行前缀比较。
-   **空间复杂度**：$O(N \times M)$，其中 $M$ 是单词的平均长度，用于存储输入的字典单词。
