## 一、 题目描述

在一个网络文件存储系统中，有众多的文件按照不同的文件夹进行分类存储。现在要对这些文件进行整理排序，以便便于快速地查找特定文件。

每个文件都有一个创建的时间戳（用整数标识，代表从特定时刻到文件创建时所经历的秒数），并且文件夹之间存在层级关系。

现要求实现一个函数，对给定的某个文件夹下的所有文件（包含子文件夹中的文件）按照创建时间戳进行排序。

## 二、输入描述

1.  首先输入一个整数 `n`，表示接下来要输入的文件及文件夹信息的行数。
2.  然后输入一指定文件夹的名称，用于说明获取哪个文件夹下的全部文件，并进行排序。
3.  最后逐行输入文件或文件夹信息，行数为第一句输入的整数 `n`。
    * 对于**文件**，每行格式为：`文件名 创建时间戳`
    * 对于**文件夹**，每行格式为：`文件夹名 -1`（-1表示这是一个文件夹）
    * **层级关系**：如果文件夹包含子文件或子文件夹，则在后序行中继续输入其内。通过缩进（四个短横线 `----`）表示层级关系（例如，子文件或子文件夹的信息相比其父文件夹缩进4个短线）。

## 三、输出描述

输出排序后的文件列表，每行格式为：`文件名 创建时间戳`，按照时间戳**从小到大**进行排序。
如果文件系统中不存在该文件夹，或者该文件夹下没有文件，则返回字符串 `"No file"`。

## 四、用例

| 输入 | 输出 | 说明 |
| :--- | :--- | :--- |
| `5`<br>`Documents`<br>`Documents -1`<br>`----file1.txt 1600000000`<br>`----file2.txt 1600000100`<br>`----SubFolder1 -1`<br>`--------file3.txt 1600002000` | `file1.txt 1600000000`<br>`file2.txt 1600000100`<br>`file3.txt 1600002000` | 找到Documents文件夹下的所有文件，按时间戳升序排列。（注：题目原示例输出有拼写错误fiel，此处修正为逻辑正确输出） |
| `5`<br>`Video`<br>`Video -1`<br>`----SubFolder0 -1`<br>`----SubFolder1 -1`<br>`----SubFolder3 -1`<br>`--------SubFolder4 -1` | `No file` | 虽然找到了Video文件夹，但其内部全是子文件夹，没有具体文件，故输出 No file。 |

## 五、备注

1.  文件和文件夹名称只会出现 `A-Z`、`a-z`、`0-9` 和字符 `.`。
2.  不会出现特殊字符。
3.  不会出现相同时间戳的文件。
4.  不会出现重名的文件或文件夹。
5.  缩进严格为4个短横线 `----` 代表一级。

## 六、题目解析

### 核心思路

这是一个典型的**树形结构解析与遍历**问题。由于输入数据是按照 DFS（深度优先）序给出的，且通过缩进表示层级，我们可以利用这一特性构建树，或者在解析过程中直接定位目标。

### 步骤详解

1.  **构建树结构**：
    * 我们需要一个 `Node` 类来存储名称、时间戳（-1为文件夹）、子节点列表。
    * **层级判断**：读取每一行时，通过计算开头的 `----` 数量来确定当前节点的深度（Level）。每4个横线代表一级。
    * **父子关系**：由于是顺序输入，当前深度为 `L` 的节点，其父节点一定是**最近一次出现的**深度为 `L-1` 的节点。我们可以使用一个 `Map<Integer, Node>` 来记录每一层最近访问的节点（或者使用栈）。
    * **根节点处理**：深度为0的节点直接作为根节点（或森林中的树根）。

2.  **查找目标文件夹**：
    * 构建完树后，遍历整棵树找到名称与输入的目标文件夹名称相同的节点 `targetNode`。
    * 如果找不到，直接输出 "No file"。

3.  **收集文件**：
    * 从 `targetNode` 开始进行 DFS 或 BFS 遍历。
    * 收集所有 `time != -1` 的节点（即文件）。

4.  **排序与输出**：
    * 如果收集到的文件列表为空，输出 "No file"。
    * 否则，按照时间戳进行升序排序。
    * 按格式输出。

### 算法实现 (Java)

```java
import java.util.*;

public class Main {
    // 定义节点类
    static class Node {
        String name;
        long time; // -1 表示文件夹
        List<Node> children = new ArrayList<>();

        public Node(String name, long time) {
            this.name = name;
            this.time = time;
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNext()) return;

        int n = Integer.parseInt(sc.nextLine());
        String targetFolder = sc.nextLine();

        // 记录每一层最近的一个节点，用于构建父子关系
        // key: level (0, 1, 2...), value: Node
        Map<Integer, Node> levelMap = new HashMap<>();
        List<Node> roots = new ArrayList<>(); // 存储所有根节点

        // 1. 解析输入构建树
        for (int i = 0; i < n; i++) {
            String line = sc.nextLine();
            
            // 计算层级：统计前缀 '-' 的个数
            int dashCount = 0;
            while (dashCount < line.length() && line.charAt(dashCount) == '-') {
                dashCount++;
            }
            int level = dashCount / 4;

            // 解析名称和时间戳
            // 去掉前缀的横线
            String content = line.substring(dashCount);
            // 按照最后一个空格分割名称和时间戳
            int lastSpaceIndex = content.lastIndexOf(' ');
            String name = content.substring(0, lastSpaceIndex);
            String timeStr = content.substring(lastSpaceIndex + 1);
            long time = Long.parseLong(timeStr);

            Node node = new Node(name, time);

            // 建立树关系
            if (level == 0) {
                roots.add(node);
            } else {
                // 父节点是上一层最近的一个节点
                Node parent = levelMap.get(level - 1);
                if (parent != null) {
                    parent.children.add(node);
                }
            }
            // 更新当前层的最新节点
            levelMap.put(level, node);
        }

        // 2. 查找目标文件夹节点
        Node targetNode = null;
        // 因为题目说不重名，可以用BFS或DFS在所有roots中找
        Queue<Node> queue = new LinkedList<>(roots);
        while (!queue.isEmpty()) {
            Node curr = queue.poll();
            if (curr.name.equals(targetFolder) && curr.time == -1) {
                targetNode = curr;
                break;
            }
            // 即使不是目标，也要继续找子节点
            queue.addAll(curr.children);
        }

        if (targetNode == null) {
            System.out.println("No file");
            return;
        }

        // 3. 收集目标文件夹下的所有文件
        List<Node> resultFiles = new ArrayList<>();
        collectFiles(targetNode, resultFiles);

        // 4. 排序并输出
        if (resultFiles.isEmpty()) {
            System.out.println("No file");
        } else {
            // 按时间戳升序排序
            resultFiles.sort(Comparator.comparingLong(o -> o.time));
            
            for (Node file : resultFiles) {
                System.out.println(file.name + " " + file.time);
            }
        }
    }

    // 递归收集文件
    private static void collectFiles(Node root, List<Node> result) {
        // 如果是文件（时间戳不是-1），加入结果集
        if (root.time != -1) {
            result.add(root);
        }
        // 递归处理子节点（文件也可能被误挂在下面？题目逻辑文件夹才有子节点，但遍历一下无妨）
        for (Node child : root.children) {
            collectFiles(child, result);
        }
    }
}
```
