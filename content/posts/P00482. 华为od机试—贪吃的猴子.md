## 一、 题目描述

一只贪吃的猴子，来到一个果园，发现许多串香蕉排成一行，每串香蕉上有若干根香蕉。每串香蕉的根数由数组 `numbers` 给出。

猴子获取香蕉，**每次都只能从行的开头或者末尾获取**，并且只能获取 $N$ 次，求猴子最多能获取多少根香蕉。

## 二、输入描述

1.  第一行为数组 `numbers` 的长度。
2.  第二行为数组 `numbers` 的值，每个数字通过空格分开。
3.  第三行输入为 $N$，表示获取的次数。

## 三、输出描述

按照题目要求能获取的最大数值。
 
## 四、用例

**示例 1**

```text
输入：
7
1 2 2 7 3 6 1
3

输出：
10
```

**示例 2**

```text
输入：
3
1 2 3
3

输出：
6

说明：
全部获取所有的香蕉，因此最终根数为 1+2+3 = 6
```

**示例 3**

```text
输入：
4
4 2 2 3
2

输出：
7

说明：
第一次获取香蕉为行的开头（4），第二次获取为行的末尾（3），因此最终根数为 4+3 = 7
```

## 五、备注

* 如果 $N$ 大于数组长度，则相当于获取所有香蕉。
* 题目核心在于如何在有限次操作中，组合左边和右边的元素使其和最大。

## 六、题目解析

### 1. 解题思路

这就使得题目变成了一个经典的**滑动窗口**或者**逆向思维**的问题。

猴子只能拿 $N$ 次，且只能从头或尾拿。这意味着猴子拿走的元素虽然在原数组中可能是不连续的（一部分在头，一部分在尾），但在逻辑上，我们可以看作是一个滑动的过程。

假设猴子拿了 $x$ 个左边的香蕉，那么它必然拿了 $N-x$ 个右边的香蕉。

我们可以使用滑动窗口的思想来解决：
1.  **初始状态**：假设猴子把 $N$ 次机会全部用在左边（即取数组前 $N$ 个元素），计算此时的总和，作为初始的 `maxSum`。
2.  **滑动调整**：
    * 我们从左边的选择中“退回”一个（即左边少拿一个），然后从右边“拿”一个（即右边多拿一个）。
    * 更新当前的总和。
    * 比较并更新 `maxSum`。
3.  **循环执行**：直到左边一个都不拿，全部 $N$ 次机会都用在右边为止。

### 2. 算法流程

设数组为 `nums`，长度为 `len`，我们要取 `n` 个。

1.  计算 `nums[0] + ... + nums[n-1]` 的和，记为 `currentSum`。
2.  设置 `maxSum = currentSum`。
3.  遍历 `i` 从 0 到 `n-1`：
    * 左边我们要放弃的元素下标是 `n - 1 - i`。
    * 右边我们要新加入的元素下标是 `len - 1 - i`。
    * `currentSum = currentSum - nums[n - 1 - i] + nums[len - 1 - i]`。
    * `maxSum = max(maxSum, currentSum)`。
4.  输出 `maxSum`。

### 3. 参考代码

#### Java 实现

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        // 1. 输入数组长度
        if (!sc.hasNextInt()) return;
        int len = sc.nextInt();
        
        // 2. 输入数组内容
        int[] nums = new int[len];
        for (int i = 0; i < len; i++) {
            nums[i] = sc.nextInt();
        }
        
        // 3. 输入获取次数 N
        int n = sc.nextInt();
        
        // 边界处理：如果 n 大于等于长度，直接返回所有元素之和
        if (n >= len) {
            int sum = 0;
            for (int num : nums) sum += num;
            System.out.println(sum);
            return;
        }
        
        // 核心逻辑：滑动窗口
        // 步骤 1: 先假设全部取左边的 n 个
        int currentSum = 0;
        for (int i = 0; i < n; i++) {
            currentSum += nums[i];
        }
        
        int maxSum = currentSum;
        
        // 步骤 2: 逐步将左边的元素替换为右边的元素
        // i 代表从右边取的个数，从 1 到 n
        for (int i = 1; i <= n; i++) {
            // 左边少拿一个：下标为 n - i
            int leftVal = nums[n - i];
            // 右边多拿一个：下标为 len - i
            int rightVal = nums[len - i];
            
            currentSum = currentSum - leftVal + rightVal;
            maxSum = Math.max(maxSum, currentSum);
        }
        
        System.out.println(maxSum);
    }
}
```

#### Python 实现

```python
import sys

def solve():
    try:
        # 读取所有输入
        input_data = sys.stdin.read().split()
        if not input_data:
            return
            
        iterator = iter(input_data)
        length = int(next(iterator))
        
        nums = []
        for _ in range(length):
            nums.append(int(next(iterator)))
            
        n = int(next(iterator))
        
        # 边界情况：拿取的数量大于等于数组长度
        if n >= length:
            print(sum(nums))
            return

        # 初始窗口：全拿左边的 n 个
        current_sum = sum(nums[:n])
        max_sum = current_sum
        
        # 滑动窗口：左减右加
        # 此时 nums[n-1] 是左边最后一个被选的，nums[-1] 是右边第一个待选的
        for i in range(n):
            # 减去左边第 n-1-i 个
            # 加上右边倒数第 1+i 个 (即 len-1-i)
            current_sum = current_sum - nums[n - 1 - i] + nums[length - 1 - i]
            max_sum = max(max_sum, current_sum)
            
        print(max_sum)
        
    except StopIteration:
        pass

if __name__ == "__main__":
    solve()
```
