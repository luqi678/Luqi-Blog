## 二、 题目描述

疫情期间需要大家保证一定的社交距离，公司组织开交流会议。座位一排共 $N$ 个座位，编号分别为 $[0, N-1]$。

要求员工一个接着一个进入会议室，并且可以在任何时候离开会议室。

满足：
1. 每当一个员工进入时，需要坐到最大社交距离（最大化自己和其他人的距离的座位）；
2. 如果有多个这样的座位，则坐到索引最小的那个座位。

## 三、 输入描述

1. 会议室座位总数 `seatNum`。($1 \le seatNum \le 500$)
2. 员工的进出顺序 `seatOrLeave` 数组，元素值为 1，表示进场；元素值为负数，表示出场。
   - 例如 `-4` 表示坐在位置 4 的员工离开（保证有员工坐在该座位上）。
   - 特殊：位置 0 的员工不会离开。

## 四、 输出描述

最后进来的员工坐在第几个位置。如果位置已满，则输出 -1。

## 五、 用例

### 用例1

**输入**
```
10
[1,1,1,1,-4,1]
```
**输出**
```
5
```
**说明**
1. `seat -> 1`: 坐 0。空在任何位置都行，但要索引最小，即位置 0。
2. `seat -> 1`: 坐 9。要和旁边（位置0）距离最远，即位置 9。
3. `seat -> 1`: 坐 4。0 和 9 中间，距离最大，即位置 4。
4. `seat -> 1`: 坐 2。0 到 4 距离为 2，4 到 9 距离为 2，选索引最小的位置 2。
5. `leave -> -4`: 4 号座位员工离开。
6. `seat -> 1`: 此时有座位 {0, 2, 9}。0 到 2 距离 1，2 到 9 距离 3（索引 5），最大社交距离位置为 5。输出 5。

### 用例2

**输入**
```
5
[1,1,1]
```
**输出**
```
2
```

## 六、 题目解析

### 1. 题目分析
本题属于典型的**线性空间模拟题**。我们需要维护一个座位的占用状态，并在每次有人进入时，计算所有可能的空位中，哪一个位置距离最近的邻居最远（即社交距离最大）。

**关键定义：**
- **社交距离**：当前选定的位置到其左右最近邻居的距离的最小值。
- **边界情况**：
    - 如果会议室全空，第一个人坐索引 0。
    - 如果会议室只坐了人，入座时需考虑两端（索引 0 和 $N-1$）以及人与人之间的空隙。
    - 两人座位 $A$ 和 $B$ 中间的最佳位置是 $P = A + (B - A) / 2$，社交距离为 $(B - A) / 2$。
    - 索引 0 如果没坐人，社交距离是到第一个坐着的人的距离。
    - 索引 $N-1$ 如果没坐人，社交距离是到最后一个坐着的人的距离。

### 2. 解题思路
1.  **数据结构选择**：由于需要频繁查找空隙和插入/删除座位，使用 `TreeSet<Integer>` 来存储已占用的座位索引，这样集合内的元素天然有序，方便计算相邻间距。
2.  **处理逻辑**：
    -   **进场 (1)**：
        -   若 `seated` 为空，选 0。
        -   若不为空且不满：
            1.  **检查索引 0**：如果 0 没被占，候选距离为 `seated.first()`，候选位置 0。
            2.  **检查中间空隙**：遍历有序座位，对于相邻的两个座位 $left$ 和 $right$，中间位置 $pos = left + (right - left) / 2$。如果 $(right - left) / 2 > maxDist$，更新位置和距离。
            3.  **检查末尾 $N-1$**：如果 $N-1$ 没被占，候选距离为 $(N - 1) - seated.last()`。如果大于最大距离，更新。
        -   若全满，记为 -1。
    -   **离场 (<0)**：
        -   将 `-op` 从 `seated` 中移除。
3.  **注意**：输出最后一次入场成功的座位。如果最后的操作是离场，输出应为上一次入场的位置。但根据题意和用例，通常指处理完所有指令后，最后一个“1”对应的结果。

### 3. 代码实现

#### Java 实现

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int seatNum = Integer.parseInt(sc.nextLine().trim());
        String line = sc.nextLine().trim();
        // 解析数组格式 [1,1,1,-4,1]
        line = line.substring(1, line.length() - 1);
        String[] parts = line.split(",");
        int[] ops = new int[parts.length];
        for (int i = 0; i < parts.length; i++) {
            ops[i] = Integer.parseInt(parts[i].trim());
        }

        TreeSet<Integer> seated = new TreeSet<>();
        int lastPos = -1;

        for (int op : ops) {
            if (op == 1) {
                if (seated.size() == seatNum) {
                    lastPos = -1;
                    continue;
                }

                int bestPos = -1;
                int maxDist = 0;

                if (seated.isEmpty()) {
                    bestPos = 0;
                } else {
                    // 1. 考虑坐 0 号位置
                    if (!seated.contains(0)) {
                        bestPos = 0;
                        maxDist = seated.first();
                    }

                    // 2. 考虑中间位置
                    Integer[] currentSeats = seated.toArray(new Integer[0]);
                    for (int i = 0; i < currentSeats.length - 1; i++) {
                        int left = currentSeats[i];
                        int right = currentSeats[i + 1];
                        int dist = (right - left) / 2;
                        if (dist > maxDist) {
                            maxDist = dist;
                            bestPos = left + dist;
                        }
                    }

                    // 3. 考虑坐 seatNum - 1 号位置
                    if (!seated.contains(seatNum - 1)) {
                        int dist = (seatNum - 1) - seated.last();
                        if (dist > maxDist) {
                            maxDist = dist;
                            bestPos = seatNum - 1;
                        }
                    }
                }

                if (bestPos != -1) {
                    seated.add(bestPos);
                }
                lastPos = bestPos;
            } else {
                // 离场
                int leavePos = -op;
                seated.remove(leavePos);
            }
        }
        System.out.println(lastPos);
    }
}
```

### 4. 复杂度分析
- **时间复杂度**：$O(M \cdot N)$，其中 $M$ 是指令数量，$N$ 是座位数。每次进场操作需要遍历当前所有已坐的人。由于 $N, M \le 500$，总复杂度在 $2.5 \times 10^5$ 左右，完全满足机考要求。
- **空间复杂度**：$O(N)$，用于存储座位的占用情况。
